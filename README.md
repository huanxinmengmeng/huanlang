## 项目概述

幻语（HuanLang）是一款现代化的编程语言，支持**中文**、**拼音**和**英文**三种关键词体系。开发者可以根据自己的习惯自由选择关键词语言，极大降低了学习曲线，同时为国际化协作和 AI 代码生成提供了灵活性。

**版本**: 0.0.1
**发布日期**: 2026年4月

### 核心特性

- **三语关键词系统**：支持中文（`如果`）、拼音（`ruo`）和英文（`if`）三种关键词，可自由混用
- **现代化语法设计**：融合多语言优点，语法简洁优雅
- **完整类型系统**：内置丰富的数据类型，支持泛型
- **模块化设计**：支持模块、导入、结构、特征等组织代码
- **跨平台支持**：设计支持多种后端目标平台
- **标准库完整**：提供 IO、集合、字符串、数学等标准库

## 三语关键词系统

幻语的核心特性是支持中文、拼音、英文三种关键词体系。同一源文件中可以混合使用三种语言的关键词。

### 控制流关键词示例

| 中文 | 拼音 | 英文 | 说明 |
|------|------|------|------|
| `令` | `ling` | `let` | 变量声明 |
| `定` | `ding` | `const` | 常量声明 |
| `若` | `ruo` | `if` | 条件分支 |
| `则` | `ze` | `then` | 条件 then |
| `否则` | `fouze` | `else` | 条件 else |
| `结束` | `jieshu` | `end` | 块结束 |
| `当` | `dang` | `while` | 条件循环 |
| `重复` | `chongfu` | `repeat` | 固定次数循环 |
| `对于` | `duiyu` | `for` | 遍历循环 |
| `函数` | `hanshu` | `func` | 函数定义 |
| `返回` | `fanhui` | `return` | 返回语句 |

### 混用示例

```hl
let 年龄 = 25
若 年龄 > 18 then
    显示("成年")
end
```

三种风格等价转换：
```hl
// 中文风格
令 年龄 设为 25
若 年龄 大于 18 则
    显示("成年")
结束

// 英文风格
let age = 25
if age gt 18 then
    print("adult")
end
```

## 安装与构建

### 系统要求

- Rust 1.70 或更高版本
- 支持 Windows、macOS、Linux

### 构建步骤

```bash
# 克隆项目
git clone https://gitee.com/huanxinmengmeng/huanlang.git
cd huanlang

# 构建项目
cargo build --release

# 运行测试
cargo test

# 构建 LLVM 后端（可选）
cargo build --release --features llvm-backend
```

### 安装

```bash
# 安装到系统
cargo install --path .

# 或复制可执行文件
cp target/release/huan /usr/local/bin/
```

## 使用方法

### 命令行接口

```bash
# 编译并运行
huan run example.hl

# 仅编译
huan build example.hl -o output

# 类型检查
huan check example.hl

# 格式化代码
huan fmt example.hl

# 词法分析
huan lex example.hl

# 语法分析
huan parse example.hl

# 显示版本
huan version

# 显示帮助
huan help
```

### 语法示例

#### Hello World

```hl
#!/usr/bin/env huan run

导入 "标准库/核心"

函数 主() 返回 整数
开始
    显示("你好，幻语！")
    显示("欢迎使用幻语编程语言")
    返回 0
结束
```

#### 变量与类型

```hl
// 变量声明
令 x = 10
定 PI = 3.14159

// 类型标注
令 姓名: 字符串 = "幻语"
令 年龄: 整数 = 25

// 结构体
结构 点
    字段 横坐标: 浮点64
    字段 纵坐标: 浮点64
结束
```

#### 控制流

```hl
// 条件分支
若 x > 0 则
    显示("正数")
否则
    显示("非正数")
结束

// 循环
当 x > 0 循环
开始
    x = x - 1
结束

重复 5 次
开始
    显示("Hello")
结束

对于 i 在 0..10 循环
开始
    显示(i.到字符串())
结束
```

#### 函数

```hl
函数 求和(甲: 整数, 乙: 整数) 返回 整数
开始
    返回 甲 + 乙
结束

// 闭包
定 加一 = |x| { x + 1 }
```

#### 模式匹配

```hl
匹配 数字
开始
    1 => 显示("一")
    2 => 显示("二")
    3 => 显示("三")
    默认 => 显示("其他")
结束
```

## 技术细节

### 词法分析器

词法分析器负责将 UTF-8 编码的源代码字符串转换为 Token 流。

**主要功能**：
- 跳过空白字符和注释（支持 `--` 单行注释和 `/* */` 多行注释）
- 识别关键词、标识符、字面量、符号
- 跟踪源码位置（行号、列号、字节偏移）
- 处理 CJK 统一表意字符（中文字符）作为标识符

**支持的字面量**：
- 整数：`42`、`0xFF`、`0b1010`、`0o77`
- 浮点数：`3.14`、`1e-10`
- 字符串：支持转义字符 `\n`、`\t`、`\\` 等
- 字符：`'A'`
- 布尔：`真`/`假` 或 `true`/`false`

### 语法解析器

语法解析器采用**递归下降**与 **Pratt 解析器**相结合的算法。

**支持的语法结构**：
- 顶层定义：函数、结构、特征、实现、模块、导入等
- 表达式：算术运算、逻辑运算、函数调用、闭包等
- 语句：变量声明、循环、条件分支、模式匹配等

### 语义分析

语义分析器负责：
- 符号表管理（作用域、变量类型）
- 类型检查
- 控制流分析

## 标准库 API

### IO 模块

```hl
导入 标准库.IO

IO.显示("消息")           // 打印消息
IO.显示行("消息")         // 打印消息并换行
IO.读取行() -> 字符串     // 读取一行输入
```

### 集合模块

```hl
导入 标准库.集合

// 列表
令 列表 = 列表:整数.新建()
列表.添加(1)
列表.获取(0)

// 字典
令 字典 = 字典:字符串_整数.新建()
字典.设置("键", 42)
字典.获取("键")
```

### 字符串模块


## 常见问题（FAQ）

**Q1: 幻语适合哪些应用场景？**

A: 幻语是一门全栈语言，适用于：
- 系统编程（操作系统内核、驱动）
- 嵌入式开发（单片机、裸机）
- 后端服务（Web服务器、微服务）
- 桌面应用（跨平台GUI）
- AI应用开发（模型推理、Agent）

**Q2: 幻语与 Rust/C++ 相比如何？**

A: 性能方面，幻语通过LLVM生成与Rust/C++同级的机器码。语法方面，幻语提供中文原生支持，更适合中文开发者。内存安全方面，所有权模式提供了类似Rust的安全保障，但默认模式更灵活。

**Q3: 如何将现有C/C++项目迁移到幻语？**

A: 可通过以下步骤逐步迁移：
1. 使用 `huan import-lib` 导入C/C++库生成绑定
2. 用幻语重写部分模块，通过FFI与原代码交互
3. 使用 `huan transpile` 辅助转换简单函数
4. 逐步替换，直到完全迁移

**Q4: 幻语的学习曲线如何？**

A: 对于有编程经验的开发者，掌握基础语法约需1-2天。中文关键词降低了记忆负担。所有权系统对于熟悉Rust的开发者是熟悉的，新开发者可在默认模式下先忽略，逐步学习。

**Q5: 目前有没有在线Playground可以尝试？**

A: 暂时没有独立的在线Playground。您可以在本地克隆仓库并构建编译器，或下载发布版本的二进制文件进行尝试。我们也欢迎社区贡献者帮助搭建在线Playground。

**Q6: 为什么有两个仓库？我应该用哪个？**

A: Gitee 是国内访问更快的主仓库，所有开发活动和 Issue 跟踪均在此进行。GitHub 是自动同步的镜像，方便国际用户访问和参与。建议国内开发者优先使用 Gitee。

```bash

huan run examples/functions/函数与递归全览.hl

# 运行高级特性示例
huan run examples/advanced/高级特性全览.hl
```

## 💡 特色示例

### 三语关键词系统

幻语支持中文、拼音、英文三种关键词混用：

```huanlang
# 中文风格
令 数字 为 42
若 数字 > 10 则
    显示("大于10")
结束

# 拼音风格
ling 数字 为 42
ruo 数字 > 10 ze
    显示("大于10")
jieshu

# 英文风格
let 数字 为 42
if 数字 > 10 then
    显示("大于10")
end

# 混合风格
令 数字 为 42
if 数字 > 10 then
    显示("大于10")
end
```

### 函数式编程

```huanlang
令 数字 为 [1, 2, 3, 4, 5]

# 映射 - 求平方
令 平方 为 数字.映射(|x| { x * x })

# 过滤 - 取偶数
令 偶数 为 数字.过滤(|x| { x % 2 == 0 })

# 折叠 - 求和
令 和 为 数字.折叠(0, |a, b| { a + b })

# 链式调用
令 结果 为 数字
    .过滤(|x| { x % 2 == 0 })
    .映射(|x| { x * x })
    .折叠(0, |a, b| { a + b })
```

### 模式匹配

```huanlang
匹配 数字
    0 => 显示("零")
    1 => 显示("一")
    2 或 3 => 显示("二或三")
    4..9 => 显示("4到9")
    默认 => 显示("其他")
结束
```

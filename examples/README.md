# 幻语编程语言示例

本目录包含了使用幻语编程语言的示例程序。

## 示例列表

### 基础示例

#### 1. hello_world.hl
最简单的Hello World程序，展示基本的程序结构。

#### 2. variables.hl
变量和类型示例，展示幻语支持的数据类型。

#### 3. functions.hl
函数示例，包括函数定义、调用和递归。

#### 4. control_flow.hl
控制流示例，展示循环和条件判断。

#### 5. collections.hl
集合示例，展示列表和映射的使用。

#### 6. fibonacci.hl
斐波那契数列示例，展示递归函数的应用。

### 进阶示例

#### 7. oop.hl
面向对象编程示例，展示结构体和方法。

#### 8. algorithms.hl
排序算法示例，展示常见算法实现。

#### 9. error_handling.hl
错误处理示例，展示错误类型和结果类型。

### 实用示例

#### 10. file_processing.hl
文件处理示例，展示文件读写操作。

**内容：**
- 文本文件读写
- 文件复制
- 目录操作
- 文件存在性检查

#### 11. network_example.hl
网络请求示例，展示HTTP请求。

**内容：**
- GET/POST请求
- JSON数据处理
- 网络连接检查
- HTTP头信息

#### 12. concurrency_example.hl
并发编程示例，展示多线程操作。

**内容：**
- 任务组
- 通道通信
- 互斥锁
- 异步Future

#### 13. data_processing.hl
数据处理示例，展示数据转换和聚合。

**内容：**
- JSON数据解析
- 数据过滤
- 数据转换
- 数据聚合和统计

## 快速运行

```bash
# 运行基础示例
huan run examples/hello_world.hl
huan run examples/variables.hl
huan run examples/functions.hl

# 运行进阶示例
huan run examples/oop.hl
huan run examples/algorithms.hl
huan run examples/error_handling.hl

# 运行实用示例
huan run examples/file_processing.hl
huan run examples/network_example.hl
huan run examples/concurrency_example.hl
huan run examples/data_processing.hl
```

## 学习路径

### 第一阶段：基础（1-5）
1. hello_world.hl - 程序入口
2. variables.hl - 数据类型
3. functions.hl - 函数
4. control_flow.hl - 控制流
5. collections.hl - 集合

### 第二阶段：进阶（6-9）
6. fibonacci.hl - 递归
7. oop.hl - 面向对象
8. algorithms.hl - 算法
9. error_handling.hl - 错误处理

### 第三阶段：实用（10-13）
10. file_processing.hl - 文件IO
11. network_example.hl - 网络
12. concurrency_example.hl - 并发
13. data_processing.hl - 数据处理

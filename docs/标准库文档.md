# 幻语标准库文档

本目录包含了幻语编程语言的标准库文档。

## 模块列表

### 核心模块

#### 1. IO 模块 (io/)
文件和控制台输入输出操作。

**功能：**
- 文件读写操作
- 路径处理
- 目录操作
- 缓冲I/O

**使用示例：**
```
导入 IO;

函数 主() -> 整数 {
    变量 内容 = IO.读取文件("test.txt");
    IO.写入文件("output.txt", "你好，幻语！");

    打印("当前目录：" + IO.当前目录());
    返回 0;
}
```

#### 2. 集合模块 (collections/)
列表、映射、集合等数据结构。

**功能：**
- 列表操作：添加、删除、获取、遍历
- 映射操作：设置、获取、删除
- 集合操作：交集、并集、差集

**使用示例：**
```
导入 集合;

函数 主() -> 整数 {
    变量 list = 集合.新建列表<整数>();
    list 添加 1;
    list 添加 2;

    变量 map = 集合.新建映射<字符串, 整数>();
    map 设置 "age" 为 25;

    返回 0;
}
```

#### 3. 字符串模块 (string/)
字符串处理和操作。

**功能：**
- 字符串拼接
- 字符串搜索
- 字符串替换
- 字符串分割

**使用示例：**
```
导入 字符串;

函数 主() -> 整数 {
    变量 s = "Hello, 幻语!";
    变量 upper = 字符串.转大写(s);
    变量 lower = 字符串.转小写(s);
    变量 parts = 字符串.分割(s, ",");

    打印("大写：" + upper);
    打印("小写：" + lower);
    打印("分割：" + parts);

    返回 0;
}
```

#### 4. 数学模块 (math/)
数学运算和函数。

**功能：**
- 基本数学运算
- 三角函数
- 对数和指数
- 随机数生成

**使用示例：**
```
导入 数学;

函数 主() -> 整数 {
    变量 pi = 数学.PI;
    变量 e = 数学.E;
    变量 sqrt2 = 数学.开方(2);
    变量 random = 数学.随机();

    打印("PI：" + pi);
    打印("E：" + e);
    打印("sqrt(2)：" + sqrt2);
    打印("随机数：" + random);

    返回 0;
}
```

#### 5. 网络模块 (net/)
网络通信功能。

**功能：**
- HTTP请求
- TCP/UDP连接
- DNS解析
- 套接字操作

**使用示例：**
```
导入 网络;

函数 主() -> 整数 {
    // 发送HTTP GET请求
    变量 response = 网络.HTTP.请求("GET", "https://api.example.com");

    // 创建TCP连接
    变量 conn = 网络.TCP.连接("example.com", 8080);

    // 发送数据
    conn.发送("Hello");

    返回 0;
}
```

#### 6. 系统模块 (system/)
系统相关功能和信息。

**功能：**
- 命令行参数
- 环境变量
- 系统信息
- 进程管理

**使用示例：**
```
导入 系统;

函数 主() -> 整数 {
    变量 args = 系统.命令行参数();
    变量 cwd = 系统.当前目录();
    变量 env_val = 系统.获取环境变量("PATH");

    打印("命令行参数：" + args);
    打印("当前目录：" + cwd);
    打印("PATH：" + env_val);

    返回 0;
}
```

### 并发模块

#### 7. 并发模块 (concurrent/)
并发编程支持。

**功能：**
- 通道（Channel）
- 任务组（TaskGroup）
- 同步原语（Mutex, RwLock）
- 原子操作

**使用示例：**
```
导入 并发;

函数 主() -> 整数 {
    变量 channel = 并发.通道<整数>();

    // 创建任务
    并发.任务(|| {
        channel.发送(42);
    });

    变量 value = channel.接收();
    打印("收到：" + value);

    返回 0;
}
```

### 时间模块

#### 8. 时间模块 (time/)
时间和日期操作。

**功能：**
- 获取当前时间
- 时间戳转换
- 格式化输出
- 睡眠和计时

**使用示例：**
```
导入 时间;

函数 主() -> 整数 {
    变量 now = 时间.现在();
    变量 timestamp = 时间.时间戳();
    变量 formatted = 时间.格式化(now, "%Y-%m-%d %H:%M:%S");

    打印("当前时间：" + formatted);
    打印("时间戳：" + timestamp);

    时间.睡眠(1000); // 睡眠1秒

    返回 0;
}
```

## 标准库约定

### 模块导入
```huan
导入 模块名;          // 导入整个模块
导入 模块名.子模块;    // 导入特定子模块
```

### 命名规范
- 模块名：小写，使用下划线分隔
- 函数名：支持中文、英文、拼音
- 类型名：首字母大写

### 错误处理
标准库函数返回 `结果<R, E>` 类型来表示操作是否成功。

```huan
变量 result = IO.读取文件("test.txt");
匹配 result {
    成功(content) => 打印("文件内容：" + content),
    错误(e) => 打印("读取失败：" + e.描述),
}
```

## 添加新模块

要添加新的标准库模块：

1. 在 `src/stdlib/` 目录下创建新模块
2. 实现模块功能
3. 在 `mod.rs` 中导出模块
4. 编写文档和使用示例
5. 添加测试用例

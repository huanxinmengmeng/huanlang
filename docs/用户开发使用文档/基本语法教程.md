# 基本语法教程

本文档介绍幻语编程语言的基本语法特性和使用方法。

## 变量声明

### 基本变量

```hl
// 使用 "令" 关键字声明可变变量
令 x = 42;

// 使用 "定" 关键字声明常量
定 PI = 3.14159;

// 显式类型标注
令 y: 整数 = 100;
令 name: 字符串 = "幻语";
```

### 类型系统

幻语支持以下基本类型：

- **整数类型**：`整数`、`i8`、`i16`、`i32`、`i64`、`u8`、`u16`、`u32`、`u64`
- **浮点类型**：`f32`、`f64`
- **布尔类型**：`布尔`
- **字符类型**：`字符`
- **字符串类型**：`字符串`
- **单元类型**：`单元`

## 控制流

### 条件语句

```hl
令 年龄 = 18;

若 年龄 >= 18 则
    打印("成年人");
否则
    打印("未成年人");
结束
```

### 循环语句

#### while 循环

```hl
令 i = 0;
当 i < 10 时
    打印("循环次数: " + i);
    i = i + 1;
结束
```

#### repeat 循环

```hl
重复 5 次
    打印("重复执行");
结束
```

#### for 循环

```hl
对于 每个 数字 在 [1, 2, 3, 4, 5] 中
    打印("数字: " + 数字);
结束
```

## 函数定义

### 基本函数

```hl
函数 加法(a: 整数, b: 整数) -> 整数 {
    返回 a + b;
}
```

### 函数调用

```hl
令 结果 = 加法(10, 20);
打印("结果: " + 结果);
```

## 结构体

### 定义结构体

```hl
结构 人 {
    姓名: 字符串,
    年龄: 整数,
    性别: 字符串,
}
```

### 使用结构体

```hl
令 张三 = 人 {
    姓名: "张三",
    年龄: 30,
    性别: "男",
};

打印(张三.姓名 + " 的年龄是 " + 张三.年龄);
```

## 错误处理

```hl
函数 除法(a: 整数, b: 整数) -> 结果<整数, 字符串> {
    若 b == 0 则
        返回 错误("除数不能为零");
    否则
        返回 成功(a / b);
    结束
}

令 结果 = 除法(10, 2);
匹配 结果 {
    成功(值) => 打印("结果: " + 值),
    错误(信息) => 打印("错误: " + 信息),
}
```

## 模块系统

### 导入模块

```hl
导入 "标准库/控制台" 作为 控制台;

控制台.打印("使用导入的模块");
```

### 导出模块

```hl
模块 数学 {
    函数 加法(a: 整数, b: 整数) -> 整数 {
        返回 a + b;
    }
}

导出 数学;
```

| 否则 | fouze | else | 条件分支 |

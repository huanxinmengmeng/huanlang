## 环境搭建

### 1. 安装 Rust 工具链

幻语编译器是基于 Rust 开发的，因此需要先安装 Rust 工具链：

```bash
# 使用 rustup 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 验证安装
rustc --version
cargo --version
```

### 2. 克隆幻语项目

```bash
git clone https://gitee.com/huanxinmengmeng/huanlang.git
cd huanlang
```

### 3. 构建幻语编译器

```bash
cargo build --release
```

构建完成后，可执行文件将位于 `target/release` 目录中。

## 第一个程序

### 1. 创建源文件

创建一个名为 `你好世界.hl` 的文件，内容如下：

```hl
// 你好世界示例
令 主函数() {
    打印("你好，世界！");
}
```

### 2. 编译与运行

```bash
# 编译
./target/release/huanlang build 你好世界.hl

# 运行
./target/release/huanlang run 你好世界.hl
```

## 基本开发流程

1. **编写代码**：创建 `.hl` 或 `.幻` 扩展名的源文件
2. **编译检查**：使用 `huanlang check` 命令检查代码
3. **构建项目**：使用 `huanlang build` 命令构建项目
4. **运行程序**：使用 `huanlang run` 命令运行程序
5. **测试验证**：使用 `huanlang test` 命令运行测试

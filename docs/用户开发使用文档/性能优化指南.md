## 1. 概述

幻语编程语言提供了完整的性能优化工具链，帮助开发者快速定位和解决性能瓶颈。本指南介绍如何使用这些工具来优化您的程序性能。

### 核心性能优化模块

幻语编译器包含以下性能优化模块：

- **性能剖析器（Profiler）**：测量代码执行时间和频率
- **内存优化器（Memory Optimizer）**：内存池、对象池、环形缓冲区
- **日志系统（Logger）**：结构化日志、多输出支持
- **基准测试框架（Benchmark）**：性能回归检测
- **调试工具（Debugger）**：栈追踪、错误报告

这些模块都已集成在 `huanlang::core::performance` 命名空间中。

## 2. 性能剖析器使用

### 2.1 基本概念

性能剖析器（Profiler）是一个强大的工具，可以帮助您：

- 测量函数执行时间
- 统计函数调用次数
- 记录执行路径
- 生成性能报告

### 2.2 基本使用

在 Rust 代码中使用 Profiler：

```rust
use huanlang::core::performance::profiler::{Profiler, ScopedTimer};

fn main() {
    // 创建 Profiler 实例
    let profiler = Profiler::new();
    
    // 开始计时
    profiler.start_timer("整体运行");
    
    // 使用 ScopedTimer 自动计时
    {
        let _timer = ScopedTimer::new(&profiler, "函数A");
        // 执行函数 A 的代码
    }
    
    // 手动开始和结束计时
    profiler.start_timer("函数B");
    // 执行函数 B 的代码
    profiler.end_timer("函数B");
    
    // 增加计数器
    profiler.increment_counter("请求数", 1);
    
    // 记录追踪点
    use std::collections::HashMap;
    let mut metadata = HashMap::new();
    metadata.insert("状态".to_string(), "完成".to_string());
    profiler.record_trace_point("处理完成", metadata);
    
    // 结束整体计时
    profiler.end_timer("整体运行");
    
    // 获取统计信息
    let stats = profiler.get_stats();
    println!("{:?}", stats);
    
    // 生成报告
    let report = profiler.report();
    println!("{}", report);
    
    // 重置 Profiler
    profiler.reset();
}
```

### 2.3 Profiler API 详解

#### Profiler 创建

```rust
// 创建默认 Profiler（最大 1000 个追踪点）
let profiler = Profiler::new();

// 创建自定义最大追踪点数的 Profiler
let profiler = Profiler::with_max_trace_points(5000);
```

#### 计时器操作

```rust
// 开始计时
profiler.start_timer("函数名称");

// 结束计时
profiler.end_timer("函数名称");

// 使用 ScopedTimer 自动管理
{
    let _timer = ScopedTimer::new(&profiler, "代码块");
    // 代码块结束时自动调用 end_timer
}
```

#### 计数器操作

```rust
// 增加计数器
profiler.increment_counter("事件名称", 1);

// 多次增加
profiler.increment_counter("事件名称", 10);
```

#### 追踪点记录

```rust
use std::collections::HashMap;

let mut metadata = HashMap::new();
metadata.insert("用户ID".to_string(), "12345".to_string());
metadata.insert("操作".to_string(), "登录".to_string());

profiler.record_trace_point("用户登录", metadata);
```

#### 获取统计信息

```rust
// 获取完整统计
let stats = profiler.get_stats();

// 访问计时器统计
for (name, timer) in &stats.timers {
    println!("{}: {:?} 次, 总时间 {:?}", name, timer.count, timer.total_time);
    println!("  平均: {:?}, 最小: {:?}, 最大: {:?}", timer.avg_time, timer.min_time, timer.max_time);
}

// 访问计数器统计
for (name, counter) in &stats.counters {
    println!("{}: {} ({:?} 次)", name, counter.value, counter.count);
}

// 访问追踪点
for trace in &stats.trace_points {
    println!("{} at {:?}: {:?}", trace.name, trace.timestamp, trace.metadata);
}
```

#### 生成报告

```rust
let report = profiler.report();
println!("{}", report);
```

输出示例：
```

--- 计数器 ---
请求数: 100 (100 次)
```

#### 重置

```rust
profiler.reset();  // 清空所有数据
```

## 3. 内存优化

### 3.1 内存追踪器

内存追踪器可以帮助监控内存使用情况：

```rust
use huanlang::core::performance::memory::MemoryTracker;

fn main() {
    let mut tracker = MemoryTracker::new();
    
    // 记录分配
    tracker.track_allocation(1024);  // 分配 1024 字节
    tracker.track_allocation(512);   // 分配 512 字节
    
    // 记录释放
    tracker.track_deallocation(1024);  // 释放 1024 字节
    
    // 获取统计
    let stats = tracker.get_stats();
    println!("总分配: {} 次", stats.total_allocations);
    println!("总释放: {} 次", stats.total_deallocations);
    println!("当前使用: {} 字节", stats.current_bytes);
    println!("峰值使用: {} 字节", stats.peak_bytes);
    
    // 重置
    tracker.reset();
}
```

### 3.2 内存池

内存池可以减少频繁的内存分配和释放：

```rust
use huanlang::core::performance::memory::{MemoryPool, PoolConfig};

fn main() {
    // 创建配置
    let config = PoolConfig::default_config(1024);  // 1KB 块大小
    
    // 或者使用预定义配置
    let small_config = PoolConfig::small_object_config();  // 小对象
    let medium_config = PoolConfig::medium_object_config();  // 中等对象
    
    // 创建内存池
    let mut pool = MemoryPool::new(config);
    
    // 分配内存
    if let Some(mut chunk) = pool.allocate() {
        // 使用 chunk
        chunk[0] = 0xFF;
        
        // 释放回池
        pool.deallocate(chunk);
    }
    
    // 获取池状态
    println!("可用块数: {}", pool.free_count());
    println!("已分配块数: {}", pool.allocated_count());
}
```

### 3.3 对象池

对象池用于重用对象实例，避免频繁的创建和销毁：

```rust
use huanlang::core::performance::memory::ObjectPool;

#[derive(Debug)]
struct Connection {
    id: u64,
    connected: bool,
}

impl Connection {
    fn new() -> Self {
        Connection { id: 0, connected: false }
    }
    
    fn connect(&mut self) {
        self.connected = true;
    }
    
    fn disconnect(&mut self) {
        self.connected = false;
    }
}

fn main() {
    // 创建对象池
    let mut pool = ObjectPool::new(|| Connection::new());
    
    // 预热池
    pool.warm(10);  // 预创建 10 个对象
    
    // 获取对象
    let mut conn = pool.acquire();
    conn.id = 1;
    conn.connect();
    
    // 使用对象
    println!("使用连接: {:?}", conn);
    
    // 释放回池
    conn.disconnect();
    pool.release(conn);
    
    // 查看池状态
    println!("可用: {}, 在用: {}", pool.available_count(), pool.in_use_count());
}
```

### 3.4 环形缓冲区

环形缓冲区适合用于生产者-消费者场景：

```rust
use huanlang::core::performance::memory::RingBuffer;

fn main() {
    // 创建容量为 10 的环形缓冲区
    let mut buffer = RingBuffer::<i32>::new(10);
    
    // 写入数据
    for i in 0..5 {
        buffer.write(i);
    }
    
    println!("是否已满: {}", buffer.is_full());
    println!("使用空间: {}", buffer.used_space());
    
    // 读取数据
    while let Some(value) = buffer.read() {
        println!("读取: {}", value);
    }
    
    println!("是否为空: {}", buffer.is_empty());
}
```

针对小对象的优化分配器：

## 4. 基准测试

### 4.1 基准测试组

```rust
use huanlang::core::performance::bench::{BenchmarkGroup, BenchmarkConfig};

fn main() {
    // 创建基准测试组
    let mut group = BenchmarkGroup::new("字符串操作");
    
    // 自定义配置
    let config = BenchmarkConfig {
        iterations: 10000,  // 迭代次数
        warmup: 100,        // 预热次数
        name: "自定义".to_string(),
    };
    
    // 或者使用 Builder 风格
    group = group.iterations(5000).warmup(50);
    
    // 运行基准测试
    group.bench("字符串连接", || {
        let _ = "Hello".to_string() + " " + "World";
    });
    
    group.bench_with_result("字符串分割", || {
        "a,b,c,d,e".split(',').collect::<Vec<_>>()
    });
    
    // 打印结果
    group.print_results();
    
    // 获取结果列表
    let results = group.results();
    for result in results {
        println!("{}: {:.2} ns", result.name, result.mean_ns);
    }
}
```

### 4.3 性能回归检测

```rust
use huanlang::core::performance::bench::{RegressionDetector, BenchmarkResult};

fn main() {
    let mut detector = RegressionDetector::new(0.05);  // 5% 阈值
    
    // 保存基线
    let baseline = BenchmarkResult {
        name: "测试".to_string(),
        mean_ns: 100.0,
        median_ns: 100.0,
        std_dev_ns: 10.0,
        min_ns: 90,
        max_ns: 110,
        iterations: 1000,
    };
    detector.save_baseline("测试", baseline);
    
    // 添加当前结果
    let current = BenchmarkResult {
        name: "测试".to_string(),
        mean_ns: 110.0,  // 慢了 10%
        median_ns: 110.0,
        std_dev_ns: 10.0,
        min_ns: 100,
        max_ns: 120,
        iterations: 1000,
    };
    detector.add_current("测试", current);
    
    // 检测回归
    let regressions = detector.detect();
    println!("发现 {} 个回归", regressions.len());
    
    // 打印比较
    detector.print_comparison();
}
```

### 5.1 编译错误处理

```rust
use huanlang::core::performance::debug::{
    CompileError, ErrorLevel, ErrorSpan, ErrorRegistry
};

fn main() {
    // 创建错误
    let mut error = CompileError::new("E001", "类型不匹配");
    
    // 添加上下文
    let span = ErrorSpan {
        file: "main.hl".to_string(),
        line: 42,
        column: Some(15),
        end_line: None,
        end_column: None,
        label: "这里有问题".to_string(),
        text: None,
    };
    error = error.with_span(span);
    
    // 添加建议
    error = error.with_suggestion("考虑使用类型转换");
    
    // 设置错误级别
    error = error.with_level(ErrorLevel::Error);
    
    // 添加详情
    error.add_detail("期望类型: 整数");
    error.add_detail("实际类型: 字符串");
    
    // 打印错误
    println!("{}", error);
    
    // 转换为 JSON
    println!("{}", error.to_json());
    
    // 使用错误注册表
    let registry = ErrorRegistry::new();
    if let Some(explain) = registry.explain("E001") {
        println!("E001: {}", explain);
    }
    
    // 获取所有错误
    let all = registry.all_errors();
    for (code, desc) in all {
        println!("{}: {}", code, desc);
    }
}
```

## 6. 日志系统

### 6.1 基本使用

```rust
use huanlang::core::performance::logger::{
    Logger, Level, ConsoleOutput, FileOutput, LevelFilter, ModuleFilter
};

fn main() {
    // 创建 Logger
    let mut logger = Logger::new();
    
    // 或者带模块名

    // 记录日志
    logger.trace("这是追踪消息");
    logger.debug("这是调试消息");
    logger.info("这是信息消息");
    logger.warn("这是警告消息");
    logger.error("这是错误消息");
    
    // 使用 log 方法
    logger.log(Level::Info, "直接使用 log 方法");
    
    // 带字段的日志

    let mut fields = HashMap::new();

    fields.insert("操作".to_string(), "登录".to_string());
    logger.log_with_fields(Level::Info, "用户操作", fields);
    
    // 刷新输出
    logger.flush();
}
```

```rust
        }
```

```rust
            Box::new(CustomOutput)
        }
    }
    
    logger.add_output(Box::new(CustomOutput));
}
```

## 7. 实际案例

### 7.1 Web 服务器性能优化

```rust
use huanlang::core::performance::*;
use std::sync::Arc;

struct WebServer {
    profiler: Arc<Profiler>,
    logger: Logger,
    connection_pool: ObjectPool<Connection>,
}

impl WebServer {
    fn new() -> Self {
        let profiler = Arc::new(Profiler::new());
        let mut logger = Logger::with_module("服务器");
        logger.add_output(Box::new(logger::ConsoleOutput::new()));
        
        let mut pool = ObjectPool::new(|| Connection::new());
        pool.warm(100);
        
        WebServer { profiler, logger, connection_pool: pool }
    }
    
    fn handle_request(&mut self, request: Request) -> Response {
        let _timer = profiler::ScopedTimer::new(&self.profiler, "处理请求");
        self.profiler.increment_counter("请求数", 1);
        
        self.logger.info(format!("处理请求: {}", request.path));
        
        let mut conn = self.connection_pool.acquire();
        
        // ... 处理逻辑 ...
        
        self.connection_pool.release(conn);
        Response::ok()
    }
    
    fn report_stats(&self) {
        let report = self.profiler.report();
        self.logger.info(report);
    }
}
```

```rust
fn demo_profiler() {
    println!("--- 性能剖析器演示 ---");
    
    let profiler = Profiler::new();
    
    // 使用 scoped timer
    {
        let _timer = ScopedTimer::new(&profiler, "示例函数");
        // 模拟工作
        std::thread::sleep(std::time::Duration::from_millis(100));
    }
    
    // 增加计数器
    profiler.increment_counter("示例计数", 1);
    
    // 记录追踪点
    let mut metadata = HashMap::new();
    metadata.insert("状态".to_string(), "完成".to_string());
    profiler.record_trace_point("演示完成", metadata);
    
    // 打印报告
    println!("性能报告:\n{}", profiler.report());
    println!();
}

fn demo_logger() {
    println!("--- 日志系统演示 ---");
    
    let mut logger = Logger::with_module("演示");
    logger.add_output(Box::new(logger::ConsoleOutput::new()));
    logger.add_filter(Box::new(logger::LevelFilter::new(logger::Level::Trace)));
    
    logger.trace("追踪级别消息");
    logger.debug("调试级别消息");
    logger.info("信息级别消息");
    logger.warn("警告级别消息");
    logger.error("错误级别消息");
    
    println!();
}

fn demo_memory() {

    // 内存池
    let config = memory::PoolConfig::default_config(1024);
    let mut pool = memory::MemoryPool::new(config);
    let chunk = pool.allocate().unwrap();
    println!("从内存池分配了 {} 字节", chunk.len());
    pool.deallocate(chunk);
    println!("可用块数: {}", pool.free_count());
    
    // 环形缓冲区
    let mut buffer = memory::RingBuffer::new(5);
    for i in 0..5 {
        buffer.write(i);
    }
    println!("环形缓冲区使用: {} / 5", buffer.used_space());
```

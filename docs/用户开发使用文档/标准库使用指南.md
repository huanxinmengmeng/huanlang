# 标准库使用指南

本文档介绍幻语标准库的主要模块和使用方法。

## 控制台模块

### 基本输出

```hl
导入 "标准库/控制台" 作为 控制台;

// 打印文本
控制台.打印("Hello, World!");

// 打印带换行的文本
控制台.打印行("Hello with newline");

// 打印错误信息
控制台.打印错误("Error message");
```

### 输入

```hl
// 读取用户输入
令 姓名 = 控制台.读取行("请输入您的姓名: ");
控制台.打印("您好, " + 姓名);

// 读取数字
令 年龄 = 控制台.读取整数("请输入您的年龄: ");
```

### 文本样式

```hl
// 彩色文本
控制台.打印彩色("红色文本", 控制台.颜色.红);

// 粗体文本
控制台.打印粗体("粗体文本");

// 下划线文本
控制台.打印下划线("下划线文本");
```

## 系统模块

### 环境变量

```hl
导入 "标准库/系统" 作为 系统;

// 获取环境变量
令 路径 = 系统.获取环境变量("PATH");
控制台.打印("PATH: " + 路径);

// 设置环境变量
系统.设置环境变量("MY_VAR", "value");
```

### 命令行参数

```hl
// 解析命令行参数
令 参数 = 系统.命令行参数();
控制台.打印("参数数量: " + 参数.长度());

对于 每个 参数 在 参数 中
    控制台.打印("参数: " + 参数);
结束
```

### 文件系统

```hl
// 检查文件是否存在
若 系统.文件存在("test.txt") 则
    控制台.打印("文件存在");
否则
    控制台.打印("文件不存在");
结束

// 创建目录
系统.创建目录("new_dir");
```

## 集合模块

### 列表

```hl
导入 "标准库/集合" 作为 集合;

// 创建列表
令 列表 = 集合.列表::新();

// 添加元素
列表.添加(1);
列表.添加(2);
列表.添加(3);

// 遍历列表
对于 每个 元素 在 列表 中
    控制台.打印("元素: " + 元素);
结束

// 获取元素
令 第一个 = 列表.获取(0);
```

### 映射

```hl
// 创建映射
令 映射 = 集合.映射::新();

// 添加键值对
映射.插入("name", "幻语");
映射.插入("version", "1.0");

// 获取值
令 名称 = 映射.获取("name");
控制台.打印("名称: " + 名称);
```

### 集合

```hl
// 创建集合
令 集合 = 集合.集合::新();

// 添加元素
集合.插入(1);
集合.插入(2);
集合.插入(1); // 重复元素会被忽略

// 检查元素是否存在
若 集合.包含(1) 则
    控制台.打印("集合包含 1");
结束
```

## 字符串模块

### 字符串操作

```hl
导入 "标准库/字符串" 作为 字符串;

令 s = "Hello, World!";

// 长度
控制台.打印("长度: " + s.长度());

// 子字符串
令 substr = s.子串(0, 5); // "Hello"

// 查找
令 位置 = s.查找("World");

// 替换
令 new_str = s.替换("World", "幻语");

// 分割
令 部分 = s.分割(", ");
```

### 格式化

```hl
// 格式化字符串
令 格式化 = 字符串.格式化("{} + {} = {}", 1, 2, 3);
控制台.打印(格式化); // "1 + 2 = 3"
```

## 数学模块

```hl
导入 "标准库/数学" 作为 数学;

// 基本运算
令 最大值 = 数学.最大值(10, 20);
令 最小值 = 数学.最小值(10, 20);
令 绝对值 = 数学.绝对值(-5);

// 三角函数
令 正弦 = 数学.正弦(数学.PI / 2);
令 余弦 = 数学.余弦(0);

// 指数和对数
令 幂 = 数学.幂(2, 3); // 8
令 对数 = 数学.自然对数(数学.E); // 1
```

## 随机模块

```hl
导入 "标准库/随机" 作为 随机;

// 初始化随机数生成器
随机.初始化();

// 生成随机整数
令 随机数 = 随机.整数(1, 100);
控制台.打印("随机数: " + 随机数);

// 生成随机浮点数
令 随机浮点数 = 随机.浮点数(0.0, 1.0);

// 随机布尔值
令 随机布尔 = 随机.布尔();
```

## 时间模块

```hl
导入 "标准库/时间" 作为 时间;

// 获取当前时间
令 当前时间 = 时间.当前时间();
控制台.打印("当前时间: " + 当前时间.字符串());

// 时间间隔
时间.睡眠(1000); // 睡眠1秒

// 计时器
令 计时器 = 时间.计时器::新();
// 执行一些操作
令 耗时 = 计时器.耗时();
控制台.打印("耗时: " + 耗时.毫秒() + " 毫秒");
```

## 网络模块

### HTTP 客户端

```hl
导入 "标准库/网络" 作为 网络;

// 创建 HTTP 客户端
令 客户端 = 网络.HTTP客户端::新();

// 发送 GET 请求
令 响应 = 客户端.获取("https://api.example.com");
控制台.打印("状态码: " + 响应.状态码());
控制台.打印("响应体: " + 响应.文本());

// 发送 POST 请求
令 数据 = 网络.JSON::解析("{\"name\": \"test\"}");
令 post响应 = 客户端. post("https://api.example.com", 数据);
```

### TCP 服务器

```hl
// 创建 TCP 服务器
令 服务器 = 网络.TCP服务器::监听("127.0.0.1:8080");

// 接受连接
当 真 时
    令 连接 = 服务器.接受();
    连接.发送("Hello from server");
    连接.关闭();
结束
```

## 加密模块

```hl
导入 "标准库/加密" 作为 加密;

// MD5 哈希
令 md5 = 加密.MD5("hello");
控制台.打印("MD5: " + md5);

// SHA256 哈希
令 sha256 = 加密.SHA256("hello");
控制台.打印("SHA256: " + sha256);

// 生成随机密钥
令 密钥 = 加密.生成随机密钥(32);
控制台.打印("密钥: " + 密钥);
```

## 序列化模块

### JSON 操作

```hl
导入 "标准库/序列化" 作为 序列化;

// 序列化对象
令 对象 = {
    "name": "幻语",
    "version": "1.0",
    "features": ["语法简洁", "性能优异"]
};

令 json字符串 = 序列化.JSON::序列化(对象);
控制台.打印(json字符串);

// 反序列化 JSON
令 解析结果 = 序列化.JSON::反序列化(json字符串);
控制台.打印("名称: " + 解析结果["name"]);
```

### 二进制序列化

```hl
// 序列化到二进制
令 数据 = [1, 2, 3, 4, 5];
令 二进制 = 序列化.二进制::序列化(数据);

// 反序列化
令 反序列化数据 = 序列化.二进制::反序列化(二进制);
```

## 实践示例

### 完整的文件处理示例

```hl
导入 "标准库/控制台" 作为 控制台;
导入 "标准库/输入输出" 作为 输入输出;

函数 主函数() {
    // 写入文件
    输入输出.写入文件("example.txt", "Hello from 幻语!");
    控制台.打印("文件已创建");
    
    // 读取文件
    令 内容 = 输入输出.读取文件("example.txt");
    控制台.打印("文件内容: " + 内容);
    
    // 检查文件大小
    令 大小 = 输入输出.文件大小("example.txt");
    控制台.打印("文件大小: " + 大小 + " 字节");
}
```

### 网络请求示例

```hl
导入 "标准库/控制台" 作为 控制台;
导入 "标准库/网络" 作为 网络;

函数 主函数() {
    令 客户端 = 网络.HTTP客户端::新();
    令 响应 = 客户端.获取("https://api.github.com/users/huanlang-dev");
    
    控制台.打印("状态码: " + 响应.状态码());
    控制台.打印("响应体: " + 响应.文本());
}
```

# 性能优化与调试支持使用指南

## 概述

本指南详细介绍幻语编程语言提供的性能优化和调试支持功能，包括代码实现和使用示例。所有功能都已在 `src/performance` 目录下实现为 Rust 代码。

## 1. 性能剖析工具

### 1.1 性能剖析器 (profiler.rs)

性能剖析器提供程序运行时的性能数据采集和分析功能。

**主要功能**：
- CPU 时间测量
- 函数调用频率统计
- 自定义追踪点
- perf 映射文件生成
- 火焰图数据导出

**使用示例**：

```rust
use huanlang::performance::Profiler;

fn main() {
    // 创建性能剖析器
    let profiler = Profiler::new();

    // 开始计时
    profiler.start_timer("complex_operation");

    // 执行代码
    let result = complex_calculation(30);

    // 结束计时
    profiler.end_timer("complex_operation");

    // 获取统计信息
    let stats = profiler.get_stats();

    // 打印结果
    println!("Timer stats: {:?}", stats.timers);
    println!("Counter stats: {:?}", stats.counters);
}
```

**作用域计时器**（自动管理开始/结束）：

```rust
fn example() {
    let profiler = Profiler::new();

    // 使用作用域计时器
    {
        let _timer = ScopedTimer::new(&profiler, "auto_timer");
        // 执行被计时的代码
        do_something();
    }
    // 计时器自动结束
}
```

**计数器使用**：

```rust
fn example() {
    let profiler = Profiler::new();

    profiler.increment_counter("requests", 1);
    profiler.increment_counter("bytes_sent", 1024);

    let stats = profiler.get_stats();
    println!("Requests: {}", stats.counters["requests"].value);
}
```

## 2. 内存优化模块

### 2.1 内存追踪器 (memory.rs)

内存追踪器提供内存使用追踪和统计功能。

**使用示例**：

```rust
use huanlang::performance::MemoryTracker;

fn example() {
    let tracker = MemoryTracker::new();

    // 记录分配
    tracker.track_allocation(100);

    // 记录释放
    tracker.track_deallocation(100);

    // 获取统计
    let stats = tracker.get_stats();
    println!("Total allocations: {}", stats.total_allocations);
    println!("Current bytes: {}", stats.current_bytes);
    println!("Peak bytes: {}", stats.peak_bytes);
}
```

### 2.2 内存池

内存池用于重用固定大小的内存块，减少分配开销。

```rust
use huanlang::performance::{MemoryPool, PoolConfig};

fn example() {
    // 创建配置
    let config = PoolConfig::default_config(64);

    // 创建内存池
    let mut pool = MemoryPool::new(config);

    // 分配内存
    let chunk = pool.allocate();
    if let Some(mut data) = chunk {
        // 使用内存
        data[0] = 42;
    }

    // 释放回池
    if let Some(data) = chunk {
        pool.deallocate(data);
    }

    println!("Free chunks: {}", pool.free_count());
    println!("Allocated chunks: {}", pool.allocated_count());
}
```

**预配置的池**：

```rust
// 小对象池（64字节块）
let small_pool = PoolConfig::small_object_config();

// 中等对象池（1024字节块）
let medium_pool = PoolConfig::medium_object_config();
```

### 2.3 小对象分配器

```rust
use huanlang::performance::SmallObjectAllocator;

fn example() {
    let mut allocator = SmallObjectAllocator::new();

    // 分配小对象（自动对齐到8字节边界）
    let chunk = allocator.allocate(50).unwrap();

    // 释放
    allocator.deallocate(chunk);

    // 获取内存使用
    println!("Memory usage: {} bytes", allocator.get_memory_usage());
}
```

### 2.4 对象池

用于复用大型对象，如数据库连接、线程等。

```rust
use huanlang::performance::ObjectPool;

fn example() {
    // 创建对象池
    let mut pool = ObjectPool::new(|| Vec::<i32>::new());

    // 预热池
    pool.warm(10);

    // 获取对象
    let mut obj = pool.acquire();
    obj.push(1);
    obj.push(2);

    // 归还对象
    pool.release(obj);

    println!("Available: {}", pool.available_count());
    println!("In use: {}", pool.in_use_count());
}
```

### 2.5 环形缓冲区

高效的生产者-消费者数据结构。

```rust
use huanlang::performance::RingBuffer;

fn example() {
    let mut buffer = RingBuffer::<i32>::new(10);

    // 写入数据
    buffer.write(1);
    buffer.write(2);
    buffer.write(3);

    // 读取数据
    while let Some(value) = buffer.read() {
        println!("Read: {}", value);
    }

    // 检查状态
    println!("Is empty: {}", buffer.is_empty());
    println!("Is full: {}", buffer.is_full());
    println!("Used space: {}", buffer.used_space());
}
```

### 2.6 内存对齐

```rust
use huanlang::performance::MemoryAligner;

fn example() {
    // 对齐到 8 字节边界
```

## 3. 调试工具模块

### 3.1 栈跟踪 (debug.rs)

```rust
use huanlang::performance::{StackTrace, DebugHelper};

fn example() {
    // 捕获栈跟踪
    let trace = StackTrace::capture();

    // 格式化为字符串
    println!("Stack trace:\n{}", trace);

    // 获取函数名
    if let Some(name) = trace.get_function_name() {
        println!("Main function: {}", name);
    }
}
```

**调试辅助函数**：

```rust
fn example() {
    let value = 42;

    // 打印变量
    DebugHelper::print_variable("my_var", &value);

    // 打印类型
    DebugHelper::print_type(&value);

    // 打印地址
    DebugHelper::print_address(&value);

    // 打印调用栈
    DebugHelper::print_stack_trace();

    // 断言
    DebugHelper::assert(value > 0, "Value should be positive");
    DebugHelper::assert_eq(1, 1, "Values should be equal");
}
```

### 3.2 错误报告

```rust
use huanlang::performance::{

fn example() {

        .with_suggestion("考虑使用 转为整数() 方法")
        .with_level(ErrorLevel::Error);

    // 打印错误

    // 导出 JSON
    println!("{}", error.to_json());
}
```

## 4. 日志系统

## 5. 基准测试框架

### 5.1 基本使用 (bench.rs)

```rust
use huanlang::performance::bench::{BenchmarkGroup, BenchmarkResult, Blackhole};

fn example() {
    let mut group = BenchmarkGroup::new("String Operations")
        .iterations(1000)
        .warmup(100);

    // 添加基准测试
    group.bench("String concatenation", || {
        let result = "Hello".to_string() + " " + "World";
        Blackhole::consume(result);
    });

    group.bench("String splitting", || {
        let text = "a,b,c,d,e,f,g,h".to_string();
        let parts: Vec<&str> = text.split(',').collect();
        Blackhole::consume(parts);
    });

    // 打印结果
    group.print_results();
}
```

### 5.2 使用宏进行快速基准测试

```rust
use huanlang::performance::bench::bench;

fn example() {
    let result = bench!("Quick bench", 1000, || {
        let sum: i32 = (1..1000).sum();
        sum
    });

    println!("Mean: {:.2} ns", result.mean_ns);
    println!("Median: {:.2} ns", result.median_ns);
}
```

### 5.3 防止编译器优化

```rust
use huanlang::performance::bench::Blackhole;

fn example() {
    let data = vec![1i32; 1000];

    // 防止优化掉计算
    let sum: i32 = data.iter().sum();
    Blackhole::consume(sum);

    // 或者使用 return
    let result = some_computation();
    Blackhole::r#return(result);
}
```

### 5.4 性能回归检测

```rust
use huanlang::performance::bench::{BenchmarkRunner, RegressionDetector};

fn example() {
    let mut detector = RegressionDetector::new(0.05); // 5% threshold

    // 保存基线
    detector.save_baseline("operation", BenchmarkResult {
        name: "operation".to_string(),
        mean_ns: 100.0,
        median_ns: 100.0,
        std_dev_ns: 10.0,
        min_ns: 90,
        max_ns: 110,
        iterations: 100,
    });

    // 添加当前结果
    detector.add_current("operation", BenchmarkResult {
        name: "operation".to_string(),
        mean_ns: 120.0,
        median_ns: 120.0,
        std_dev_ns: 10.0,
        min_ns: 110,
        max_ns: 130,
        iterations: 100,
    });

    // 检测回归
    let regressions = detector.detect();
    detector.print_comparison();
}
```

## 6. 性能优化配置

### 6.1 优化级别配置

在 `幻语包.toml` 中配置：

```toml
[profile.release]
opt-level = 3          # 激进优化
lto = true             # 链接时优化
codegen-units = 1      # 单代码生成单元
panic = "abort"        # 移除栈展开代码
strip = true           # 移除符号表

[profile.dev]
opt-level = 0          # 无优化
debug = true           # 完整调试信息
```

### 6.2 LTO 配置

```toml
[profile.release.lto]
thin = true            # Thin LTO
jobs = 4               # 并行任务数
import-instr-limit = 100
```

### 6.3 PGO 配置

```toml
[profile.pgo]
generate = true
use = "path/to/profile.profdata"
```

## 12. 常见问题

### Q: 如何选择优化级别？

- **开发调试**：使用 `-O0`
- **快速测试**：使用 `-O1`
- **生产发布**：使用 `-O2`（默认）
- **性能关键**：使用 `-O3`
- **嵌入式/移动端**：使用 `-Os`

### Q: 如何减少二进制大小？

- 启用 `strip = true`
- 使用 `panic = "abort"`
- 启用 LTO
- 使用 `-Oz` 优化级别

### Q: 如何调试性能问题？

1. 使用 perf 生成火焰图
2. 使用基准测试定位热点
3. 使用内存追踪检查泄漏

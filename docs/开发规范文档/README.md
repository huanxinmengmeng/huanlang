## 语言特性概览

幻语是一种支持**中文、拼音、英文**三种编程形式的多范式编程语言，具有以下核心特性：

| 特性 | 说明 |
|:-----|:-----|
| **三语关键词** | 支持中文、拼音、英文关键词，可任意混用 |
| **内存安全** | 可选的所有权模型和借用检查器 |
| **结构化并发** | 内置任务、通道、同步原语支持 |
| **裸机编程** | 支持内联汇编和外设寄存器定义 |
| **AI友好** | 专为AI辅助编程设计的HLA交换格式 |
| **多后端** | 支持LLVM、WASM、GPU等多种编译目标 |

## 快速开始

### 开发环境

```bash
# 克隆仓库
git clone https://gitee.com/huanxinmengmeng/huanlang
cd huanlang

# 构建项目
cargo build --release

# 运行示例
./target/release/huan run examples/test_run.hl
```

### 贡献指南

1. 先阅读相关规范文档，了解设计要求
2. 遵循项目的代码风格与测试要求
3. 提交Pull Request时请附带测试用例

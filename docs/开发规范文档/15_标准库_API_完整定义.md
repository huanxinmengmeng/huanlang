# 15. 标准库 API 完整定义

## 15.1 标准库概述

幻语标准库（`幻语.*`）提供了跨平台的常用功能接口，涵盖基本类型操作、集合、文件 I/O、网络通信、并发、加密、序列化等。标准库 API 设计遵循以下原则：

- **一致性**：命名风格、参数顺序、错误处理方式保持一致
- **零开销抽象**：高级 API 在编译后不产生额外运行时开销
- **显式错误处理**：可能失败的操作返回 `结果<T, E>` 类型
- **中文原生**：所有公开 API 均使用中文命名（同时提供拼音/英文别名）

## 15.2 核心模块（`幻语.核心`）

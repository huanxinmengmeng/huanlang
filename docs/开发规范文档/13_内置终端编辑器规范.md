# 13. 内置终端编辑器规范

## 13.1 编辑器概述

幻语内置一个**类 Vim 的终端文本编辑器**，命名为 **“幻语编辑”** （`huan edit`），提供完整的代码编辑体验。编辑器完全运行于终端中，支持 256 色和真彩色，无需图形界面。

**核心特性**：
- **Vim 风格的模态编辑**：Normal / Insert / Visual 三种模式，支持完整的 Vim 键位映射
- **实时语法高亮**：基于 Tree-sitter 的增量解析，支持中/拼/英关键词着色
- **内置 LSP 客户端**：直接连接幻语 LSP 服务器，提供补全、诊断、跳转定义、悬停提示
- **多光标编辑**：支持多个光标同时编辑
- **中文输入法优化**：自动处理输入法组合事件，消除输入卡顿
- **文件树与导航**：侧边栏文件浏览器，快速切换文件
- **集成终端**：可嵌入终端模拟器，直接运行 `huan` 命令
- **高度可配置**：支持主题、按键映射、缩进风格等个性化配置

## 13.2 启动与命令行接口

### 13.2.1 命令格式

```bash
huan edit [文件] [选项]
```

### 13.2.2 选项

| 选项 | 说明 |
|------|------|
| `--只读` | 以只读模式打开文件 |
| `--行 <行号>` | 打开文件并跳转到指定行 |
| `--列 <列号>` | 与 `--行` 配合，跳转到指定列 |
| `--配置 <文件>` | 使用指定的配置文件（默认 `~/.huan/config.toml`） |
| `--无插件` | 禁用所有插件和 LSP 集成 |
| `--服务器 <地址>` | 连接到指定的 LSP 服务器 |

### 13.2.3 示例

```bash
# 打开文件
huan edit 程序.hl

# 打开文件并跳转到第 42 行
huan edit 程序.hl --行 42

# 以只读模式查看文件
huan edit 程序.hl --只读

# 使用自定义配置
huan edit --配置 ~/my_huan_config.toml
```

```rust
pub struct Cursor {
    pub offset: usize,   // 字符偏移
    pub line: usize,     // 行号（1-based）
    pub column: usize,   // 列号（1-based）
    pub anchor: Option<usize>,  // 选择锚点（用于可视模式）
}

impl CursorSet {
    pub fn new() -> Self {
        Self {
            cursors: vec![Cursor::default()],
            primary: 0,
        }
    }
    
    pub fn primary(&self) -> &Cursor {
        &self.cursors[self.primary]
    }
    
    pub fn primary_mut(&mut self) -> &mut Cursor {
        &mut self.cursors[self.primary]
    }
    
    pub fn all(&self) -> &[Cursor] {
        &self.cursors
    }
    
    pub fn add_cursor(&mut self, cursor: Cursor) {
        self.cursors.push(cursor);
    }
    
    pub fn remove_secondary_cursors(&mut self) {
        self.cursors.truncate(1);
        self.primary = 0;
    }
    
    pub fn sort_and_dedup(&mut self) {
        // 按偏移排序，移除重复
        self.cursors.sort_by_key(|c| c.offset);
        self.cursors.dedup_by_key(|c| c.offset);
        self.primary = 0;
    }
}
```

```rust
pub struct InputHandler {
    keymap: HashMap<Mode, HashMap<Key, Action>>,
}

impl InputHandler {
    pub fn new(config: &EditorConfig) -> Self {
        let mut handler = Self { keymap: HashMap::new() };
        handler.init_default_keymaps();
        handler.apply_custom_keymaps(&config.keymap);
        handler
    }
    
    pub fn handle_key(&self, editor: &mut Editor, key: Key) -> Result<(), EditorError> {
        let mode_keymap = self.keymap.get(&editor.mode)
            .ok_or(EditorError::NoKeymapForMode)?;
        
        if let Some(action) = mode_keymap.get(&key) {
            self.execute_action(editor, action)
        } else {
            // 未映射的按键：在插入模式下输入字符
            if editor.mode == Mode::Insert {
                if let Key::Char(ch) = key {
                    editor.insert_char(ch);
                }
            }
            Ok(())
        }
    }
    
    fn init_default_keymaps(&mut self) {
        // Normal 模式映射
        let mut normal = HashMap::new();
        normal.insert(Key::Char('h'), Action::MoveLeft);
        normal.insert(Key::Char('j'), Action::MoveDown);
        normal.insert(Key::Char('k'), Action::MoveUp);
        normal.insert(Key::Char('l'), Action::MoveRight);
        normal.insert(Key::Char('i'), Action::EnterMode(Mode::Insert));
        normal.insert(Key::Char('v'), Action::EnterMode(Mode::Visual));
        normal.insert(Key::Char('V'), Action::EnterMode(Mode::VisualLine));
        normal.insert(Key::Char('x'), Action::DeleteChar);
        normal.insert(Key::Char('d'), Action::Delete);
        normal.insert(Key::Char('y'), Action::Yank);
        normal.insert(Key::Char('p'), Action::Paste);
        normal.insert(Key::Char('u'), Action::Undo);
        normal.insert(Key::Ctrl('r'), Action::Redo);
        normal.insert(Key::Char(':'), Action::EnterMode(Mode::Command));
        normal.insert(Key::Ctrl('n'), Action::AddCursorNextMatch);
        // ... 更多映射
        self.keymap.insert(Mode::Normal, normal);
        
        // Insert 模式映射
        let mut insert = HashMap::new();
        insert.insert(Key::Escape, Action::EnterMode(Mode::Normal));

        // ... 更多映射
```

## 13.5 语法高亮器

### 13.5.1 基于 Tree-sitter 的高亮引擎

```rust
pub struct SyntaxHighlighter {
    /// Tree-sitter 解析器
    parser: tree_sitter::Parser,
    /// 幻语语言定义
    language: tree_sitter::Language,
    /// 当前语法树
    tree: Option<tree_sitter::Tree>,
    /// 主题配置
    theme: Theme,
    /// 增量解析缓存
    cache: HighlightCache,
}

impl SyntaxHighlighter {
    pub fn new() -> Self {
        let mut parser = tree_sitter::Parser::new();
        let language = tree_sitter_huan::language();
        parser.set_language(language).unwrap();
        
        Self {
            parser,
            language,
            tree: None,
            theme: Theme::default(),
            cache: HighlightCache::new(),
        }
    }
    
    /// 增量更新语法树
    pub fn update(&mut self, rope: &Rope, edits: &[Edit]) {
        let source = rope.to_string();
        if let Some(old_tree) = &mut self.tree {
            for edit in edits {
                old_tree.edit(&tree_sitter::InputEdit {
                    start_byte: edit.start,
                    old_end_byte: edit.end,
                    new_end_byte: edit.start + edit.text.len(),
                    start_position: tree_sitter::Point::new(0, 0),  // 简化
                    old_end_position: tree_sitter::Point::new(0, 0),
                    new_end_position: tree_sitter::Point::new(0, 0),
                });
            }
            self.tree = self.parser.parse(&source, Some(old_tree));
        } else {
            self.tree = self.parser.parse(&source, None);
        }
    }
    
    /// 获取指定范围的高亮信息
    pub fn highlight(&mut self, rope: &Rope, start_line: usize, end_line: usize) -> Vec<HighlightSpan> {
        if self.tree.is_none() {
            self.update(rope, &[]);
        }
        
        let tree = self.tree.as_ref().unwrap();
        let root = tree.root_node();
        
        let mut spans = Vec::new();
        self.collect_highlights(root, &mut spans, start_line, end_line);
        spans
    }
    
    fn collect_highlights(&self, node: tree_sitter::Node, spans: &mut Vec<HighlightSpan>, 
                          start_line: usize, end_line: usize) {
        let node_start = node.start_position();
        let node_end = node.end_position();
        
        if node_end.row < start_line || node_start.row > end_line {
            return;
        }
        
        if let Some(highlight) = self.get_highlight_for_node(&node) {
            spans.push(HighlightSpan {
                start_offset: node.start_byte(),
                end_offset: node.end_byte(),
                highlight,
            });
        }
        
        for child in node.children(&mut node.walk()) {
            self.collect_highlights(child, spans, start_line, end_line);
        }
    }
    
    fn get_highlight_for_node(&self, node: &tree_sitter::Node) -> Option<Highlight> {
        match node.kind() {
            "keyword" => Some(self.theme.keyword),
            "type" => Some(self.theme.type_),
            "function" => Some(self.theme.function),
            "string" => Some(self.theme.string),
            "comment" => Some(self.theme.comment),
            "number" => Some(self.theme.number),
            "operator" => Some(self.theme.operator),
            _ => None,
        }
    }
}
```

### 13.6.2 LSP 连接管理

```rust
pub struct LspConnection {
    stdin: ChildStdin,
    stdout: BufReader<ChildStdout>,
    process: Child,
    request_id: AtomicU64,
}

impl LspConnection {
    pub fn spawn(command: &str) -> Result<Self, LspError> {
        let mut parts = command.split_whitespace();
        let program = parts.next().ok_or(LspError::InvalidCommand)?;
        let args: Vec<&str> = parts.collect();
        
        let mut process = std::process::Command::new(program)
            .args(args)
            .stdin(Stdio::piped())
            .stdout(Stdio::piped())
            .stderr(Stdio::inherit())
            .spawn()
            .map_err(|e| LspError::SpawnFailed(e.to_string()))?;
        
        let stdin = process.stdin.take().unwrap();
        let stdout = BufReader::new(process.stdout.take().unwrap());
        
        Ok(Self {
            stdin,
            stdout,
            process,
            request_id: AtomicU64::new(1),
        })
    }
    
    fn next_id(&self) -> RequestId {
        RequestId::Number(self.request_id.fetch_add(1, Ordering::SeqCst))
    }
    
    pub fn request<P: Serialize, R: DeserializeOwned>(&mut self, method: &str, params: P) -> Result<R, LspError> {
        let id = self.next_id();
        let request = json!({
            "jsonrpc": "2.0",
            "id": id,
            "method": method,
            "params": params,
        });
        
        writeln!(self.stdin, "{}", request.to_string())?;
        self.stdin.flush()?;
        
        // 读取响应（简化，实际需要处理服务器推送的通知）
        let mut line = String::new();
        self.stdout.read_line(&mut line)?;
        let response: ResponseMessage = serde_json::from_str(&line)?;
        
        match response {
            ResponseMessage::Result { result, .. } => Ok(serde_json::from_value(result)?),
            ResponseMessage::Error { error, .. } => Err(LspError::ResponseError(error.message)),
        }
    }
    
    pub fn notify<P: Serialize>(&mut self, method: &str, params: P) -> Result<(), LspError> {
        let notification = json!({
            "jsonrpc": "2.0",
            "method": method,
            "params": params,
        });
        
        writeln!(self.stdin, "{}", notification.to_string())?;
        self.stdin.flush()?;
        Ok(())
    }
}
```

```rust
pub struct TerminalRenderer {
    /// 终端尺寸
    size: TerminalSize,
    /// 输出缓冲
    buffer: String,
    /// 上一帧的屏幕状态（用于增量更新）
    prev_screen: Vec<Vec<Cell>>,
}

#[derive(Debug, Clone, Copy)]
pub struct TerminalSize {
    pub width: u16,
    pub height: u16,
}

#[derive(Debug, Clone)]
struct Cell {
    ch: char,
    fg: Color,
    bg: Color,
    bold: bool,
    italic: bool,
    underline: bool,
}

impl TerminalRenderer {
    pub fn new() -> Self {
        Self {
            size: Self::get_terminal_size(),
            buffer: String::with_capacity(1024 * 1024),
            prev_screen: Vec::new(),
        }
    }
    
    fn get_terminal_size() -> TerminalSize {
        if let Some((w, h)) = term_size::dimensions() {
            TerminalSize { width: w as u16, height: h as u16 }
        } else {
            TerminalSize { width: 80, height: 24 }
        }
    }
    
    pub fn render(&mut self, editor: &Editor) {
        self.size = Self::get_terminal_size();
        self.buffer.clear();
        
        // 隐藏光标
        self.buffer.push_str("\x1b[?25l");
        
        // 渲染编辑区域
        self.render_editor_area(editor);
        
        // 渲染状态栏
        self.render_status_bar(editor);
        
        // 渲染命令面板（如果激活）
        if editor.mode == Mode::Command || editor.mode == Mode::Palette {
            self.render_command_palette(editor);
        }
        
        // 移动光标到正确位置
        self.render_cursor(editor);
        
        // 显示光标
        self.buffer.push_str("\x1b[?25h");
        
        // 输出到终端
        print!("{}", self.buffer);
        std::io::stdout().flush().unwrap();
    }
    
    fn render_editor_area(&mut self, editor: &Editor) {
        let viewport_height = (self.size.height - 2) as usize;  // 减去状态栏和命令栏
        let viewport_width = self.size.width as usize;
        
        // 计算滚动偏移
        let scroll_offset = self.calculate_scroll_offset(editor, viewport_height);
        
        for line_offset in 0..viewport_height {
            let actual_line = scroll_offset + line_offset;
            if actual_line >= editor.buffer.line_count() {
                // 空白行，显示 '~'
                self.render_line(line_offset, 0, "~", &editor.theme.line_number);
                continue;
            }
            
            // 渲染行号
            let line_num = format!("{:>4} ", actual_line + 1);
            self.render_line(line_offset, 0, &line_num, &editor.theme.line_number);
            
            // 获取行内容
            let line_text = editor.buffer.line(actual_line);
            
            // 获取语法高亮
            let highlights = editor.highlighter.highlight(&editor.buffer, actual_line, actual_line);
            
            // 渲染带高亮的文本
            self.render_highlighted_line(line_offset, 5, &line_text, &highlights, editor);
        }
    }
    
    fn render_highlighted_line(&mut self, line: usize, start_col: usize, text: &str, 
                               highlights: &[HighlightSpan], editor: &Editor) {
        let mut current_col = start_col;
        let mut current_style = Style::default();
        
        for ch in text.chars() {
            // 查找当前位置的高亮样式
            if let Some(hl) = highlights.iter().find(|h| h.contains_offset(current_offset)) {
                current_style = hl.style;
            }
            
            // 输出带样式的字符
            self.set_style(&current_style);
            self.buffer.push(ch);
            current_col += 1;
        }
        
        self.reset_style();
    }
}
```

```rust
pub struct FileTree {
    root: PathBuf,
    expanded: HashSet<PathBuf>,
    selected: Option<PathBuf>,
}

impl FileTree {
    pub fn new(root: PathBuf) -> Self {
        Self {
            root,
            expanded: HashSet::new(),
            selected: None,
        }
    }
    
    pub fn render(&self) -> Vec<String> {
        let mut lines = Vec::new();
        self.render_dir(&self.root, 0, &mut lines);
        lines
    }
    
    fn render_dir(&self, path: &Path, depth: usize, lines: &mut Vec<String>) {
        let name = path.file_name().unwrap_or_default().to_string_lossy();
        let prefix = "  ".repeat(depth);
        let expand_marker = if path.is_dir() {
            if self.expanded.contains(path) { "▼" } else { "▶" }
        } else {
            " "
        };
        lines.push(format!("{} {} {}", prefix, expand_marker, name));
        
        if path.is_dir() && self.expanded.contains(path) {
            if let Ok(entries) = std::fs::read_dir(path) {
                for entry in entries.flatten() {
                    self.render_dir(&entry.path(), depth + 1, lines);
                }
            }
        }
    }
}
```

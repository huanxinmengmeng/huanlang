# 6. 语义分析与类型系统

## 6.1 概述

语义分析阶段接收语法解析器输出的抽象语法树（AST），执行以下核心任务：

- **符号解析**：建立符号表，将标识符绑定到定义
- **类型检查与推导**：基于 Hindley-Milner 算法推导表达式类型，验证类型兼容性
- **所有权检查**（可选）：在所有权模式下，验证借用规则与生命周期
- **语义错误报告**：收集并报告类型不匹配、未定义变量、重复定义等错误

语义分析的结果是一个经过类型标注的 AST，以及一个可供后续编译阶段使用的符号表。

## 6.2 符号表（Symbol Table）

符号表管理程序中所有声明的符号（变量、函数、类型等），并维护作用域嵌套关系。

### 6.2.1 符号定义

```rust
#[derive(Debug, Clone, PartialEq)]
pub struct Symbol {
    /// 符号名称
    pub name: String,
    /// 符号种类
    pub kind: SymbolKind,
    /// 符号类型（对于变量/函数）
    pub ty: Type,
    /// 是否可变（对于变量）
    pub mutable: bool,
    /// 可见性
    pub visibility: Visibility,
    /// 定义所在的源码范围
    pub span: SourceSpan,
    /// 定义所在的作用域深度（用于快速比较）
    pub scope_depth: usize,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum SymbolKind {
    Variable,       // 局部变量 / 全局变量
    Constant,       // 常量
    Function,       // 函数
    Struct,         // 结构体类型
    Trait,          // 特征
    Type,           // 类型别名
    Module,         // 模块
    GenericParam,   // 泛型参数
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum Visibility {
    Private,            // 模块内可见
    Public,             // 公开可见
    Exported(String),   // 导出为指定名称（用于 FFI）
}
```

### 6.2.3 符号表主结构

```rust
pub struct SymbolTable {
    /// 作用域栈，索引 0 为全局作用域
    scopes: Vec<Scope>,
    /// 当前作用域索引
    current_scope: usize,
    /// 模块路径（用于完全限定名）
    module_path: Vec<String>,
    /// 导入的符号映射（名称 → 完全限定名）
    imports: HashMap<String, String>,
}

impl SymbolTable {
    pub fn new() -> Self {
        let global = Scope::new(None, 0, ScopeNode::Module);
        Self {
            scopes: vec![global],
            current_scope: 0,
            module_path: Vec::new(),
            imports: HashMap::new(),
        }
    }

    /// 进入新作用域
    pub fn enter_scope(&mut self, node: ScopeNode) {
        let depth = self.scopes[self.current_scope].depth + 1;
        let scope = Scope::new(Some(self.current_scope), depth, node);
        self.scopes.push(scope);
        self.current_scope = self.scopes.len() - 1;
    }

    /// 离开当前作用域
    pub fn exit_scope(&mut self) {
        if let Some(parent) = self.scopes[self.current_scope].parent {
            self.current_scope = parent;
        }
    }

    /// 在当前作用域定义符号
    pub fn define(
        &mut self,
        name: String,
        kind: SymbolKind,
        ty: Type,
        mutable: bool,
        visibility: Visibility,
        span: SourceSpan,
    ) -> Result<(), SemanticError> {
        let scope = &mut self.scopes[self.current_scope];
        if scope.symbols.contains_key(&name) {
            let existing = scope.symbols.get(&name).unwrap();
            return Err(SemanticError::DuplicateDefinition {
                name,
                first: existing.span,
                second: span,
            });
        }
        let depth = scope.depth;
        scope.symbols.insert(name.clone(), Symbol {
            name,
            kind,
            ty,
            mutable,
            visibility,
            span,
            scope_depth: depth,
        });
        Ok(())
    }

    /// 从当前作用域开始向外解析符号
    pub fn resolve(&self, name: &str) -> Option<&Symbol> {
        // 先检查导入
        if let Some(full_name) = self.imports.get(name) {
            // 简化：导入解析需要完全限定名查找
            // 实际实现需维护导入的符号映射
            return self.resolve_fully_qualified(full_name);
        }

        let mut scope_idx = self.current_scope;
        loop {
            let scope = &self.scopes[scope_idx];
            if let Some(sym) = scope.symbols.get(name) {
                return Some(sym);
            }
            if let Some(parent) = scope.parent {
                scope_idx = parent;
            } else {
                break;
            }
        }
        None
    }

    /// 可修改地解析符号
    pub fn resolve_mut(&mut self, name: &str) -> Option<&mut Symbol> {
        let mut scope_idx = self.current_scope;
        loop {
            let scope = &mut self.scopes[scope_idx];
            if scope.symbols.contains_key(name) {
                return scope.symbols.get_mut(name);
            }
            if let Some(parent) = scope.parent {
                scope_idx = parent;
            } else {
                break;
            }
        }
        None
    }

    /// 解析完全限定名（如 `模块::子模块::符号`）
    pub fn resolve_fully_qualified(&self, path: &str) -> Option<&Symbol> {
        // 实际实现需维护模块树
        None
    }

    /// 获取当前作用域深度
    pub fn current_depth(&self) -> usize {
        self.scopes[self.current_scope].depth
    }

    /// 获取所有作用域（用于调试）
    pub fn all_scopes(&self) -> &[Scope] {
        &self.scopes
    }
}
```

## 6.3 类型推导器（TypeInfer）

类型推导器基于 Hindley-Milner 算法，支持泛型、类型变量和 let 多态。

### 6.3.1 类型推导器结构

### 6.3.2 类型变量管理

```rust
impl TypeInfer {
    /// 生成新的类型变量
    fn fresh_var(&mut self) -> Type {
        let var = self.next_var;
        self.next_var += 1;
        Type::Var(var)
    }

    /// 生成指定数量的新类型变量
    fn fresh_vars(&mut self, count: usize) -> Vec<Type> {
```

### 6.3.3 合一（Unification）算法

```rust
impl TypeInfer {
    /// 对类型应用当前替换
    fn apply_subst(&self, ty: &Type) -> Type {
        match ty {
            Type::Var(v) => {
                if let Some(t) = self.substitutions.get(v) {
                    self.apply_subst(t)
                } else {
                    ty.clone()
                }
            }
            Type::List(inner) => Type::List(Box::new(self.apply_subst(inner))),
            Type::Array(inner, size) => Type::Array(Box::new(self.apply_subst(inner)), size.clone()),
            Type::Map(k, v) => Type::Map(
                Box::new(self.apply_subst(k)),
                Box::new(self.apply_subst(v)),
            ),
            Type::Ptr(inner) => Type::Ptr(Box::new(self.apply_subst(inner))),
            Type::Option(inner) => Type::Option(Box::new(self.apply_subst(inner))),
            Type::Func(params, ret) => Type::Func(
                params.iter().map(|t| self.apply_subst(t)).collect(),
                Box::new(self.apply_subst(ret)),
            ),
            _ => ty.clone(),
        }
    }

    /// 检查类型变量是否出现在某类型中（发生检查）
    fn occurs(&self, var: usize, ty: &Type) -> bool {
        match ty {
            Type::Var(v) => *v == var,
            Type::List(inner) => self.occurs(var, inner),
            Type::Array(inner, _) => self.occurs(var, inner),
            Type::Map(k, v) => self.occurs(var, k) || self.occurs(var, v),
            Type::Ptr(inner) => self.occurs(var, inner),
            Type::Option(inner) => self.occurs(var, inner),
            Type::Func(params, ret) => {
                params.iter().any(|t| self.occurs(var, t)) || self.occurs(var, ret)
            }
            _ => false,
        }
    }

    /// 合一两个类型
    fn unify(&mut self, t1: Type, t2: Type) -> Result<(), TypeError> {
        let t1 = self.apply_subst(&t1);
        let t2 = self.apply_subst(&t2);

        match (t1, t2) {
            (Type::Var(v1), Type::Var(v2)) if v1 == v2 => Ok(()),
            (Type::Var(v), t) => {
                if self.occurs(v, &t) {
                    Err(TypeError::Cyclic { span: SourceSpan::dummy() })
                } else {
                    self.substitutions.insert(v, t);
                    Ok(())
                }
            }
            (t, Type::Var(v)) => {
                if self.occurs(v, &t) {
                    Err(TypeError::Cyclic { span: SourceSpan::dummy() })
                } else {
                    self.substitutions.insert(v, t);
                    Ok(())
                }
            }
            (Type::Int, Type::Int) => Ok(()),
            (Type::I8, Type::I8) => Ok(()),
            (Type::I16, Type::I16) => Ok(()),
            (Type::I32, Type::I32) => Ok(()),
            (Type::I64, Type::I64) => Ok(()),
            (Type::U8, Type::U8) => Ok(()),
            (Type::U16, Type::U16) => Ok(()),
            (Type::U32, Type::U32) => Ok(()),
            (Type::U64, Type::U64) => Ok(()),
            (Type::F32, Type::F32) => Ok(()),
            (Type::F64, Type::F64) => Ok(()),
            (Type::Bool, Type::Bool) => Ok(()),
            (Type::Char, Type::Char) => Ok(()),
            (Type::String, Type::String) => Ok(()),
            (Type::Unit, Type::Unit) => Ok(()),

            (Type::List(a), Type::List(b)) => self.unify(*a, *b),
            (Type::Array(a, s1), Type::Array(b, s2)) => {
                // 大小也需要相等（常量求值后）
                self.unify(*a, *b)
            }
            (Type::Map(k1, v1), Type::Map(k2, v2)) => {
                self.unify(*k1, *k2)?;
                self.unify(*v1, *v2)
            }
            (Type::Ptr(a), Type::Ptr(b)) => self.unify(*a, *b),
            (Type::Option(a), Type::Option(b)) => self.unify(*a, *b),
            (Type::Func(p1, r1), Type::Func(p2, r2)) => {
                if p1.len() != p2.len() {
                    return Err(TypeError::Mismatch {
                        expected: Type::Func(p1, r1),
                        found: Type::Func(p2, r2),
                        span: SourceSpan::dummy(),
                    });
                }
                for (a, b) in p1.into_iter().zip(p2) {
                    self.unify(a, b)?;
                }
                self.unify(*r1, *r2)
            }
            (Type::Named(p1), Type::Named(p2)) if p1 == p2 => Ok(()),
            (a, b) => Err(TypeError::Mismatch {
                expected: a,
                found: b,
                span: SourceSpan::dummy(),
            }),
        }
    }
}
```

### 6.3.4 表达式类型推导

```rust
impl TypeInfer {
    /// 推导表达式的类型
    pub fn infer_expr(&mut self, expr: &Expr) -> Result<Type, TypeError> {
        let span = expr.span();
        match expr {
            Expr::IntLit(_, _) => Ok(Type::Int),
            Expr::FloatLit(_, _) => Ok(Type::F64),
            Expr::StringLit(_, _) => Ok(Type::String),
            Expr::CharLit(_, _) => Ok(Type::Char),
            Expr::BoolLit(_, _) => Ok(Type::Bool),
            Expr::Null(_) => Ok(self.fresh_var()), // 可赋值给任意可选类型

            Expr::Ident(ident) => {
                self.lookup_type(&ident.name)
                    .cloned()
                    .ok_or_else(|| TypeError::UnboundVariable {
                        name: ident.name.clone(),
                        span: ident.span,
                    })
            }

            Expr::BinaryOp { op, left, right, span } => {
                let t_left = self.infer_expr(left)?;
                let t_right = self.infer_expr(right)?;

                match op {
                    BinaryOp::Add | BinaryOp::Sub | BinaryOp::Mul | BinaryOp::Div | BinaryOp::Mod => {
                        // 算术运算要求左右类型一致
                        self.unify(t_left.clone(), t_right.clone())
                            .map_err(|e| e.with_span(*span))?;
                        Ok(t_left)
                    }
                    BinaryOp::And | BinaryOp::Or => {
                        self.unify(t_left, Type::Bool).map_err(|e| e.with_span(*span))?;
                        self.unify(t_right, Type::Bool).map_err(|e| e.with_span(*span))?;
                        Ok(Type::Bool)
                    }
                    BinaryOp::Eq | BinaryOp::Ne | BinaryOp::Gt | BinaryOp::Lt
                    | BinaryOp::Ge | BinaryOp::Le => {
                        self.unify(t_left, t_right).map_err(|e| e.with_span(*span))?;
                        Ok(Type::Bool)
                    }
                    BinaryOp::Shl | BinaryOp::Shr | BinaryOp::BitAnd
                    | BinaryOp::BitOr | BinaryOp::BitXor => {
                        // 位运算要求整数类型
                        if !t_left.is_integer() || !t_right.is_integer() {
                            return Err(TypeError::InvalidOperation {
                                op: format!("{:?}", op),
                                left: t_left,
                                right: t_right,
                                span: *span,
                            });
                        }
                        self.unify(t_left.clone(), t_right).map_err(|e| e.with_span(*span))?;
                        Ok(t_left)
                    }
                    BinaryOp::Assign => {
                        self.unify(t_left, t_right).map_err(|e| e.with_span(*span))?;
                        Ok(Type::Unit)
                    }
                    _ => {
                        // 复合赋值等
                        self.unify(t_left.clone(), t_right).map_err(|e| e.with_span(*span))?;
                        Ok(Type::Unit)
                    }
                }
            }

            Expr::UnaryOp { op, expr, span } => {
                let t_expr = self.infer_expr(expr)?;
                match op {
                    UnaryOp::Not => {
                        self.unify(t_expr, Type::Bool).map_err(|e| e.with_span(*span))?;
                        Ok(Type::Bool)
                    }
                    UnaryOp::Neg => {
                        if !t_expr.is_numeric() {
                            return Err(TypeError::InvalidUnaryOperation {
                                op: "负号".to_string(),
                                ty: t_expr,
                                span: *span,
                            });
                        }
                        Ok(t_expr)
                    }
                    UnaryOp::BitNot => {
                        if !t_expr.is_integer() {
                            return Err(TypeError::InvalidUnaryOperation {
                                op: "按位非".to_string(),
                                ty: t_expr,
                                span: *span,
                            });
                        }
                        Ok(t_expr)
                    }
                }
            }

            Expr::Call { func, args, span } => {
                let t_func = self.infer_expr(func)?;
                match t_func {
                    Type::Func(param_tys, ret_ty) => {
                        if param_tys.len() != args.len() {
                            return Err(TypeError::ArgCountMismatch {
                                expected: param_tys.len(),
                                found: args.len(),
                                span: *span,
                            });
                        }
                        for (arg, expected) in args.iter().zip(param_tys.iter()) {
                            let t_arg = self.infer_expr(arg)?;
                            self.unify(t_arg, expected.clone()).map_err(|e| e.with_span(arg.span()))?;
                        }
                        Ok(*ret_ty)
                    }
                    _ => Err(TypeError::NotCallable {
                        ty: t_func,
                        span: func.span(),
                    }),
                }
            }

            Expr::MethodCall { receiver, method, args, span } => {
                // 方法调用需要查找 receiver 类型的方法表
                let t_receiver = self.infer_expr(receiver)?;
                self.resolve_method(&t_receiver, &method.name, args, *span)
            }

            Expr::Field { target, field, span } => {
                let t_target = self.infer_expr(target)?;
                self.resolve_field(&t_target, &field.name, *span)
            }

            Expr::Index { target, index, span } => {
                let t_target = self.infer_expr(target)?;
                let t_index = self.infer_expr(index)?;
                self.unify(t_index, Type::Int).map_err(|e| e.with_span(index.span()))?;

                match t_target {
                    Type::List(inner) => Ok(*inner),
                    Type::Array(inner, _) => Ok(*inner),
                    Type::Map(k, v) => {
                        // 索引类型需与键类型一致
                        self.unify(t_index, *k).map_err(|e| e.with_span(*span))?;
                        Ok(*v)
                    }
                    Type::String => Ok(Type::Char),
                    _ => Err(TypeError::NotIndexable {
                        ty: t_target,
                        span: *span,
                    }),
                }
            }

            Expr::List(elements, span) => {
                if elements.is_empty() {
                    Ok(Type::List(Box::new(self.fresh_var())))
                } else {
                    let mut elem_ty = self.infer_expr(&elements[0])?;
                    for elem in &elements[1..] {
                        let t = self.infer_expr(elem)?;
                        self.unify(elem_ty.clone(), t).map_err(|e| e.with_span(elem.span()))?;
                    }
                    Ok(Type::List(Box::new(elem_ty)))
                }
            }

            Expr::Map(pairs, span) => {
                if pairs.is_empty() {
                    Ok(Type::Map(Box::new(self.fresh_var()), Box::new(self.fresh_var())))
                } else {
                    let mut key_ty = self.infer_expr(&pairs[0].0)?;
                    let mut val_ty = self.infer_expr(&pairs[0].1)?;
                    for (k, v) in &pairs[1..] {
                        let tk = self.infer_expr(k)?;
                        let tv = self.infer_expr(v)?;
                        self.unify(key_ty.clone(), tk).map_err(|e| e.with_span(k.span()))?;
                        self.unify(val_ty.clone(), tv).map_err(|e| e.with_span(v.span()))?;
                    }
                    Ok(Type::Map(Box::new(key_ty), Box::new(val_ty)))
                }
            }

            Expr::Struct { path, fields, span } => {
                // 查找结构体定义，验证字段类型
                let struct_name = path.segments.last().unwrap().name.clone();
                let struct_def = self.symbol_table.resolve(&struct_name)
                    .ok_or_else(|| TypeError::UndefinedType {
                        name: struct_name,
                        span: path.span,
                    })?;

                // 实际实现需展开结构体定义，检查字段
                // 这里简化为返回命名类型
                Ok(Type::Named(path.clone()))
            }

            Expr::Closure { params, return_type, body, span } => {
                self.env.push(HashMap::new());
                let mut param_tys = Vec::new();
                for (name, anno) in params {
                    let ty = anno.clone().unwrap_or_else(|| self.fresh_var());
                    self.env.last_mut().unwrap().insert(name.name.clone(), ty.clone());
                    param_tys.push(ty);
                }

                let saved_return = self.current_return_type.clone();
                let ret_ty = return_type.clone().unwrap_or_else(|| self.fresh_var());
                self.current_return_type = Some(ret_ty.clone());

                for stmt in body {
                    self.infer_stmt(stmt)?;
                }

                self.current_return_type = saved_return;
                self.env.pop();

                Ok(Type::Func(param_tys, Box::new(ret_ty)))
            }

            Expr::IfExpr { cond, then_expr, else_expr, span } => {
                let t_cond = self.infer_expr(cond)?;
                self.unify(t_cond, Type::Bool).map_err(|e| e.with_span(cond.span()))?;

                let t_then = self.infer_expr(then_expr)?;
                let t_else = self.infer_expr(else_expr)?;

                self.unify(t_then.clone(), t_else).map_err(|e| e.with_span(*span))?;
                Ok(t_then)
            }

            Expr::Match { expr, arms, default, span } => {
                let t_expr = self.infer_expr(expr)?;
                let result_ty = self.fresh_var();

                for (pattern, arm_expr) in arms {
                    self.check_pattern(pattern, &t_expr)?;
                    self.env.push(HashMap::new());
                    self.bind_pattern(pattern, &t_expr)?;
                    let t_arm = self.infer_expr(arm_expr)?;
                    self.unify(result_ty.clone(), t_arm).map_err(|e| e.with_span(arm_expr.span()))?;
                    self.env.pop();
                }

                if let Some(default_expr) = default {
                    let t_default = self.infer_expr(default_expr)?;
                    self.unify(result_ty.clone(), t_default).map_err(|e| e.with_span(default_expr.span()))?;
                }

                Ok(result_ty)
            }

            Expr::Try { expr, span } => {
                let t_expr = self.infer_expr(expr)?;
                match t_expr {
                    Type::Option(inner) => Ok(*inner),
                    Type::Result(ok, _) => Ok(*ok),
                    _ => Err(TypeError::InvalidTry {
                        ty: t_expr,
                        span: *span,
                    }),
                }
            }

            Expr::TypeAssertion { expr, ty, span } => {
                let t_expr = self.infer_expr(expr)?;
                // 检查是否可以转换
                self.can_cast(&t_expr, ty).map_err(|e| e.with_span(*span))?;
                Ok(ty.clone())
            }

            Expr::Asm(asm) => {
                // 内联汇编的类型推导：根据输出约束推断
                self.infer_asm(asm)
            }
        }
    }

    /// 查找变量类型
    fn lookup_type(&self, name: &str) -> Option<&Type> {
        for scope in self.env.iter().rev() {
            if let Some(ty) = scope.get(name) {
                return Some(ty);
            }
        }
        None
    }

    /// 解析字段类型
    fn resolve_field(&mut self, ty: &Type, field: &str, span: SourceSpan) -> Result<Type, TypeError> {
        match ty {
            Type::Named(path) => {
                let struct_name = path.segments.last().unwrap().name.clone();
                // 查找结构体定义并返回字段类型
                // 实际实现需访问结构体定义表
                Err(TypeError::UndefinedField {
                    ty: ty.clone(),
                    field: field.to_string(),
                    span,
                })
            }
            _ => Err(TypeError::NoField {
                ty: ty.clone(),
                span,
            }),
        }
    }

    /// 解析方法调用类型
    fn resolve_method(&mut self, receiver: &Type, method: &str, args: &[Expr], span: SourceSpan) -> Result<Type, TypeError> {
        // 实际实现需查找方法表
        Err(TypeError::UndefinedMethod {
            ty: receiver.clone(),
            method: method.to_string(),
            span,
        })
    }

    /// 检查模式是否与类型匹配
    fn check_pattern(&mut self, pattern: &Pattern, ty: &Type) -> Result<(), TypeError> {
        match pattern {
            Pattern::Wildcard(_) => Ok(()),
            Pattern::Ident(_) => Ok(()),
            Pattern::Literal(lit) => {
                let t_lit = self.infer_expr(lit)?;
                self.unify(t_lit, ty.clone()).map_err(|e| e.with_span(pattern.span()))
            }
            Pattern::Struct { path, fields, span } => {
                // 检查结构体模式
                Ok(())
            }
            Pattern::List(patterns, span) => {
                match ty {
                    Type::List(inner) => {
                        for p in patterns {
                            self.check_pattern(p, inner)?;
                        }
                        Ok(())
                    }
                    _ => Err(TypeError::PatternMismatch {
                        pattern: format!("{:?}", pattern),
                        ty: ty.clone(),
                        span: *span,
                    }),
                }
            }
            Pattern::Or(p1, p2, span) => {
                self.check_pattern(p1, ty)?;
                self.check_pattern(p2, ty)
            }
        }
    }

    /// 将模式中的变量绑定到当前环境
    fn bind_pattern(&mut self, pattern: &Pattern, ty: &Type) -> Result<(), TypeError> {
        match pattern {
            Pattern::Ident(ident) => {
                self.env.last_mut().unwrap().insert(ident.name.clone(), ty.clone());
                Ok(())
            }
            Pattern::Struct { fields, .. } => {
                for (field, sub_pattern) in fields {
                    let field_ty = self.resolve_field(ty, &field.name, field.span)?;
                    self.bind_pattern(sub_pattern, &field_ty)?;
                }
                Ok(())
            }
            Pattern::List(patterns, _) => {
                match ty {
                    Type::List(inner) => {
                        for p in patterns {
                            self.bind_pattern(p, inner)?;
                        }
                        Ok(())
                    }
                    _ => Ok(()),
                }
            }
            _ => Ok(()),
        }
    }

    /// 检查类型转换是否合法
    fn can_cast(&self, from: &Type, to: &Type) -> Result<(), TypeError> {
        // 基本数值类型之间允许转换
        if from.is_numeric() && to.is_numeric() {
            return Ok(());
        }
        // 指针之间允许转换
        if matches!(from, Type::Ptr(_)) && matches!(to, Type::Ptr(_)) {
            return Ok(());
        }
        Err(TypeError::InvalidCast {
            from: from.clone(),
            to: to.clone(),
        })
    }

    /// 推导内联汇编的类型
    fn infer_asm(&mut self, asm: &InlineAsm) -> Result<Type, TypeError> {
        // 根据输出操作数推导
        if asm.outputs.is_empty() {
            Ok(Type::Unit)
        } else {
            // 简化：返回第一个输出操作数的类型
            self.infer_expr(&asm.outputs[0].expr)
        }
    }
}
```

```rust
impl TypeInfer {
    /// 检查语句的类型正确性
    pub fn infer_stmt(&mut self, stmt: &Stmt) -> Result<(), TypeError> {
        match stmt {
            Stmt::Let { name, ty, value, span } => {
                let t_value = self.infer_expr(value)?;
                if let Some(anno) = ty {
                    self.unify(t_value.clone(), anno.clone()).map_err(|e| e.with_span(*span))?;
                }
                self.env.last_mut().unwrap().insert(name.name.clone(), ty.clone().unwrap_or(t_value));
                Ok(())
            }

            Stmt::Const { name, ty, value, span } => {
                let t_value = self.infer_expr(value)?;
                if let Some(anno) = ty {
                    self.unify(t_value.clone(), anno.clone()).map_err(|e| e.with_span(*span))?;
                }
                self.env.last_mut().unwrap().insert(name.name.clone(), ty.clone().unwrap_or(t_value));
                Ok(())
            }

            Stmt::Assign { target, value, span } => {
                let t_target = self.infer_expr(target)?;
                let t_value = self.infer_expr(value)?;
                self.unify(t_target, t_value).map_err(|e| e.with_span(*span))
            }

            Stmt::If { cond, then_block, else_ifs, else_block, span } => {
                let t_cond = self.infer_expr(cond)?;
                self.unify(t_cond, Type::Bool).map_err(|e| e.with_span(cond.span()))?;

                self.env.push(HashMap::new());
                for s in then_block { self.infer_stmt(s)?; }
                self.env.pop();

                for (cond, block) in else_ifs {
                    let t_cond = self.infer_expr(cond)?;
                    self.unify(t_cond, Type::Bool).map_err(|e| e.with_span(cond.span()))?;
                    self.env.push(HashMap::new());
                    for s in block { self.infer_stmt(s)?; }
                    self.env.pop();
                }

                if let Some(block) = else_block {
                    self.env.push(HashMap::new());
                    for s in block { self.infer_stmt(s)?; }
                    self.env.pop();
                }
                Ok(())
            }

            Stmt::Match { expr, arms, default, span } => {
                let t_expr = self.infer_expr(expr)?;

                for (pattern, block) in arms {
                    self.check_pattern(pattern, &t_expr)?;
                    self.env.push(HashMap::new());
                    self.bind_pattern(pattern, &t_expr)?;
                    for s in block { self.infer_stmt(s)?; }
                    self.env.pop();
                }

                if let Some(block) = default {
                    self.env.push(HashMap::new());
                    for s in block { self.infer_stmt(s)?; }
                    self.env.pop();
                }
                Ok(())
            }

            Stmt::Repeat { count, body, span } => {
                let t_count = self.infer_expr(count)?;
                self.unify(t_count, Type::Int).map_err(|e| e.with_span(count.span()))?;

                self.env.push(HashMap::new());
                for s in body { self.infer_stmt(s)?; }
                self.env.pop();
                Ok(())
            }

            Stmt::While { cond, body, span } => {
                let t_cond = self.infer_expr(cond)?;
                self.unify(t_cond, Type::Bool).map_err(|e| e.with_span(cond.span()))?;

                self.env.push(HashMap::new());
                for s in body { self.infer_stmt(s)?; }
                self.env.pop();
                Ok(())
            }

            Stmt::ForEach { var, iterable, body, span } => {
                let t_iter = self.infer_expr(iterable)?;
                let elem_ty = match t_iter {
                    Type::List(inner) => *inner,
                    Type::Array(inner, _) => *inner,
                    Type::String => Type::Char,
                    _ => return Err(TypeError::NotIterable {
                        ty: t_iter,
                        span: iterable.span(),
                    }),
                };

                self.env.push(HashMap::new());
                self.env.last_mut().unwrap().insert(var.name.clone(), elem_ty);
                for s in body { self.infer_stmt(s)?; }
                self.env.pop();
                Ok(())
            }

            Stmt::Return(expr, span) => {
                if let Some(expected) = &self.current_return_type {
                    if let Some(expr) = expr {
                        let t_expr = self.infer_expr(expr)?;
                        self.unify(t_expr, expected.clone()).map_err(|e| e.with_span(*span))?;
                    } else {
                        self.unify(Type::Unit, expected.clone()).map_err(|e| e.with_span(*span))?;
                    }
                } else {
                    return Err(TypeError::ReturnOutsideFunction { span: *span });
                }
                Ok(())
            }

            Stmt::Break(_) | Stmt::Continue(_) => {
                // 需在循环上下文中检查
                Ok(())
            }

            Stmt::Asm(asm, span) => {
                self.infer_asm(asm)?;
                Ok(())
            }

            Stmt::Expr(expr, _) => {
                self.infer_expr(expr)?;
                Ok(())
            }
        }
    }
}
```

### 6.3.6 顶层项类型检查

```rust
impl TypeInfer {
    /// 检查整个程序
    pub fn infer_program(&mut self, program: &Program) -> Result<(), Vec<TypeError>> {
        let mut errors = Vec::new();

        // 第一遍：收集所有顶层定义到符号表
        for item in program {
            if let Err(e) = self.collect_top_level(item) {
                errors.push(e);
            }
        }

        // 第二遍：类型检查每个项
        for item in program {
            if let Err(e) = self.check_item(item) {
                errors.push(e);
            }
        }

        if errors.is_empty() {
            Ok(())
        } else {
            Err(errors)
        }
    }

    fn collect_top_level(&mut self, item: &Item) -> Result<(), TypeError> {
        match item {
            Item::Function(func) => {
                let ty = Type::Func(
                    func.params.iter().map(|(_, t)| t.clone()).collect(),
                    Box::new(func.return_type.clone()),
                );
                self.symbol_table.define(
                    func.name.name.clone(),
                    SymbolKind::Function,
                    ty,
                    false,
                    if func.public { Visibility::Public } else { Visibility::Private },
                    func.span,
                ).map_err(|e| TypeError::SemanticError(e))
            }
            Item::Struct(s) => {
                self.symbol_table.define(
                    s.name.name.clone(),
                    SymbolKind::Struct,
                    Type::Unit, // 占位
                    false,
                    if s.public { Visibility::Public } else { Visibility::Private },
                    s.span,
                ).map_err(|e| TypeError::SemanticError(e))
            }
            Item::Global(g) => {
                self.symbol_table.define(
                    g.name.name.clone(),
                    if g.mutable { SymbolKind::Variable } else { SymbolKind::Constant },
                    g.ty.clone().unwrap_or(Type::Var(self.next_var)),
                    g.mutable,
                    Visibility::Private,
                    g.span,
                ).map_err(|e| TypeError::SemanticError(e))
            }
            // 其他项类似处理
            _ => Ok(()),
        }
    }

    fn check_item(&mut self, item: &Item) -> Result<(), TypeError> {
        match item {
            Item::Function(func) => self.check_function(func),
            Item::Struct(s) => self.check_struct(s),
            Item::Impl(imp) => self.check_impl(imp),
            Item::Global(g) => {
                let t_value = self.infer_expr(&g.value)?;
                if let Some(anno) = &g.ty {
                    self.unify(t_value, anno.clone()).map_err(|e| e.with_span(g.span))?;
                }
                Ok(())
            }
            _ => Ok(()),
        }
    }

    fn check_function(&mut self, func: &Function) -> Result<(), TypeError> {
        self.env.push(HashMap::new());
        self.current_return_type = Some(func.return_type.clone());

        for (name, ty) in &func.params {
            self.env.last_mut().unwrap().insert(name.name.clone(), ty.clone());
        }

        for stmt in &func.body {
            self.infer_stmt(stmt)?;
        }

        // 检查函数体是否以 return 或隐式返回结尾
        // （具体规则：最后一条语句若不是 return，且返回类型非 Unit，则报错）

        self.env.pop();
        self.current_return_type = None;
        Ok(())
    }

    fn check_struct(&mut self, s: &Struct) -> Result<(), TypeError> {
        for (_, ty, _) in &s.fields {
            // 验证字段类型是否已定义
            self.validate_type(ty)?;
        }
        Ok(())
    }

    fn check_impl(&mut self, imp: &Impl) -> Result<(), TypeError> {
        for method in &imp.methods {
            self.check_function(method)?;
        }
        Ok(())
    }

    fn validate_type(&self, ty: &Type) -> Result<(), TypeError> {
        match ty {
            Type::Named(path) => {
                let name = path.segments.last().unwrap().name.clone();
                if self.symbol_table.resolve(&name).is_none() {
                    return Err(TypeError::UndefinedType {
                        name,
                        span: path.span,
                    });
                }
                Ok(())
            }
            Type::List(inner) => self.validate_type(inner),
            Type::Array(inner, _) => self.validate_type(inner),
            Type::Map(k, v) => {
                self.validate_type(k)?;
                self.validate_type(v)
            }
            Type::Ptr(inner) => self.validate_type(inner),
            Type::Option(inner) => self.validate_type(inner),
            Type::Func(params, ret) => {
                for p in params { self.validate_type(p)?; }
                self.validate_type(ret)
            }
            _ => Ok(()),
        }
    }
}
```

## 6.5 所有权检查器（可选）

所有权检查器在启用所有权模式时运行，验证借用规则。

```rust
pub struct BorrowChecker {
    /// 当前活跃的借用（内存位置 → 借用状态）
    loans: HashMap<MemoryLocation, Loan>,
    /// 符号表（用于获取变量信息）
    symbol_table: SymbolTable,
    /// 当前作用域深度
    scope_depth: usize,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
struct MemoryLocation {
    var_id: usize,  // 变量的唯一标识
}

#[derive(Debug, Clone)]
enum Loan {
    Shared(usize),  // 共享借用计数
    Mutable,        // 可变借用（只能有一个）
}

impl BorrowChecker {
    pub fn new(symbol_table: SymbolTable) -> Self {
        Self {
            loans: HashMap::new(),
            symbol_table,
            scope_depth: 0,
        }
    }

    pub fn check_function(&mut self, func: &Function) -> Result<(), BorrowError> {
        self.scope_depth = 0;
        for stmt in &func.body {
            self.check_stmt(stmt)?;
        }
        Ok(())
    }

    fn check_stmt(&mut self, stmt: &Stmt) -> Result<(), BorrowError> {
        match stmt {
            Stmt::Let { name, value, span, .. } => {
                self.check_expr(value)?;
                // 记录新变量
                Ok(())
            }
            Stmt::Assign { target, value, span } => {
                self.check_expr(target)?;
                self.check_expr(value)?;
                Ok(())
            }
            Stmt::Expr(expr, _) => {
                self.check_expr(expr)?;
                Ok(())
            }
            // 其他语句递归检查
            _ => Ok(()),
        }
    }

    fn check_expr(&mut self, expr: &Expr) -> Result<(), BorrowError> {
        match expr {
            Expr::Ident(ident) => {
                // 检查变量是否可读
                if let Some(sym) = self.symbol_table.resolve(&ident.name) {
                    if !sym.mutable {
                        // 不可变变量，无需借用检查
                        return Ok(());
                    }
                    // 检查当前借用状态
                    let loc = MemoryLocation { var_id: 0 }; // 实际需获取变量 ID
                    if let Some(Loan::Mutable) = self.loans.get(&loc) {
                        return Err(BorrowError::CannotBorrowWhileMutable {
                            name: ident.name.clone(),
                            span: ident.span,
                        });
                    }
                }
                Ok(())
            }
            Expr::BinaryOp { op, left, right, .. } => {
                self.check_expr(left)?;
                self.check_expr(right)?;
                Ok(())
            }
            Expr::Call { func, args, .. } => {
                self.check_expr(func)?;
                for arg in args {
                    self.check_expr(arg)?;
                }
                Ok(())
            }
            Expr::MethodCall { receiver, args, .. } => {
                self.check_expr(receiver)?;
                for arg in args {
                    self.check_expr(arg)?;
                }
                Ok(())
            }
            Expr::Field { target, .. } => {
                self.check_expr(target)?;
                Ok(())
            }
            // 其他表达式递归检查
            _ => Ok(()),
        }
    }
}

#[derive(Debug, Clone, PartialEq)]
pub enum BorrowError {
    UseAfterMove {
        name: String,
        span: SourceSpan,
    },
    CannotBorrowWhileMutable {
        name: String,
        span: SourceSpan,
    },
    CannotBorrowMutableWhileShared {
        name: String,
        span: SourceSpan,
    },
    LifetimeMismatch {
        span: SourceSpan,
    },
}
```

## 6.6 语义分析驱动

```rust
pub struct SemanticAnalyzer {
    symbol_table: SymbolTable,
    type_infer: TypeInfer,
    borrow_checker: Option<BorrowChecker>,
    errors: Vec<SemanticError>,
}

impl SemanticAnalyzer {
    pub fn new(enable_ownership: bool) -> Self {
        let symbol_table = SymbolTable::new();
        let type_infer = TypeInfer::new(symbol_table.clone());
        let borrow_checker = if enable_ownership {
            Some(BorrowChecker::new(symbol_table.clone()))
        } else {
            None
        };
        Self {
            symbol_table,
            type_infer,
            borrow_checker,
            errors: Vec::new(),
        }
    }

    pub fn analyze(&mut self, program: &Program) -> Result<SymbolTable, Vec<SemanticError>> {
        // 类型检查
        if let Err(errs) = self.type_infer.infer_program(program) {
            self.errors.extend(errs.into_iter().map(|e| SemanticError::TypeError(e)));
        }

        // 所有权检查（如果启用）
        if let Some(checker) = &mut self.borrow_checker {
            for item in program {
                if let Item::Function(func) = item {
                    if let Err(e) = checker.check_function(func) {
                        self.errors.push(SemanticError::BorrowError(e));
                    }
                }
            }
        }

        if self.errors.is_empty() {
            Ok(self.symbol_table.clone())
        } else {
            Err(std::mem::take(&mut self.errors))
        }
    }
}
```

# 6. 语义分析与类型系统

**版本**: v0.4.0

> **说明**: 本章详细描述幻语的语义分析器和类型系统，包括符号表管理、Hindley-Milner类型推导和语义错误检查。

## 6.1 概述

语义分析阶段接收语法解析器输出的抽象语法树（AST），执行以下核心任务：

- **符号解析**：建立符号表，将标识符绑定到其定义，处理作用域嵌套、导入和模块路径。
- **类型检查与推导**：基于 Hindley‑Milner 类型推导算法，对每个表达式推导类型，并验证类型兼容性；支持泛型、let‑多态。
- **所有权检查**（可选）：在 `使用 所有权` 或 `--ownership` 编译模式下，对变量借用、移动进行编译期验证，保障内存安全。
- **语义错误报告**：收集并报告类型不匹配、未定义变量、重复定义等错误。

语义分析完成后，AST 将被标注上完整的类型信息，符号表可供后续编译阶段使用。

## 6.2 符号表

符号表管理程序中所有声明的符号（变量、函数、类型等），并维护作用域嵌套关系。

### 6.2.1 符号定义

```rust
#[derive(Debug, Clone, PartialEq)]
pub struct Symbol {
    /// 符号名称
    pub name: String,
    /// 符号种类
    pub kind: SymbolKind,
    /// 符号类型（对于变量/函数）
    pub ty: Type,
    /// 是否可变（对于变量）
    pub mutable: bool,
    /// 可见性
    pub visibility: Visibility,
    /// 定义所在的源码范围
    pub span: SourceSpan,
    /// 定义所在的作用域深度（用于快速比较）
    pub scope_depth: usize,
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum SymbolKind {
    Variable,       // 局部变量 / 全局变量
    Constant,       // 常量
    Function,       // 函数
    Struct,         // 结构体类型
    Trait,          // 特征
    Type,           // 类型别名
    Module,         // 模块
    GenericParam,   // 泛型参数
}

#[derive(Debug, Clone, PartialEq, Eq)]
pub enum Visibility {
    Private,            // 模块内可见
    Public,             // 公开可见
    Exported(String),   // 导出为指定名称（用于 FFI）
}
```

pub struct Scope {

### 6.2.3 符号表主结构

```rust
pub struct SymbolTable {
    /// 作用域栈，索引 0 为全局作用域
    scopes: Vec<Scope>,
    /// 当前作用域索引
    current_scope: usize,
    /// 模块路径（用于完全限定名）
    module_path: Vec<String>,
    /// 导入的符号映射（名称 → 完全限定名）
    imports: HashMap<String, String>,
}

impl SymbolTable {
    pub fn new() -> Self {
        let global = Scope::new(None, 0, ScopeNode::Module);
        Self {
            scopes: vec![global],
            current_scope: 0,
            module_path: Vec::new(),
            imports: HashMap::new(),
        }
    }

    /// 进入新作用域
    pub fn enter_scope(&mut self, node: ScopeNode) {
        let depth = self.scopes[self.current_scope].depth + 1;
        let scope = Scope::new(Some(self.current_scope), depth, node);
        self.scopes.push(scope);
        self.current_scope = self.scopes.len() - 1;
    }

    /// 离开当前作用域
    pub fn exit_scope(&mut self) {
        if let Some(parent) = self.scopes[self.current_scope].parent {
            self.current_scope = parent;
        }
    }

    /// 在当前作用域定义符号
    pub fn define(
        &mut self,
        name: String,
        kind: SymbolKind,
        ty: Type,
        mutable: bool,
        visibility: Visibility,
        span: SourceSpan,
    ) -> Result<(), SemanticError> {
        let scope = &mut self.scopes[self.current_scope];
        if scope.symbols.contains_key(&name) {
            let existing = scope.symbols.get(&name).unwrap();
            return Err(SemanticError::DuplicateDefinition {
                name,
                first: existing.span,
                second: span,
            });
        }
        let depth = scope.depth;
        scope.symbols.insert(name.clone(), Symbol {
            name,
            kind,
            ty,
            mutable,
            visibility,
            span,
            scope_depth: depth,
        });
        Ok(())
    }

    /// 从当前作用域开始向外解析符号
    pub fn resolve(&self, name: &str) -> Option<&Symbol> {
        // 先检查导入
        if let Some(full_name) = self.imports.get(name) {
            return self.resolve_fully_qualified(full_name);
        }

        let mut scope_idx = self.current_scope;
        loop {
            let scope = &self.scopes[scope_idx];
            if let Some(sym) = scope.symbols.get(name) {
                return Some(sym);
            }
            if let Some(parent) = scope.parent {
                scope_idx = parent;
            } else {
                break;
            }
        }
        None
    }

    /// 可修改地解析符号
    pub fn resolve_mut(&mut self, name: &str) -> Option<&mut Symbol> {
        let mut scope_idx = self.current_scope;
        loop {
            let scope = &mut self.scopes[scope_idx];
            if scope.symbols.contains_key(name) {
                return scope.symbols.get_mut(name);
            }
            if let Some(parent) = scope.parent {
                scope_idx = parent;
            } else {
                break;
            }
        }
        None
    }

    /// 解析完全限定名
    pub fn resolve_fully_qualified(&self, path: &str) -> Option<&Symbol> {
        // 实际实现需维护模块树
        None
    }

    /// 添加导入映射
    pub fn add_import(&mut self, alias: String, full_path: String) {
        self.imports.insert(alias, full_path);
    }

    /// 进入模块上下文
    pub fn enter_module(&mut self, name: String) {
        self.module_path.push(name);
        self.enter_scope(ScopeNode::Module);
    }

    /// 离开模块上下文
    pub fn exit_module(&mut self) {
        self.module_path.pop();
        self.exit_scope();
    }
}
```

## 6.3 类型推导器（TypeInfer）

类型推导器基于 Hindley‑Milner 算法，支持泛型、类型变量和 let‑多态。

impl TypeInfer {

        self.next_var += 1;

### 6.3.3 合一（Unification）算法

```rust
impl TypeInfer {
    /// 对类型应用当前替换
    fn apply_subst(&self, ty: &Type) -> Type {
        match ty {
            Type::Var(v) => {
                if let Some(t) = self.substitutions.get(v) {
                    self.apply_subst(t)
                } else {
                    ty.clone()
                }
            }
            Type::List(inner) => Type::List(Box::new(self.apply_subst(inner))),
            Type::Array(inner, size) => Type::Array(Box::new(self.apply_subst(inner)), size.clone()),
            Type::Map(k, v) => Type::Map(
                Box::new(self.apply_subst(k)),
                Box::new(self.apply_subst(v)),
            ),
            Type::Ptr(inner) => Type::Ptr(Box::new(self.apply_subst(inner))),
            Type::Option(inner) => Type::Option(Box::new(self.apply_subst(inner))),
            Type::Func(params, ret) => Type::Func(
                params.iter().map(|t| self.apply_subst(t)).collect(),
                Box::new(self.apply_subst(ret)),
            ),
            _ => ty.clone(),
        }
    }

    /// 检查类型变量是否出现在某类型中（发生检查）
    fn occurs(&self, var: usize, ty: &Type) -> bool {
        match ty {
            Type::Var(v) => *v == var,
            Type::List(inner) => self.occurs(var, inner),
            Type::Array(inner, _) => self.occurs(var, inner),
            Type::Map(k, v) => self.occurs(var, k) || self.occurs(var, v),
            Type::Ptr(inner) => self.occurs(var, inner),
            Type::Option(inner) => self.occurs(var, inner),
            Type::Func(params, ret) => {
                params.iter().any(|t| self.occurs(var, t)) || self.occurs(var, ret)
            }
            _ => false,
        }
    }

    /// 合一两个类型
    fn unify(&mut self, t1: Type, t2: Type) -> Result<(), TypeError> {
        let t1 = self.apply_subst(&t1);
        let t2 = self.apply_subst(&t2);

        match (t1, t2) {
            (Type::Var(v1), Type::Var(v2)) if v1 == v2 => Ok(()),
            (Type::Var(v), t) => {
                if self.occurs(v, &t) {
                    Err(TypeError::Cyclic { span: SourceSpan::dummy() })
                } else {
                    self.substitutions.insert(v, t);
                    Ok(())
                }
            }
            (t, Type::Var(v)) => {
                if self.occurs(v, &t) {
                    Err(TypeError::Cyclic { span: SourceSpan::dummy() })
                } else {
                    self.substitutions.insert(v, t);
                    Ok(())
                }
            }
            (Type::Int, Type::Int) => Ok(()),
            (Type::Bool, Type::Bool) => Ok(()),
            (Type::String, Type::String) => Ok(()),
            (Type::Unit, Type::Unit) => Ok(()),
            (Type::List(a), Type::List(b)) => self.unify(*a, *b),
            (Type::Map(k1, v1), Type::Map(k2, v2)) => {
                self.unify(*k1, *k2)?;
                self.unify(*v1, *v2)
            }
            (Type::Func(p1, r1), Type::Func(p2, r2)) => {
                if p1.len() != p2.len() {
                    return Err(TypeError::Mismatch {
                        expected: Type::Func(p1, r1),
                        found: Type::Func(p2, r2),
                        span: SourceSpan::dummy(),
                    });
                }
                for (a, b) in p1.into_iter().zip(p2) {
                    self.unify(a, b)?;
                }
                self.unify(*r1, *r2)
            }
            (Type::Named(p1), Type::Named(p2)) if p1 == p2 => Ok(()),
            (a, b) => Err(TypeError::Mismatch {
                expected: a,
                found: b,
                span: SourceSpan::dummy(),
            }),
        }
    }
}
```

### 6.3.4 表达式类型推导

```rust
impl TypeInfer {
    /// 推导表达式的类型
    pub fn infer_expr(&mut self, expr: &Expr) -> Result<Type, TypeError> {
        let span = expr.span();
        match expr {
            Expr::IntLit(_, _) => Ok(Type::Int),
            Expr::FloatLit(_, _) => Ok(Type::F64),
            Expr::StringLit(_, _) => Ok(Type::String),
            Expr::CharLit(_, _) => Ok(Type::Char),
            Expr::BoolLit(_, _) => Ok(Type::Bool),
            Expr::Null(_) => Ok(self.fresh_var()),

            Expr::Ident(ident) => {
                self.lookup_type(&ident.name)
                    .cloned()
                    .ok_or_else(|| TypeError::UnboundVariable {
                        name: ident.name.clone(),
                        span: ident.span,
                    })
            }

            Expr::BinaryOp { op, left, right, span } => {
                let t_left = self.infer_expr(left)?;
                let t_right = self.infer_expr(right)?;

                match op {
                    BinaryOp::Add | BinaryOp::Sub | BinaryOp::Mul | BinaryOp::Div | BinaryOp::Mod => {
                        self.unify(t_left.clone(), t_right.clone())?;
                        Ok(t_left)
                    }
                    BinaryOp::And | BinaryOp::Or => {
                        self.unify(t_left, Type::Bool)?;
                        self.unify(t_right, Type::Bool)?;
                        Ok(Type::Bool)
                    }
                    BinaryOp::Eq | BinaryOp::Ne | BinaryOp::Gt | BinaryOp::Lt
                    | BinaryOp::Ge | BinaryOp::Le => {
                        self.unify(t_left, t_right)?;
                        Ok(Type::Bool)
                    }
                    BinaryOp::Assign => {
                        self.unify(t_left, t_right)?;
                        Ok(Type::Unit)
                    }
                    _ => {
                        self.unify(t_left.clone(), t_right)?;
                        Ok(Type::Unit)
                    }
                }
            }

            Expr::UnaryOp { op, expr, span } => {
                let t_expr = self.infer_expr(expr)?;
                match op {
                    UnaryOp::Not => {
                        self.unify(t_expr, Type::Bool)?;
                        Ok(Type::Bool)
                    }
                    UnaryOp::Neg => {
                        if !t_expr.is_numeric() {
                            return Err(TypeError::InvalidUnaryOp { op: "负号".into(), ty: t_expr, span: *span });
                        }
                        Ok(t_expr)
                    }
                    UnaryOp::BitNot => {
                        if !t_expr.is_integer() {
                            return Err(TypeError::InvalidUnaryOp { op: "按位非".into(), ty: t_expr, span: *span });
                        }
                        Ok(t_expr)
                    }
                }
            }

            Expr::Call { func, args, span } => {
                let t_func = self.infer_expr(func)?;
                match t_func {
                    Type::Func(param_tys, ret_ty) => {
                        if param_tys.len() != args.len() {
                            return Err(TypeError::ArgCountMismatch {
                                expected: param_tys.len(),
                                found: args.len(),
                                span: *span,
                            });
                        }
                        for (arg, expected) in args.iter().zip(param_tys.iter()) {
                            let t_arg = self.infer_expr(arg)?;
                            self.unify(t_arg, expected.clone())?;
                        }
                        Ok(*ret_ty)
                    }
                    _ => Err(TypeError::NotCallable { ty: t_func, span: *span }),
                }
            }

            // 其他表达式分支类似...
            _ => Ok(Type::Unit),
        }
    }

    fn lookup_type(&self, name: &str) -> Option<&Type> {
        for scope in self.env.iter().rev() {
            if let Some(ty) = scope.get(name) {
                return Some(ty);
            }
        }
        None
    }
}
```

### 6.3.5 语句类型检查

```rust
impl TypeInfer {
    pub fn infer_stmt(&mut self, stmt: &Stmt) -> Result<(), TypeError> {
        match stmt {
            Stmt::Let { name, ty, value, span } => {
                let t_value = self.infer_expr(value)?;
                if let Some(anno) = ty {
                    self.unify(t_value.clone(), anno.clone())?;
                }
                self.env.last_mut().unwrap().insert(name.name.clone(), ty.clone().unwrap_or(t_value));
                Ok(())
            }
            Stmt::Assign { target, value, span } => {
                let t_target = self.infer_expr(target)?;
                let t_value = self.infer_expr(value)?;
                self.unify(t_target, t_value)
            }
            Stmt::If { cond, then_block, else_ifs, else_block, span } => {
                let t_cond = self.infer_expr(cond)?;
                self.unify(t_cond, Type::Bool)?;
                self.env.push(HashMap::new());
                for s in then_block { self.infer_stmt(s)?; }
                self.env.pop();
                for (cond, block) in else_ifs {
                    let t_cond = self.infer_expr(cond)?;
                    self.unify(t_cond, Type::Bool)?;
                    self.env.push(HashMap::new());
                    for s in block { self.infer_stmt(s)?; }
                    self.env.pop();
                }
                if let Some(block) = else_block {
                    self.env.push(HashMap::new());
                    for s in block { self.infer_stmt(s)?; }
                    self.env.pop();
                }
                Ok(())
            }
            Stmt::Return(expr, span) => {
                if let Some(expected) = &self.current_return_type {
                    if let Some(expr) = expr {
                        let t_expr = self.infer_expr(expr)?;
                        self.unify(t_expr, expected.clone())?;
                    } else {
                        self.unify(Type::Unit, expected.clone())?;
                    }
                    Ok(())
                } else {
                    Err(TypeError::ReturnOutsideFunction { span: *span })
                }
            }
            // 其他语句类型...
            _ => Ok(()),
        }
    }
}
```

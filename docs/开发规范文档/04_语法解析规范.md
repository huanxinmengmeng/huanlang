# 4. 语法解析规范

## 4.1 语法解析器概述

语法解析器（Parser）负责将词法分析器输出的 Token 流转换为抽象语法树（AST）。幻语解析器采用**递归下降**与 **Pratt 解析器**相结合的算法，能够高效处理表达式优先级，并对语法错误提供良好的恢复与报告。

**主要职责**：
- 验证 Token 序列是否符合幻语语法规则
- 构建完整的抽象语法树（AST）
- 记录语法错误，并在可能的情况下恢复以继续解析
- 维护源码位置信息以提供准确的错误报告

## 4.2 解析错误类型

```rust
#[derive(Debug, Clone, PartialEq)]
pub enum ParseError {
    /// 意外的 Token（期望某个 Token，但找到了其他 Token）
    UnexpectedToken {
        expected: String,
        found: Token,
        span: SourceSpan,
    },
    /// 文件意外结束
    UnexpectedEof {
        expected: String,
    },
    /// 通用语法错误
    InvalidSyntax {
        message: String,
        span: SourceSpan,
    },
    /// 重复定义（同一作用域内多次定义相同名称）
    DuplicateDefinition {
        name: String,
        first: SourceSpan,
        second: SourceSpan,
    },
    /// 致命错误，无法继续解析
    Fatal(String),
}
```

## 4.3 解析器结构

```rust
pub struct Parser {
    /// Token 流
    tokens: Vec<Token>,
    /// 当前读取位置
    pos: usize,
    /// 收集的解析错误
    errors: Vec<ParseError>,
    /// 关键词映射表（用于识别三语关键词）
    keyword_table: KeywordTable,
}

impl Parser {
    pub fn new(tokens: Vec<Token>) -> Self {
        Self {
            tokens,
            pos: 0,
            errors: Vec::new(),
            keyword_table: KeywordTable::new(),
        }
    }

    // ========== 辅助方法 ==========

    /// 获取当前 Token
    fn current(&self) -> Option<&Token> {
        self.tokens.get(self.pos)
    }

    /// 获取下一个 Token（不移动位置）
    fn peek(&self) -> Option<&Token> {
        self.tokens.get(self.pos + 1)
    }

    /// 移动至下一个 Token 并返回当前 Token
    fn advance(&mut self) -> Option<&Token> {
        if self.pos < self.tokens.len() {
            self.pos += 1;
            self.tokens.get(self.pos - 1)
        } else {
            None
        }
    }

    /// 检查当前 Token 是否属于指定的 TokenKind
    fn check(&self, kind: TokenKind) -> bool {
        self.current().map(|t| t.kind == kind).unwrap_or(false)
    }

    /// 检查当前 Token 是否为指定三语关键词之一
    fn check_keyword(&self, keywords: &[&str]) -> bool {
        if let Some(tok) = self.current() {
            for kw in keywords {
                if let Some(kind) = self.keyword_table.get(kw) {
                    if tok.kind == kind {
                        return true;
                    }
                }
            }
        }
        false
    }

    /// 如果当前 Token 匹配指定 TokenKind，则消耗它并返回 Some(Token)
    fn eat(&mut self, kind: TokenKind) -> Option<Token> {
        if self.check(kind.clone()) {
            Some(self.advance().unwrap().clone())
        } else {
            None
        }
    }

    /// 期望当前 Token 为指定 TokenKind，否则报错
    fn expect(&mut self, kind: TokenKind, expected: &str) -> Result<Token, ParseError> {
        if let Some(tok) = self.current() {
            if tok.kind == kind {
                Ok(self.advance().unwrap().clone())
            } else {
                Err(ParseError::UnexpectedToken {
                    expected: expected.to_string(),
                    found: tok.clone(),
                    span: tok.span,
                })
            }
        } else {
            Err(ParseError::UnexpectedEof {
                expected: expected.to_string(),
            })
        }
    }

    /// 检查是否已到达 Token 流末尾
    fn is_eof(&self) -> bool {
        self.pos >= self.tokens.len()
            || matches!(self.current().map(|t| &t.kind), Some(TokenKind::Eof))
    }

    /// 获取当前 Token 的源码范围（若已 EOF 则返回空范围）
    fn current_span(&self) -> SourceSpan {
        self.current()
            .map(|t| t.span)
            .unwrap_or_else(|| SourceSpan::new(
                SourcePosition::new(0, 0, 0),
                SourcePosition::new(0, 0, 0),
            ))
    }

    /// 错误恢复：跳过若干 Token 直到遇到同步 Token
    fn synchronize(&mut self) {
        self.advance();
        while !self.is_eof() {
            // 遇到这些 Token 时认为可以安全继续解析
            if self.check(TokenKind::Semicolon)
                || self.check(TokenKind::RBrace)
                || self.check_keyword(&["结束", "end", "jieshu"])
                || self.check_keyword(&["函数", "func", "hanshu"])
                || self.check_keyword(&["结构", "struct", "jiegou"])
                || self.check_keyword(&["模块", "module", "mokuai"])
                || self.check_keyword(&["导入", "import", "daoru"])
            {
                return;
            }
            self.advance();
        }
    }
}
```

## 4.4 顶层解析入口

```rust
impl Parser {
    /// 解析整个程序，返回 AST 节点列表
    pub fn parse(&mut self) -> Result<Program, ParseError> {
        let mut items = Vec::new();
        while !self.is_eof() {
            match self.parse_item() {
                Ok(item) => items.push(item),
                Err(e) => {
                    self.errors.push(e.clone());
                    self.synchronize();
                    if matches!(e, ParseError::Fatal(_)) {
                        return Err(e);
                    }
                }
            }
        }
        if !self.errors.is_empty() {
            return Err(self.errors[0].clone());
        }
        Ok(items)
    }

    /// 解析单个顶层项
    fn parse_item(&mut self) -> Result<Item, ParseError> {
        let public = self.eat(TokenKind::Pub).is_some();

        if self.check_keyword(&["模块", "module", "mokuai"]) {
            self.parse_module(public)
        } else if self.check_keyword(&["导入", "import", "daoru"]) {
            self.parse_import()
        } else if self.check_keyword(&["函数", "func", "hanshu"]) {
            self.parse_function(public)
        } else if self.check_keyword(&["结构", "struct", "jiegou"]) {
            self.parse_struct(public)
        } else if self.check_keyword(&["特征", "trait", "tezheng"]) {
            self.parse_trait(public)
        } else if self.check_keyword(&["实现", "impl", "shixian"]) {
            self.parse_impl()
        } else if self.check_keyword(&["外部", "extern"]) {
            self.parse_extern()
        } else if self.check_keyword(&["类型", "type", "leixing"]) {
            self.parse_type_alias(public)
        } else if self.check_keyword(&["令", "let", "ling"]) || self.check_keyword(&["定", "const", "ding"]) {
            self.parse_global(public)
        } else {
            let tok = self.current().cloned().unwrap_or_else(|| Token::eof(SourcePosition::new(0,0,0)));
            Err(ParseError::UnexpectedToken {
                expected: "模块、导入、函数、结构、特征、实现、外部、类型、令、定".to_string(),
                found: tok,
                span: tok.span,
            })
        }
    }
}
```

## 4.5 表达式解析（Pratt 解析器）

### 4.5.1 优先级定义

```rust
// 优先级常量，数值越大优先级越高
const PREC_LOWEST: u8 = 0;
const PREC_ASSIGN: u8 = 1;      // = += -= 等
const PREC_OR: u8 = 2;          // 或 ||
const PREC_AND: u8 = 3;         // 且 &&
const PREC_EQ: u8 = 4;          // == !=
const PREC_COMPARE: u8 = 5;     // > < >= <=
const PREC_BITOR: u8 = 6;       // |
const PREC_BITXOR: u8 = 7;      // ^
const PREC_BITAND: u8 = 8;      // &
const PREC_SHIFT: u8 = 9;       // << >>
const PREC_ADD: u8 = 10;        // + -
const PREC_MUL: u8 = 11;        // * / %
const PREC_UNARY: u8 = 12;      // 一元运算符 ! - ~
const PREC_CALL: u8 = 13;       // 函数调用、索引、成员访问
```

### 4.5.2 中缀运算符优先级映射

```rust
impl Parser {
    fn get_infix_precedence(&self, kind: &TokenKind) -> u8 {
        match kind {
            TokenKind::Assign
            | TokenKind::PlusEq | TokenKind::MinusEq | TokenKind::StarEq
            | TokenKind::SlashEq | TokenKind::PercentEq
            | TokenKind::ShlEq | TokenKind::ShrEq
            | TokenKind::BitAndEq | TokenKind::BitOrEq | TokenKind::BitXorEq => PREC_ASSIGN,

            TokenKind::Or => PREC_OR,
            TokenKind::And => PREC_AND,
            TokenKind::EqEq | TokenKind::NotEq => PREC_EQ,
            TokenKind::Lt | TokenKind::Gt | TokenKind::Le | TokenKind::Ge => PREC_COMPARE,
            TokenKind::BitOr => PREC_BITOR,
            TokenKind::BitXor => PREC_BITXOR,
            TokenKind::BitAnd => PREC_BITAND,
            TokenKind::Shl | TokenKind::Shr => PREC_SHIFT,
            TokenKind::Plus | TokenKind::Minus => PREC_ADD,
            TokenKind::Star | TokenKind::Slash | TokenKind::Percent => PREC_MUL,
            TokenKind::LParen | TokenKind::LBracket | TokenKind::Dot => PREC_CALL,
            _ => PREC_LOWEST,
        }
    }

    /// 解析表达式入口
    fn parse_expr(&mut self) -> Result<Expr, ParseError> {
        self.parse_expr_with_precedence(PREC_LOWEST)
    }

    /// 根据优先级解析表达式
    fn parse_expr_with_precedence(&mut self, min_prec: u8) -> Result<Expr, ParseError> {
        let mut lhs = self.parse_prefix()?;

        while let Some(tok) = self.current() {
            let prec = self.get_infix_precedence(&tok.kind);
            if prec < min_prec {
                break;
            }
            lhs = self.parse_infix(lhs, prec)?;
        }

        Ok(lhs)
    }
}
```

### 4.5.3 前缀表达式解析

```rust
impl Parser {
    fn parse_prefix(&mut self) -> Result<Expr, ParseError> {
        let tok = self.current().ok_or_else(|| ParseError::UnexpectedEof {
            expected: "表达式".to_string(),
        })?;
        let span = tok.span;

        match &tok.kind {
            TokenKind::IntLit(n) => {
                self.advance();
                Ok(Expr::IntLit(*n, span))
            }
            TokenKind::FloatLit(n) => {
                self.advance();
                Ok(Expr::FloatLit(*n, span))
            }
            TokenKind::StringLit(s) => {
                self.advance();
                Ok(Expr::StringLit(s.clone(), span))
            }
            TokenKind::CharLit(c) => {
                self.advance();
                Ok(Expr::CharLit(*c, span))
            }
            TokenKind::True => {
                self.advance();
                Ok(Expr::BoolLit(true, span))
            }
            TokenKind::False => {
                self.advance();
                Ok(Expr::BoolLit(false, span))
            }
            TokenKind::Null => {
                self.advance();
                Ok(Expr::Null(span))
            }
            TokenKind::Ident(name) => {
                let ident = Ident { name: name.clone(), span };
                self.advance();
                Ok(Expr::Ident(ident))
            }
            TokenKind::LParen => {
                self.advance();
                let expr = self.parse_expr()?;
                self.expect(TokenKind::RParen, "')'")?;
                Ok(expr)
            }
            TokenKind::LBracket => self.parse_list_literal(),
            TokenKind::LBrace => self.parse_map_or_struct_literal(),
            TokenKind::Minus => {
                self.advance();
                let expr = self.parse_expr_with_precedence(PREC_UNARY)?;
                Ok(Expr::UnaryOp {
                    op: UnaryOp::Neg,
                    expr: Box::new(expr),
                    span: span.merge(self.current_span()),
                })
            }
            TokenKind::Not => {
                self.advance();
                let expr = self.parse_expr_with_precedence(PREC_UNARY)?;
                Ok(Expr::UnaryOp {
                    op: UnaryOp::Not,
                    expr: Box::new(expr),
                    span: span.merge(self.current_span()),
                })
            }
            TokenKind::BitNot => {
                self.advance();
                let expr = self.parse_expr_with_precedence(PREC_UNARY)?;
                Ok(Expr::UnaryOp {
                    op: UnaryOp::BitNot,
                    expr: Box::new(expr),
                    span: span.merge(self.current_span()),
                })
            }
            TokenKind::If => self.parse_if_expr(),
            TokenKind::Match => self.parse_match_expr(),
            TokenKind::Asm => self.parse_asm_expr(),
            TokenKind::BitAnd => {
                // 闭包语法：|参数| 表达式
                self.parse_closure()
            }
            _ => Err(ParseError::UnexpectedToken {
                expected: "表达式".to_string(),
                found: tok.clone(),
                span: tok.span,
            }),
        }
    }
}
```

### 4.5.4 中缀表达式解析

```rust
impl Parser {
    fn parse_infix(&mut self, lhs: Expr, min_prec: u8) -> Result<Expr, ParseError> {
        let tok = self.current().unwrap().clone();
        self.advance();

        match &tok.kind {
            // 赋值运算符
            TokenKind::Assign | TokenKind::PlusEq | TokenKind::MinusEq | TokenKind::StarEq
            | TokenKind::SlashEq | TokenKind::PercentEq | TokenKind::ShlEq | TokenKind::ShrEq
            | TokenKind::BitAndEq | TokenKind::BitOrEq | TokenKind::BitXorEq => {
                let rhs = self.parse_expr_with_precedence(min_prec - 1)?;
                let op = match tok.kind {
                    TokenKind::Assign => BinaryOp::Assign,
                    TokenKind::PlusEq => BinaryOp::AddAssign,
                    TokenKind::MinusEq => BinaryOp::SubAssign,
                    TokenKind::StarEq => BinaryOp::MulAssign,
                    TokenKind::SlashEq => BinaryOp::DivAssign,
                    TokenKind::PercentEq => BinaryOp::ModAssign,
                    TokenKind::ShlEq => BinaryOp::ShlAssign,
                    TokenKind::ShrEq => BinaryOp::ShrAssign,
                    TokenKind::BitAndEq => BinaryOp::BitAndAssign,
                    TokenKind::BitOrEq => BinaryOp::BitOrAssign,
                    TokenKind::BitXorEq => BinaryOp::BitXorAssign,
                    _ => unreachable!(),
                };
                Ok(Expr::BinaryOp {
                    op,
                    left: Box::new(lhs),
                    right: Box::new(rhs),
                    span: lhs.span().merge(rhs.span()),
                })
            }

            // 二元运算符
            TokenKind::Plus | TokenKind::Minus | TokenKind::Star | TokenKind::Slash | TokenKind::Percent
            | TokenKind::And | TokenKind::Or | TokenKind::EqEq | TokenKind::NotEq
            | TokenKind::Lt | TokenKind::Gt | TokenKind::Le | TokenKind::Ge
            | TokenKind::Shl | TokenKind::Shr | TokenKind::BitAnd | TokenKind::BitOr | TokenKind::BitXor => {
                let rhs = self.parse_expr_with_precedence(min_prec + 1)?;
                let op = match tok.kind {
                    TokenKind::Plus => BinaryOp::Add,
                    TokenKind::Minus => BinaryOp::Sub,
                    TokenKind::Star => BinaryOp::Mul,
                    TokenKind::Slash => BinaryOp::Div,
                    TokenKind::Percent => BinaryOp::Mod,
                    TokenKind::And => BinaryOp::And,
                    TokenKind::Or => BinaryOp::Or,
                    TokenKind::EqEq => BinaryOp::Eq,
                    TokenKind::NotEq => BinaryOp::Ne,
                    TokenKind::Lt => BinaryOp::Lt,
                    TokenKind::Gt => BinaryOp::Gt,
                    TokenKind::Le => BinaryOp::Le,
                    TokenKind::Ge => BinaryOp::Ge,
                    TokenKind::Shl => BinaryOp::Shl,
                    TokenKind::Shr => BinaryOp::Shr,
                    TokenKind::BitAnd => BinaryOp::BitAnd,
                    TokenKind::BitOr => BinaryOp::BitOr,
                    TokenKind::BitXor => BinaryOp::BitXor,
                    _ => unreachable!(),
                };
                Ok(Expr::BinaryOp {
                    op,
                    left: Box::new(lhs),
                    right: Box::new(rhs),
                    span: lhs.span().merge(rhs.span()),
                })
            }

            // 函数调用
            TokenKind::LParen => {
                let mut args = Vec::new();
                if !self.check(TokenKind::RParen) {
                    loop {
                        args.push(self.parse_expr()?);
                        if !self.eat(TokenKind::Comma).is_some() {
                            break;
                        }
                    }
                }
                let rparen = self.expect(TokenKind::RParen, "')'")?;
                Ok(Expr::Call {
                    func: Box::new(lhs),
                    args,
                    span: lhs.span().merge(rparen.span),
                })
            }

            // 成员访问或方法调用
            TokenKind::Dot => {
                let field_tok = self.current().ok_or_else(|| ParseError::UnexpectedEof {
                    expected: "字段名".to_string(),
                })?;
                let field = match &field_tok.kind {
                    TokenKind::Ident(name) => Ident { name: name.clone(), span: field_tok.span },
                    _ => return Err(ParseError::UnexpectedToken {
                        expected: "字段名".to_string(),
                        found: field_tok.clone(),
                        span: field_tok.span,
                    }),
                };
                self.advance();

                if self.check(TokenKind::LParen) {
                    // 方法调用
                    self.advance();
                    let mut args = Vec::new();
                    if !self.check(TokenKind::RParen) {
                        loop {
                            args.push(self.parse_expr()?);
                            if !self.eat(TokenKind::Comma).is_some() {
                                break;
                            }
                        }
                    }
                    let rparen = self.expect(TokenKind::RParen, "')'")?;
                    Ok(Expr::MethodCall {
                        receiver: Box::new(lhs),
                        method: field,
                        args,
                        span: lhs.span().merge(rparen.span),
                    })
                } else {
                    Ok(Expr::Field {
                        target: Box::new(lhs),
                        field,
                        span: lhs.span().merge(field.span),
                    })
                }
            }

            // 索引访问
            TokenKind::LBracket => {
                let index = self.parse_expr()?;
                let rbracket = self.expect(TokenKind::RBracket, "']'")?;
                Ok(Expr::Index {
                    target: Box::new(lhs),
                    index: Box::new(index),
                    span: lhs.span().merge(rbracket.span),
                })
            }

            // 可选解包
            TokenKind::Question => {
                Ok(Expr::Try {
                    expr: Box::new(lhs),
                    span: lhs.span().merge(tok.span),
                })
            }

            // 类型转换
            TokenKind::As => {
                let ty = self.parse_type()?;
                Ok(Expr::TypeAssertion {
                    expr: Box::new(lhs),
                    ty,
                    span: lhs.span().merge(ty.span()),
                })
            }

            _ => Ok(lhs),
        }
    }
}
```

```rust
impl Parser {
    fn parse_list_literal(&mut self) -> Result<Expr, ParseError> {
        let start = self.current_span();
        self.advance(); // 跳过 '['

        let mut elements = Vec::new();
        if !self.check(TokenKind::RBracket) {
            loop {
                elements.push(self.parse_expr()?);
                if !self.eat(TokenKind::Comma).is_some() {
                    break;
                }
            }
        }
        let end = self.expect(TokenKind::RBracket, "']'")?.span;
        Ok(Expr::List(elements, start.merge(end)))
    }

    fn parse_map_or_struct_literal(&mut self) -> Result<Expr, ParseError> {
        let start = self.current_span();
        self.advance(); // 跳过 '{'

        // 空字典/结构体

            return Ok(Expr::Map(Vec::new(), start.merge(end)));

            return_type,
            body,
```

```rust
impl Parser {
    fn parse_stmt(&mut self) -> Result<Stmt, ParseError> {
        if self.check_keyword(&["令", "let", "ling"]) {
            self.parse_let()
        } else if self.check_keyword(&["定", "const", "ding"]) {
            self.parse_const()
        } else if self.check_keyword(&["若", "if", "ruo"]) {
            self.parse_if()
        } else if self.check_keyword(&["匹配", "match", "pipei"]) {
            self.parse_match()
        } else if self.check_keyword(&["当", "while", "dang"]) {
            self.parse_while()
        } else if self.check_keyword(&["重复", "repeat", "chongfu"]) {

        } else if self.check_keyword(&["对于", "for", "duiyu"]) {

        } else if self.check_keyword(&["返回", "return", "fanhui"]) {
            self.parse_return()
        } else if self.check_keyword(&["跳出", "break", "tiaochu"]) {
            let span = self.advance().unwrap().span;
            Ok(Stmt::Break(span))
        } else if self.check_keyword(&["继续", "continue", "jixu"]) {
            let span = self.advance().unwrap().span;
            Ok(Stmt::Continue(span))
        } else if self.check_keyword(&["汇编", "asm", "huibian"]) {
            self.parse_asm_stmt()
        } else {
            let expr = self.parse_expr()?;
            // 检查是否为赋值语句
            if self.check(TokenKind::Assign)
                || self.check_keyword(&["设为", "assign", "shewei"])
                || self.check(TokenKind::PlusEq) || self.check(TokenKind::MinusEq)
                || self.check(TokenKind::StarEq) || self.check(TokenKind::SlashEq)
            {
                let op_tok = self.advance().unwrap().clone();
                let value = self.parse_expr()?;
                let span = expr.span().merge(value.span());
                Ok(Stmt::Assign {
                    target: Box::new(expr),
                    value: Box::new(value),
                    span,
                })
            } else {
                Ok(Stmt::Expr(Box::new(expr), expr.span()))
            }
        }
    }

    /// 解析语句块（直到遇到 end 或 else 等）
    fn parse_block(&mut self) -> Result<Vec<Stmt>, ParseError> {
        let mut stmts = Vec::new();
        while !self.check_keyword(&["结束", "end", "jieshu", "否则", "else", "fouze"])
            && !self.is_eof()
        {
            stmts.push(self.parse_stmt()?);
        }
        Ok(stmts)
    }
```

```rust
impl Parser {
    fn parse_type(&mut self) -> Result<Type, ParseError> {
        let tok = self.current().ok_or_else(|| ParseError::UnexpectedEof {
            expected: "类型".to_string(),
        })?;

        match &tok.kind {
            TokenKind::TypeInt => { self.advance(); Ok(Type::Int) }
            TokenKind::TypeI8 => { self.advance(); Ok(Type::I8) }
            TokenKind::TypeI16 => { self.advance(); Ok(Type::I16) }
            TokenKind::TypeI32 => { self.advance(); Ok(Type::I32) }
            TokenKind::TypeI64 => { self.advance(); Ok(Type::I64) }
            TokenKind::TypeU8 => { self.advance(); Ok(Type::U8) }
            TokenKind::TypeU16 => { self.advance(); Ok(Type::U16) }
            TokenKind::TypeU32 => { self.advance(); Ok(Type::U32) }
            TokenKind::TypeU64 => { self.advance(); Ok(Type::U64) }
            TokenKind::TypeF32 => { self.advance(); Ok(Type::F32) }
            TokenKind::TypeF64 => { self.advance(); Ok(Type::F64) }
            TokenKind::TypeBool => { self.advance(); Ok(Type::Bool) }
            TokenKind::TypeChar => { self.advance(); Ok(Type::Char) }
            TokenKind::TypeString => { self.advance(); Ok(Type::String) }
            TokenKind::TypeUnit => { self.advance(); Ok(Type::Unit) }
            TokenKind::TypeList => {
                self.advance();
                self.expect(TokenKind::LBracket, "'['")?;
                let inner = self.parse_type()?;
                self.expect(TokenKind::RBracket, "']'")?;
                Ok(Type::List(Box::new(inner)))
            }
            TokenKind::TypeArray => {
                self.advance();
                self.expect(TokenKind::LBracket, "'['")?;
                let inner = self.parse_type()?;
                self.expect(TokenKind::Semicolon, "';'")?;
                let size = self.parse_expr()?;
                self.expect(TokenKind::RBracket, "']'")?;
                Ok(Type::Array(Box::new(inner), Box::new(size)))
            }
            TokenKind::TypeMap => {
                self.advance();
                self.expect(TokenKind::LBracket, "'['")?;
                let key = self.parse_type()?;
                self.expect(TokenKind::Comma, "','")?;
                let value = self.parse_type()?;
                self.expect(TokenKind::RBracket, "']'")?;
                Ok(Type::Map(Box::new(key), Box::new(value)))
            }
            TokenKind::TypePtr => {
                self.advance();
                self.expect(TokenKind::LBracket, "'['")?;
                let inner = self.parse_type()?;
                self.expect(TokenKind::RBracket, "']'")?;
                Ok(Type::Ptr(Box::new(inner)))
            }
            TokenKind::TypeOption => {
                self.advance();
                self.expect(TokenKind::LBracket, "'['")?;
                let inner = self.parse_type()?;
                self.expect(TokenKind::RBracket, "']'")?;
                Ok(Type::Option(Box::new(inner)))
            }
            TokenKind::Ident(name) => {
                let path = self.parse_path()?;
                Ok(Type::Named(path))
            }
            _ => Err(ParseError::UnexpectedToken {
                expected: "类型名称".to_string(),
                found: tok.clone(),
                span: tok.span,
            }),
        }
    }

    fn parse_path(&mut self) -> Result<Path, ParseError> {
        let mut segments = Vec::new();
        let start_span = self.current_span();

        loop {
            let ident = self.parse_ident()?;
            segments.push(ident);
            if !self.eat(TokenKind::DoubleColon).is_some() {
                break;
            }
        }

        let generics = if self.check(TokenKind::Lt) {
            Some(self.parse_generic_args()?)
        } else {
            None
        };

        Ok(Path {
            segments,
            generics,
            span: start_span.merge(self.current_span()),
        })
    }

    fn parse_pattern(&mut self) -> Result<Pattern, ParseError> {
        let tok = self.current().ok_or_else(|| ParseError::UnexpectedEof {
            expected: "模式".to_string(),
        })?;

        match &tok.kind {
            TokenKind::Ident(name) if name == "_" => {
                let span = tok.span;
                self.advance();
                Ok(Pattern::Wildcard(span))
            }
            TokenKind::Ident(_) => {
                let ident = self.parse_ident()?;
                Ok(Pattern::Ident(ident))
            }
            TokenKind::IntLit(n) => {
                let lit = Expr::IntLit(*n, tok.span);
                self.advance();
                Ok(Pattern::Literal(lit))
            }
            TokenKind::StringLit(s) => {
                let lit = Expr::StringLit(s.clone(), tok.span);
                self.advance();
                Ok(Pattern::Literal(lit))
            }
            TokenKind::True | TokenKind::False => {
                let lit = Expr::BoolLit(matches!(tok.kind, TokenKind::True), tok.span);
                self.advance();
                Ok(Pattern::Literal(lit))
            }
            TokenKind::LBracket => {
                self.advance();
                let mut elements = Vec::new();
                if !self.check(TokenKind::RBracket) {
                    loop {
                        elements.push(self.parse_pattern()?);
                        if !self.eat(TokenKind::Comma).is_some() {
                            break;
                        }
                    }
                }
                let span = tok.span.merge(self.expect(TokenKind::RBracket, "']'")?.span);
                Ok(Pattern::List(elements, span))
            }
            _ => {
                // 尝试解析为结构体模式
                let path = self.parse_path()?;
                self.expect(TokenKind::LBrace, "'{'")?;
                let mut fields = Vec::new();
                while !self.check(TokenKind::RBrace) {
                    let field = self.parse_ident()?;
                    let pattern = if self.eat(TokenKind::Colon).is_some() {
                        self.parse_pattern()?
                    } else {
                        Pattern::Ident(field.clone())
                    };
                    fields.push((field, pattern));
                    if !self.eat(TokenKind::Comma).is_some() {
                        break;
                    }
                }
                let span = tok.span.merge(self.expect(TokenKind::RBrace, "'}'")?.span);
                Ok(Pattern::Struct { path, fields, span })
            }
        }
    }
}
```

```rust
        } else {
            Vec::new()
        };

        let mut postconditions = Vec::new();
        while self.check_keyword(&["前置", "requires"]) {
            self.advance();
            self.expect(TokenKind::Colon, "':'")?;
            preconditions.push(self.parse_expr()?);
        }
        while self.check_keyword(&["后置", "ensures"]) {

            self.expect(TokenKind::Colon, "':'")?;
            postconditions.push(self.parse_expr()?);
        }

        // 函数体
        self.expect_keyword(&["开始", "begin"])?;  // 或直接 '{'
        let body = self.parse_block()?;
        let end_span = self.expect_keyword(&["结束", "end", "jieshu"])?.span;

            name,

            return_type,

    fn parse_generic_params(&mut self) -> Result<Vec<GenericParam>, ParseError> {
        self.expect(TokenKind::Lt, "'<'")?;
        let mut params = Vec::new();
        loop {
            let name = self.parse_ident()?;
            let mut bounds = Vec::new();
            if self.eat(TokenKind::Colon).is_some() {
                loop {
                    bounds.push(self.parse_path()?);
                    if !self.eat(TokenKind::Plus).is_some() {
                        break;
                    }
                }
            }
            let default = if self.eat(TokenKind::Assign).is_some() {
                Some(self.parse_type()?)
            } else {
                None
            };
            params.push(GenericParam { name, bounds, default });
            if !self.eat(TokenKind::Comma).is_some() {
                break;
            }
        }
        self.expect(TokenKind::Gt, "'>'")?;
        Ok(params)
    }

    fn parse_where_clause(&mut self) -> Result<Vec<WherePredicate>, ParseError> {
        self.expect_keyword(&["其中", "where"])?;
        let mut preds = Vec::new();
        loop {
            let ty = self.parse_type()?;
            self.expect(TokenKind::Colon, "':'")?;
            let mut bounds = Vec::new();
            loop {
                bounds.push(self.parse_path()?);
                if !self.eat(TokenKind::Plus).is_some() {
                    break;
                }
            }
            preds.push(WherePredicate::TypeBound { ty, bounds });
            if !self.eat(TokenKind::Comma).is_some() {
                break;
            }
        }
        Ok(preds)
    }
```

```rust
                items.push(ExternItem::Static { name, ty, mutable });
            } else {
                let tok = self.current().unwrap().clone();
                return Err(ParseError::UnexpectedToken {
                    expected: "函数或变量声明".to_string(),
                    found: tok,
                    span: tok.span,
                });
            }
            self.eat(TokenKind::Semicolon);
        }
        let end_span = self.expect(TokenKind::RBrace, "'}'")?.span;

        Ok(Item::Extern(ExternBlock {
            abi,
            items,
            span: start.merge(end_span),
        }))
    }
}
```

## 4.9 错误恢复与报告

解析器在遇到语法错误时，会记录错误并尝试通过 `synchronize` 方法跳过部分 Token，以便继续解析后续内容，从而在一次编译中报告多个错误。

```rust
impl Parser {
    /// 解析顶层项，带错误恢复
    fn parse_item_with_recovery(&mut self) -> Option<Item> {
        match self.parse_item() {
            Ok(item) => Some(item),
            Err(e) => {
                self.errors.push(e);
                self.synchronize();
                None
            }
        }
    }

    /// 错误恢复：跳过若干 Token 直到遇到同步 Token
    fn synchronize(&mut self) {
        while !self.is_eof() {
            // 遇到这些 Token 时认为可以安全继续解析
            if self.check(TokenKind::Semicolon)
                || self.check(TokenKind::RBrace)
                || self.check_keyword(&["结束", "end", "jieshu"])
                || self.check_keyword(&["函数", "func", "hanshu"])
                || self.check_keyword(&["结构", "struct", "jiegou"])
                || self.check_keyword(&["模块", "module", "mokuai"])
            {
                self.advance(); // 消耗同步 Token
                return;
            }
            self.advance();
        }
    }
}
```

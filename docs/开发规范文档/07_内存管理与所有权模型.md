# 7. 内存管理与所有权模型

## 7.1 内存管理概述

幻语采用**无垃圾回收（No-GC）**的内存管理策略，提供两种可选的内存安全模式：

| 模式 | 触发方式 | 内存安全保证 | 运行时开销 |
|------|---------|-------------|-----------|
| **默认模式** | 默认（无特殊标志） | 编译期无检查，依赖开发者显式管理；调试构建提供边界检查 | 零额外开销 |
| **所有权模式** | `使用 所有权` 或 `--所有权` | 编译期借用检查，保证无悬垂指针、无数据竞争 | 零额外开销（编译期检查） |

两种模式均**禁止隐式堆分配**，所有动态内存分配必须通过分配器显式完成。这一设计使幻语能够胜任从裸机嵌入式到高性能服务器的全场景内存管理需求。

## 7.2 分配器接口（Allocator）

### 7.2.1 分配器特征定义

```rust
/// 分配器特征，所有堆内存操作必须通过实现了此特征的分配器进行
pub trait Allocator {
    /// 分配至少 `size` 字节、对齐为 `align` 的内存块
    /// 
    /// # 安全性
    /// - 返回的指针必须满足对齐要求
    /// - 若分配失败，返回 `std::ptr::null_mut()`
    fn alloc(&self, size: usize, align: usize) -> *mut u8;

    /// 释放之前由 `alloc` 分配的 `size` 字节、对齐为 `align` 的内存块
    /// 
    /// # 安全性
    /// - `ptr` 必须由同一分配器的 `alloc` 返回
    /// - `size` 和 `align` 必须与分配时一致
    fn free(&self, ptr: *mut u8, size: usize, align: usize);

    /// 重新分配内存块，将原有 `old_size` 字节的数据复制到新块，并释放旧块
    /// 
    /// # 安全性
    /// - `ptr` 必须由同一分配器的 `alloc` 或 `realloc` 返回
    /// - 若返回 `null`，则原有内存块保持不变
    fn realloc(&self, ptr: *mut u8, old_size: usize, new_size: usize, align: usize) -> *mut u8;
}
```

### 7.2.2 全局默认分配器

```rust
/// 全局默认分配器（使用系统 malloc/free）
pub struct GlobalAllocator;

impl Allocator for GlobalAllocator {
    fn alloc(&self, size: usize, align: usize) -> *mut u8 {
        // 在有操作系统环境下调用 libc::malloc 或 std::alloc::alloc
        #[cfg(feature = "std")]
        {
            let layout = std::alloc::Layout::from_size_align(size, align).unwrap();
            std::alloc::alloc(layout)
        }
        #[cfg(not(feature = "std"))]
        {
            // 裸机环境下需链接自定义的 malloc 实现
            extern "C" { fn malloc(size: usize) -> *mut u8; }
            unsafe { malloc(size) }
        }
    }

    fn free(&self, ptr: *mut u8, size: usize, align: usize) {
        #[cfg(feature = "std")]
        {
            let layout = std::alloc::Layout::from_size_align(size, align).unwrap();
            std::alloc::dealloc(ptr, layout);
        }
        #[cfg(not(feature = "std"))]
        {
            extern "C" { fn free(ptr: *mut u8); }
            unsafe { free(ptr); }
        }
    }

    fn realloc(&self, ptr: *mut u8, old_size: usize, new_size: usize, align: usize) -> *mut u8 {
        #[cfg(feature = "std")]
        {
            let layout = std::alloc::Layout::from_size_align(old_size, align).unwrap();
            std::alloc::realloc(ptr, layout, new_size)
        }
        #[cfg(not(feature = "std"))]
        {
            extern "C" { fn realloc(ptr: *mut u8, size: usize) -> *mut u8; }
            unsafe { realloc(ptr, new_size) }
        }
    }
}

/// 获取全局默认分配器的单例
pub fn default_allocator() -> &'static dyn Allocator {
    static GLOBAL: GlobalAllocator = GlobalAllocator;
    &GLOBAL
}
```

### 7.2.3 嵌入式分配器示例（首次适应）

```rust
/// 简单的首次适应分配器，适用于裸机嵌入式系统
pub struct FirstFitAllocator {
    /// 堆起始地址
    heap_start: usize,
    /// 堆总大小
    heap_size: usize,
    /// 空闲链表头指针
    free_list: *mut FreeBlock,
}

struct FreeBlock {
    size: usize,
    next: *mut FreeBlock,
}

impl FirstFitAllocator {
    /// 使用指定的静态内存区域初始化分配器
    pub const fn new(heap_start: usize, heap_size: usize) -> Self {
        Self {
            heap_start,
            heap_size,
            free_list: core::ptr::null_mut(),
        }
    }

    /// 初始化空闲链表（需在运行时调用一次）
    pub unsafe fn init(&mut self) {
        let block = self.heap_start as *mut FreeBlock;
        (*block).size = self.heap_size;
        (*block).next = core::ptr::null_mut();
        self.free_list = block;
    }
}

impl Allocator for FirstFitAllocator {
    fn alloc(&self, size: usize, align: usize) -> *mut u8 {
        // 遍历空闲链表，找到第一个足够大的块
        let mut current = self.free_list;
        let mut prev: *mut FreeBlock = core::ptr::null_mut();

        while !current.is_null() {
            let block = unsafe { &mut *current };
            let block_addr = current as usize;
            let aligned_addr = (block_addr + align - 1) & !(align - 1);
            let padding = aligned_addr - block_addr;

            if block.size >= size + padding {
                // 找到合适的块
                let remaining = block.size - size - padding;
                if remaining > core::mem::size_of::<FreeBlock>() {
                    // 分割剩余部分为新的空闲块
                    let new_free = (aligned_addr + size) as *mut FreeBlock;
                    unsafe {
                        (*new_free).size = remaining;
                        (*new_free).next = block.next;
                    }
                    if prev.is_null() {
                        self.free_list = new_free;
                    } else {
                        unsafe { (*prev).next = new_free; }
                    }
                } else {
                    // 整个块分配
                    if prev.is_null() {
                        self.free_list = block.next;
                    } else {
                        unsafe { (*prev).next = block.next; }
                    }
                }
                return aligned_addr as *mut u8;
            }
            prev = current;
            current = block.next;
        }
        core::ptr::null_mut()
    }

    fn free(&self, ptr: *mut u8, size: usize, align: usize) {
        // 将释放的块插入空闲链表，并尝试合并相邻块
        let block = ptr as *mut FreeBlock;
        unsafe {
            (*block).size = size;
            (*block).next = self.free_list;
        }
        self.free_list = block;
        // 合并逻辑省略
    }

    fn realloc(&self, ptr: *mut u8, old_size: usize, new_size: usize, align: usize) -> *mut u8 {
        if new_size <= old_size {
            return ptr;
        }
        let new_ptr = self.alloc(new_size, align);
        if !new_ptr.is_null() {
            unsafe {
                core::ptr::copy_nonoverlapping(ptr, new_ptr, old_size);
            }
            self.free(ptr, old_size, align);
        }
        new_ptr
    }
}
```

## 7.3 所有权系统（可选启用）

所有权系统通过编译期静态分析保证内存安全，核心概念包括**所有权（Ownership）**、**借用（Borrowing）**和**生命周期（Lifetime）**。

### 7.3.1 所有权规则

1. **每个值有且仅有一个所有者**。
2. **当所有者离开作用域，值被自动释放**。
3. **赋值、函数传参、函数返回默认移动所有权**。
4. **实现了 `复制` 特征的类型在赋值时复制而非移动**。

### 7.3.2 复制特征（Copy）

幻语标准库为以下类型自动实现 `复制` 特征：
- 所有基本数值类型（`整数`、`整数8`~`整数64`、`无符号8`~`无符号64`、`浮点32`、`浮点64`）
- `布尔`、`字符`
- `指针<T>`（指针本身是值类型）
- 由纯 `复制` 类型组成的元组

开发者可通过 `derive(复制)` 为自定义结构体实现复制特征，前提是所有字段均为 `复制` 类型。

```hl
@派生(复制)
结构 点
    字段 横坐标: 整数
    字段 纵坐标: 整数
结束
```

### 7.3.3 借用与引用

引用使用 `借用` 关键字创建，分为**只读借用**（`&T`）和**可变借用**（`&mut T`）。

**借用规则**：
- 任意时刻，只能拥有**一个可变借用**，或**任意数量的只读借用**。
- 引用必须始终有效（生命周期不超出被借用值）。

```hl
    -- 可变借用（同时只能存在一个）
```

当编译器无法自动推断引用的有效范围时，需要显式标注生命周期参数。生命周期参数以 `'` 开头。


**生命周期省略规则**：
1. 每个引用参数自动获得独立的生命周期参数。
2. 若只有一个输入生命周期参数，则输出生命周期被赋予该参数。
3. 若方法有 `&self` 或 `&mut self` 参数，则输出生命周期被赋予 `self` 的生命周期。

### 7.3.5 借用检查器实现框架

```rust
pub struct BorrowChecker {
    /// 变量唯一标识 → 借用状态
    loans: HashMap<VarId, LoanState>,
    /// 当前作用域深度
    scope_depth: usize,
    /// 作用域栈（用于追踪变量离开作用域）
    scope_vars: Vec<Vec<VarId>>,
}

#[derive(Debug, Clone)]
enum LoanState {
    /// 未被借用
    None,
    /// 被只读借用 n 次
    Shared(usize),
    /// 被可变借用
    Mutable,
}

impl BorrowChecker {
    pub fn new() -> Self {
        Self {
            loans: HashMap::new(),
            scope_depth: 0,
            scope_vars: vec![Vec::new()],
        }
    }

    /// 进入新作用域
    pub fn enter_scope(&mut self) {
        self.scope_depth += 1;
        self.scope_vars.push(Vec::new());
    }

    /// 离开作用域，释放该作用域内声明的变量
    pub fn exit_scope(&mut self) {
        for var_id in self.scope_vars.pop().unwrap() {
            self.loans.remove(&var_id);
        }
        self.scope_depth -= 1;
    }

    /// 声明新变量
    pub fn declare_var(&mut self, var_id: VarId) {
        self.loans.insert(var_id, LoanState::None);
        self.scope_vars.last_mut().unwrap().push(var_id);
    }

    /// 检查只读借用
    pub fn check_shared_borrow(&mut self, var_id: VarId, span: SourceSpan) -> Result<(), BorrowError> {
        match self.loans.get(&var_id) {
            Some(LoanState::Mutable) => Err(BorrowError::CannotBorrowSharedWhileMutable { span }),
            Some(LoanState::Shared(n)) => {
                self.loans.insert(var_id, LoanState::Shared(n + 1));
                Ok(())
            }
            Some(LoanState::None) => {
                self.loans.insert(var_id, LoanState::Shared(1));
                Ok(())
            }
            None => Err(BorrowError::UseAfterMove { span }),
        }
    }

    /// 检查可变借用
    pub fn check_mutable_borrow(&mut self, var_id: VarId, span: SourceSpan) -> Result<(), BorrowError> {
        match self.loans.get(&var_id) {
            Some(LoanState::Mutable) => Err(BorrowError::AlreadyMutableBorrowed { span }),
            Some(LoanState::Shared(_)) => Err(BorrowError::CannotBorrowMutableWhileShared { span }),
            Some(LoanState::None) => {
                self.loans.insert(var_id, LoanState::Mutable);
                Ok(())
            }
            None => Err(BorrowError::UseAfterMove { span }),
        }
    }

    /// 释放借用（当引用离开作用域时调用）
    pub fn release_borrow(&mut self, var_id: VarId, mutable: bool) {
        if let Some(state) = self.loans.get_mut(&var_id) {
            match state {
                LoanState::Shared(n) if *n > 1 => *state = LoanState::Shared(*n - 1),
                LoanState::Shared(_) | LoanState::Mutable => *state = LoanState::None,
                _ => {}
            }
        }
    }

    /// 检查变量移动
    pub fn check_move(&mut self, var_id: VarId, span: SourceSpan) -> Result<(), BorrowError> {
        match self.loans.get(&var_id) {
            Some(LoanState::Shared(_)) => Err(BorrowError::CannotMoveWhileBorrowed { span }),
            Some(LoanState::Mutable) => Err(BorrowError::CannotMoveWhileBorrowed { span }),
            Some(LoanState::None) => {
                self.loans.remove(&var_id);
                Ok(())
            }
            None => Err(BorrowError::UseAfterMove { span }),
        }
    }
}

#[derive(Debug, Clone, PartialEq)]
pub enum BorrowError {
    UseAfterMove { span: SourceSpan },
    CannotBorrowSharedWhileMutable { span: SourceSpan },
    CannotBorrowMutableWhileShared { span: SourceSpan },
    AlreadyMutableBorrowed { span: SourceSpan },
    CannotMoveWhileBorrowed { span: SourceSpan },
}
```

```rust
pub struct Box<T> {
    ptr: *mut T,
    allocator: *mut dyn Allocator,

impl<T> Drop for Box<T> {

            let alloc = &*self.allocator;
```

```rust
#[repr(C)]
struct MapEntry {
    hash: u64,

    // 后跟 key 和 value 数据
}
```

**负载因子**：0.75，超过时扩容为原来的 2 倍。

## 7.6 编译期内存管理优化

### 7.6.1 栈分配优化

对于大小在编译期已知、且生命周期明确的小型值，编译器会优先分配在栈上，避免堆分配开销。

### 7.6.2 逃逸分析

编译器进行逃逸分析，若检测到某值不会逃逸出当前函数，则将其分配在栈上而非堆上。

### 7.6.3 内联与去虚拟化

对于通过特征对象调用的方法，若编译器能确定具体类型，则进行去虚拟化，直接调用具体实现，消除虚表开销。

## 7.1 内存管理概述

幻语采用**无垃圾回收（No‑GC）**的内存管理策略，提供两种可选的内存安全模式：

| 模式 | 触发方式 | 内存安全保证 | 运行时开销 |
|------|---------|-------------|-----------|
| **默认模式** | 默认（无特殊标志） | 编译期无检查，依赖开发者显式管理；调试构建提供边界检查 | 零额外开销 |
| **所有权模式** | `使用 所有权` 或 `--ownership` | 编译期借用检查，保证无悬垂指针、无数据竞争 | 零额外开销（编译期检查） |

两种模式均**禁止隐式堆分配**，所有动态内存分配必须通过分配器显式完成。这一设计使幻语能够胜任从裸机嵌入式到高性能服务器的全场景内存管理需求。

## 7.2 分配器接口

### 7.2.1 分配器特征定义

```rust
/// 分配器特征，所有堆内存操作必须通过实现了此特征的分配器进行
pub trait Allocator {
    /// 分配至少 `size` 字节、对齐为 `align` 的内存块
    /// 
    /// # 安全性
    /// - 返回的指针必须满足对齐要求
    /// - 若分配失败，返回 `std::ptr::null_mut()`
    fn alloc(&self, size: usize, align: usize) -> *mut u8;

    /// 释放之前由 `alloc` 分配的 `size` 字节、对齐为 `align` 的内存块
    fn free(&self, ptr: *mut u8, size: usize, align: usize);

    /// 重新分配内存块，将原有 `old_size` 字节的数据复制到新块，并释放旧块
    fn realloc(&self, ptr: *mut u8, old_size: usize, new_size: usize, align: usize) -> *mut u8;
}
```

### 7.2.2 全局默认分配器

```rust
/// 全局默认分配器（使用系统 malloc/free）
pub struct GlobalAllocator;

impl Allocator for GlobalAllocator {
    fn alloc(&self, size: usize, align: usize) -> *mut u8 {
        #[cfg(feature = "std")]
        {
            let layout = std::alloc::Layout::from_size_align(size, align).unwrap();
            std::alloc::alloc(layout)
        }
        #[cfg(not(feature = "std"))]
        {
            extern "C" { fn malloc(size: usize) -> *mut u8; }
            unsafe { malloc(size) }
        }
    }

    fn free(&self, ptr: *mut u8, size: usize, align: usize) {
        #[cfg(feature = "std")]
        {
            let layout = std::alloc::Layout::from_size_align(size, align).unwrap();
            std::alloc::dealloc(ptr, layout);
        }
        #[cfg(not(feature = "std"))]
        {
            extern "C" { fn free(ptr: *mut u8); }
            unsafe { free(ptr); }
        }
    }

    fn realloc(&self, ptr: *mut u8, old_size: usize, new_size: usize, align: usize) -> *mut u8 {
        #[cfg(feature = "std")]
        {
            let layout = std::alloc::Layout::from_size_align(old_size, align).unwrap();
            std::alloc::realloc(ptr, layout, new_size)
        }
        #[cfg(not(feature = "std"))]
        {
            extern "C" { fn realloc(ptr: *mut u8, size: usize) -> *mut u8; }
            unsafe { realloc(ptr, new_size) }
        }
    }
}

/// 获取全局默认分配器的单例
pub fn default_allocator() -> &'static dyn Allocator {
    static GLOBAL: GlobalAllocator = GlobalAllocator;
    &GLOBAL
}
```

### 7.2.3 嵌入式分配器示例（首次适应）

```rust
/// 简单的首次适应分配器，适用于裸机嵌入式系统
pub struct FirstFitAllocator {
    heap_start: usize,
    heap_size: usize,
    free_list: *mut FreeBlock,
}

struct FreeBlock {
    size: usize,
    next: *mut FreeBlock,
}

impl FirstFitAllocator {
    pub const fn new(heap_start: usize, heap_size: usize) -> Self {
        Self { heap_start, heap_size, free_list: core::ptr::null_mut() }
    }

    pub unsafe fn init(&mut self) {
        let block = self.heap_start as *mut FreeBlock;
        (*block).size = self.heap_size;
        (*block).next = core::ptr::null_mut();
        self.free_list = block;
    }
}

impl Allocator for FirstFitAllocator {
    fn alloc(&self, size: usize, align: usize) -> *mut u8 {
        let mut current = self.free_list;
        let mut prev: *mut FreeBlock = core::ptr::null_mut();
        while !current.is_null() {
            let block = unsafe { &mut *current };
            let block_addr = current as usize;
            let aligned_addr = (block_addr + align - 1) & !(align - 1);
            let padding = aligned_addr - block_addr;
            if block.size >= size + padding {
                let remaining = block.size - size - padding;
                if remaining > core::mem::size_of::<FreeBlock>() {
                    let new_free = (aligned_addr + size) as *mut FreeBlock;
                    unsafe { (*new_free).size = remaining; (*new_free).next = block.next; }
                    if prev.is_null() { self.free_list = new_free; }
                    else { unsafe { (*prev).next = new_free; } }
                } else {
                    if prev.is_null() { self.free_list = block.next; }
                    else { unsafe { (*prev).next = block.next; } }
                }
                return aligned_addr as *mut u8;
            }
            prev = current;
            current = block.next;
        }
        core::ptr::null_mut()
    }

    fn free(&self, ptr: *mut u8, size: usize, _align: usize) {
        let block = ptr as *mut FreeBlock;
        unsafe { (*block).size = size; (*block).next = self.free_list; }
        self.free_list = block;
        // 合并逻辑略
    }

    fn realloc(&self, ptr: *mut u8, old_size: usize, new_size: usize, align: usize) -> *mut u8 {
        if new_size <= old_size { return ptr; }
        let new_ptr = self.alloc(new_size, align);
        if !new_ptr.is_null() { unsafe { core::ptr::copy_nonoverlapping(ptr, new_ptr, old_size); } }
        self.free(ptr, old_size, align);
        new_ptr
    }
}
```

## 7.3 所有权系统（可选启用）

所有权系统通过编译期静态分析保证内存安全，核心概念包括**所有权**、**借用**和**生命周期**。

### 7.3.1 所有权规则

- **每个值有且仅有一个所有者**。
- **当所有者离开作用域，值被自动释放**。
- **赋值、函数传参、函数返回默认移动所有权**。
- **实现了 `复制` 特征的类型在赋值时复制而非移动**。

### 7.3.2 复制特征（Copy）

幻语标准库为以下类型自动实现 `复制` 特征：
- 所有基本数值类型
- `布尔`、`字符`
- `指针<T>`（指针本身是值类型）
- 由纯 `复制` 类型组成的元组

开发者可通过 `@派生(复制)` 为自定义结构体实现复制特征，前提是所有字段均为 `复制` 类型。

### 7.3.3 借用与引用

引用使用 `借用` 关键字创建，分为**只读借用**（`&T`）和**可变借用**（`&mut T`）。

**借用规则**：
- 任意时刻，只能拥有**一个可变借用**，或**任意数量的只读借用**。
- 引用必须始终有效（生命周期不超出被借用值）。

```hl
使用 所有权

函数 演示()
开始
    令 数据 = 列表<整数>.新建()
    数据.追加(42)
    
    令 只读 = 借用 数据        // 只读借用
    显示(只读.长度())
    
    令 可变 = 借用 可变 数据   // 可变借用，同时只能存在一个
    可变.追加(100)
结束
```

### 7.3.4 生命周期标注

当编译器无法自动推断引用有效期时，需要显式标注生命周期参数。生命周期参数以 `'` 开头。

```hl
函数 最长<'甲>(甲: 借用 '甲 字符串, 乙: 借用 '甲 字符串) -> 借用 '甲 字符串
开始
    若 甲.长度() > 乙.长度() 则 返回 甲 否则 返回 乙 结束
结束
```

**生命周期省略规则**：
1. 每个引用参数自动获得独立的生命周期参数。
2. 若只有一个输入生命周期参数，则输出生命周期被赋予该参数。
3. 若方法有 `&self` 或 `&mut self` 参数，则输出生命周期被赋予 `self` 的生命周期。

### 7.3.5 借用检查器实现框架

```rust
pub struct BorrowChecker {
    loans: HashMap<VarId, LoanState>,
    scope_depth: usize,
    scope_vars: Vec<Vec<VarId>>,
}

#[derive(Debug, Clone)]
enum LoanState {
    None,
    Shared(usize),
    Mutable,
}

impl BorrowChecker {
    pub fn new() -> Self { Self { loans: HashMap::new(), scope_depth: 0, scope_vars: vec![Vec::new()] } }

    pub fn enter_scope(&mut self) {
        self.scope_depth += 1;
        self.scope_vars.push(Vec::new());
    }

    pub fn exit_scope(&mut self) {
        for var_id in self.scope_vars.pop().unwrap() { self.loans.remove(&var_id); }
        self.scope_depth -= 1;
    }

    pub fn declare_var(&mut self, var_id: VarId) {
        self.loans.insert(var_id, LoanState::None);
        self.scope_vars.last_mut().unwrap().push(var_id);
    }

    pub fn check_shared_borrow(&mut self, var_id: VarId, span: SourceSpan) -> Result<(), BorrowError> {
        match self.loans.get(&var_id) {
            Some(LoanState::Mutable) => Err(BorrowError::CannotBorrowSharedWhileMutable { span }),
            Some(LoanState::Shared(n)) => { self.loans.insert(var_id, LoanState::Shared(n + 1)); Ok(()) }
            Some(LoanState::None) => { self.loans.insert(var_id, LoanState::Shared(1)); Ok(()) }
            None => Err(BorrowError::UseAfterMove { span }),
        }
    }

    pub fn check_mutable_borrow(&mut self, var_id: VarId, span: SourceSpan) -> Result<(), BorrowError> {
        match self.loans.get(&var_id) {
            Some(LoanState::Mutable) => Err(BorrowError::AlreadyMutableBorrowed { span }),
            Some(LoanState::Shared(_)) => Err(BorrowError::CannotBorrowMutableWhileShared { span }),
            Some(LoanState::None) => { self.loans.insert(var_id, LoanState::Mutable); Ok(()) }
            None => Err(BorrowError::UseAfterMove { span }),
        }
    }

    pub fn check_move(&mut self, var_id: VarId, span: SourceSpan) -> Result<(), BorrowError> {
        match self.loans.get(&var_id) {
            Some(LoanState::Shared(_)) | Some(LoanState::Mutable) => Err(BorrowError::CannotMoveWhileBorrowed { span }),
            Some(LoanState::None) => { self.loans.remove(&var_id); Ok(()) }
            None => Err(BorrowError::UseAfterMove { span }),
        }
    }
}

#[derive(Debug, Clone, PartialEq)]
pub enum BorrowError {
    UseAfterMove { span: SourceSpan },
    CannotBorrowSharedWhileMutable { span: SourceSpan },
    CannotBorrowMutableWhileShared { span: SourceSpan },
    AlreadyMutableBorrowed { span: SourceSpan },
    CannotMoveWhileBorrowed { span: SourceSpan },
}
```

## 7.4 智能指针

幻语标准库提供以下智能指针，基于所有权系统构建，实现自动资源管理。

### 7.4.1 盒子（Box）

`盒子<T>` 将值分配在堆上，离开作用域自动释放。内部表示为指针 + 分配器。

### 7.4.2 引用计数（Rc 与 Arc）

`Rc<T>` 提供单线程共享所有权，`Arc<T>` 使用原子操作实现线程安全的共享所有权。弱引用 `Weak<T>` 用于打破循环。

### 7.4.3 互斥锁（Mutex）与读写锁（RwLock）

`Mutex<T>` 提供内部可变性的并发保护，`RwLock<T>` 支持多读单写。两者均依赖于所有权系统与并发标记。

## 7.5 运行时数据结构内存布局

### 7.5.1 列表（List）

```rust
#[repr(C)]
pub struct HuanList {
    pub data: *mut u8,
    pub length: usize,
    pub capacity: usize,
    pub elem_size: usize,
    pub allocator: *mut dyn Allocator,
}
```
扩容策略：初始容量 0，首次追加分配容量 4，之后每次加倍。

### 7.5.2 字符串（String）

```rust
#[repr(C)]
pub struct HuanString {
    pub data: *mut u8,
    pub length: usize,
    pub allocator: *mut dyn Allocator,
}
```
字符串内部保证为有效的 UTF-8 编码，所有修改操作都会验证编码有效性。

### 7.5.3 字典（Map）

采用开放寻址法的哈希表，使用二次探测解决冲突。存储结构包含桶数组、容量、键/值大小、哈希函数、相等比较函数等字段。负载因子默认 0.75，超过时扩容为 2 倍。

## 7.6 编译期内存管理优化

### 7.6.1 栈分配优化

对于大小在编译期已知、且生命周期明确的小型值，编译器会优先分配在栈上，避免堆分配开销。

### 7.6.2 逃逸分析

编译器进行逃逸分析，若检测到某值不会逃逸出当前函数，则将其分配在栈上而非堆上。

### 7.6.3 内联与去虚拟化

对于通过特征对象调用的方法，若编译器能确定具体类型，则进行去虚拟化，直接调用具体实现，消除虚表开销。

## 7.7 与并发模型的集成

在启用所有权模式时，并发 API 与借用检查器协同工作。`Send` 和 `Sync` 特征自动实现，禁止产生数据竞争。智能指针如 `Arc` 配合 `Mutex` 可实现跨线程共享可变状态。

## 本章总结

本章详细描述了幻语内存管理与所有权模型的完整设计规范，包括：

1. **内存管理概述**：采用无垃圾回收（No-GC）策略，提供默认模式和所有权模式两种选择
2. **分配器接口**：Allocator trait定义统一的内存分配接口，包括分配、释放、重新分配等操作
3. **默认分配器**：SystemAllocator使用标准库分配器，BumpAllocator提供高效的线性分配模式
4. **列表容器**：HuanList实现动态列表，支持push、pop、len、is_empty等操作
5. **借用检查器**：BorrowChecker在编译期验证借用的合法性，防止数据竞争和悬垂指针
6. **所有权规则**：包括移动语义、Copy语义、可变借用、共享借用等规则的完整定义
7. **测试用例**：提供了分配器、列表和借用检查器的测试示例

这些规范为幻语的内存管理提供了完整的实现指导，确保了内存安全和性能的双重目标。

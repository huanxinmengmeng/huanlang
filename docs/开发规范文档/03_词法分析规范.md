# 3. 词法分析规范

## 3.1 词法分析器概述

词法分析器负责将 UTF-8 编码的源代码字符串转换为 Token 流。它是编译器前端的第一阶段，为语法解析器提供结构化的输入。

**主要职责**：
- 跳过空白字符和注释
- 识别关键词、标识符、字面量、符号
- 跟踪源码位置（行号、列号、字节偏移）
- 处理词法错误并尽可能恢复

```rust
        '\u{3400}'..='\u{4DBF}' |       // CJK 扩展 A

        '\u{2A700}'..='\u{2B73F}' |     // CJK 扩展 C
        '\u{2B740}'..='\u{2B81F}' |     // CJK 扩展 D
        '\u{2B820}'..='\u{2CEAF}' |     // CJK 扩展 E
        '\u{2CEB0}'..='\u{2EBEF}' |     // CJK 扩展 F
        '\u{30000}'..='\u{3134F}' |     // CJK 扩展 G
        '\u{F900}'..='\u{FAFF}' |       // CJK 兼容表意字符

/// 判断是否为标识符合法字符（首字符或后续字符）
fn is_identifier_char(ch: char) -> bool {
    ch.is_ascii_alphanumeric() || ch == '_' || is_cjk_char(ch)
}

/// 判断是否为标识符首字符（不能为数字）
fn is_identifier_start(ch: char) -> bool {
    ch.is_ascii_alphabetic() || ch == '_' || is_cjk_char(ch)
}

/// 判断是否为空白字符（不含换行）
fn is_whitespace(ch: char) -> bool {
    ch == ' ' || ch == '\t' || ch == '\r'
}

/// 判断是否为数字字符（十进制）
fn is_digit(ch: char) -> bool {
    ch.is_ascii_digit()
}

/// 判断是否为十六进制数字字符
fn is_hex_digit(ch: char) -> bool {
    ch.is_ascii_hexdigit()
}
```

## 3.6 词法分析器结构

```rust
pub struct Lexer<'a> {
    /// 原始源代码字符串
    source: &'a str,
    /// 转换为字符数组以便按 Unicode 标量值遍历
    chars: Vec<char>,
    /// 当前字符索引
    pos: usize,
    /// 当前行号（1-based）
    line: usize,
    /// 当前列号（1-based，按 UTF-8 字符计数）
    column: usize,
    /// 关键词映射表
    keyword_table: KeywordTable,
    /// 收集的词法错误
    errors: Vec<LexError>,
}

impl<'a> Lexer<'a> {
    pub fn new(source: &'a str) -> Self {
        Self {
            source,
            chars: source.chars().collect(),
            pos: 0,
            line: 1,
            column: 1,
            keyword_table: KeywordTable::new(),
            errors: Vec::new(),
        }
    }
    
    /// 获取当前源码位置
    fn current_position(&self) -> SourcePosition {
        SourcePosition::new(self.pos, self.line, self.column)
    }
    
    /// 获取当前字符
    fn current_char(&self) -> Option<char> {
        self.chars.get(self.pos).copied()
    }
    
    /// 获取下一个字符（不移动位置）
    fn peek_char(&self) -> Option<char> {
        self.chars.get(self.pos + 1).copied()
    }
    
    /// 获取下下个字符
    fn peek_char2(&self) -> Option<char> {
        self.chars.get(self.pos + 2).copied()
    }
    
    /// 前进一个字符，返回当前字符并更新位置
    fn advance(&mut self) -> Option<char> {
        if let Some(ch) = self.current_char() {
            self.pos += 1;
            if ch == '\n' {
                self.line += 1;
                self.column = 1;
            } else {
                self.column += 1;
            }
            Some(ch)
        } else {
            None
        }
    }
    
    /// 检查是否到达文件末尾
    fn is_eof(&self) -> bool {
        self.pos >= self.chars.len()
    }
}
```

## 3.7 主词法分析循环

```rust
impl<'a> Lexer<'a> {
    /// 将整个源文件转换为 Token 流
    pub fn tokenize(&mut self) -> (Vec<Token>, Vec<LexError>) {
        let mut tokens = Vec::new();
        while !self.is_eof() {
            match self.next_token() {
                Ok(Some(token)) => tokens.push(token),
                Ok(None) => continue,
                Err(e) => {
                    self.errors.push(e);
                    // 尝试恢复：跳过当前字符继续
                    self.advance();
                }
            }
        }
        tokens.push(Token::eof(self.current_position()));
        (tokens, std::mem::take(&mut self.errors))
    }
    
    /// 解析下一个 Token
    fn next_token(&mut self) -> Result<Option<Token>, LexError> {
        // 跳过空白和注释
        self.skip_whitespace_and_comments();
        
        if self.is_eof() {
            return Ok(None);
        }
        
        let start = self.current_position();
        let ch = self.current_char().unwrap();
        
        let token = match ch {
            // 数字字面量
            '0'..='9' => self.lex_number(start)?,
            // 字符串字面量
            '"' => self.lex_string(start)?,
            // 字符字面量
            '\'' => self.lex_char(start)?,
            // 标识符或关键词
            c if is_identifier_start(c) => self.lex_identifier_or_keyword(start)?,
            // 符号
            _ => self.lex_symbol(start)?,
        };
        
        Ok(Some(token))
    }
}
```

## 3.8 空白与注释跳过

```rust
impl<'a> Lexer<'a> {
    fn skip_whitespace_and_comments(&mut self) {
        while let Some(ch) = self.current_char() {
            match ch {
                // 空白字符
                c if is_whitespace(c) => {
                    self.advance();
                }
                // 换行符（独立处理，可记录位置）
                '\n' => {
                    self.advance();
                }
                // 单行注释 --
                '-' if self.peek_char() == Some('-') => {
                    self.advance(); // 跳过第一个 '-'
                    self.advance(); // 跳过第二个 '-'
                    self.skip_line_comment();
                }
                // 多行注释 /*
                '/' if self.peek_char() == Some('*') => {
                    self.advance(); // 跳过 '/'
                    self.advance(); // 跳过 '*'
                    self.skip_block_comment();
                }
                _ => break,
            }
        }
    }
    
    fn skip_line_comment(&mut self) {
        while let Some(ch) = self.current_char() {
            if ch == '\n' {
                break;
            }
            self.advance();
        }
    }
    
    fn skip_block_comment(&mut self) {
        let mut depth = 1;
        while depth > 0 {
            if self.is_eof() {
                // 未闭合的块注释，报错
                return;
            }
            match (self.current_char(), self.peek_char()) {
                (Some('/'), Some('*')) => {
                    self.advance();
                    self.advance();
                    depth += 1;
                }
                (Some('*'), Some('/')) => {
                    self.advance();
                    self.advance();
                    depth -= 1;
                }
                _ => {
                    self.advance();
                }
            }
        }
    }
}
```

## 3.9 数字字面量解析

```rust
impl<'a> Lexer<'a> {
    fn lex_number(&mut self, start: SourcePosition) -> Result<Token, LexError> {
        let mut lexeme = String::new();
        let mut is_float = false;
        let mut base = 10;
        
        // 处理进制前缀
        if self.current_char() == Some('0') {
            match self.peek_char() {
                Some('x') | Some('X') => {
                    base = 16;
                    lexeme.push(self.advance().unwrap()); // '0'
                    lexeme.push(self.advance().unwrap()); // 'x'
                }
                Some('b') | Some('B') => {
                    base = 2;
                    lexeme.push(self.advance().unwrap());
                    lexeme.push(self.advance().unwrap());
                }
                Some('o') | Some('O') => {
                    base = 8;
                    lexeme.push(self.advance().unwrap());
                    lexeme.push(self.advance().unwrap());
                }
                _ => {}
            }
        }
        
        while let Some(ch) = self.current_char() {
            match ch {
                '_' => {
                    // 忽略分隔符
                    lexeme.push(ch);
                    self.advance();
                }
                '0'..='9' => {
                    lexeme.push(ch);
                    self.advance();
                }
                'a'..='f' | 'A'..='F' if base == 16 => {
                    lexeme.push(ch);
                    self.advance();
                }
                '.' if base == 10 && !is_float => {
                    // 检查是否真的是浮点数的小数点（不是范围操作符）
                    if let Some(next) = self.peek_char() {
                        if next.is_ascii_digit() {
                            is_float = true;
                            lexeme.push(ch);
                            self.advance();
                            continue;
                        }
                    }
                    break;
                }
                'e' | 'E' if base == 10 => {
                    is_float = true;
                    lexeme.push(ch);
                    self.advance();
                    // 处理指数符号
                    if let Some(sign) = self.current_char() {
                        if sign == '+' || sign == '-' {
                            lexeme.push(sign);
                            self.advance();
                        }
                    }
                }
                _ => break,
            }
        }
        
        let end = self.current_position();
        let span = SourceSpan::new(start, end);
        
        let kind = if is_float {
            let val: f64 = lexeme.parse()
                .map_err(|_| LexError::NumericOverflow { pos: start })?;
            TokenKind::FloatLit(val)
        } else {
            let val = match base {
                16 => i64::from_str_radix(lexeme.trim_start_matches("0x").trim_start_matches("0X"), 16),
                2 => i64::from_str_radix(lexeme.trim_start_matches("0b").trim_start_matches("0B"), 2),
                8 => i64::from_str_radix(lexeme.trim_start_matches("0o").trim_start_matches("0O"), 8),
                _ => lexeme.parse::<i64>(),
            }.map_err(|_| LexError::NumericOverflow { pos: start })?;
            TokenKind::IntLit(val)
        };
        
        Ok(Token::new(kind, span, lexeme))
    }
}
```

## 3.10 字符串字面量解析

```rust
impl<'a> Lexer<'a> {
    fn lex_string(&mut self, start: SourcePosition) -> Result<Token, LexError> {
        self.advance(); // 跳过开头的 "
        let mut content = String::new();
        let mut lexeme = String::new();
        lexeme.push('"');
        
        while let Some(ch) = self.current_char() {
            if ch == '"' {
                lexeme.push(ch);
                self.advance();
                let end = self.current_position();
                let span = SourceSpan::new(start, end);
                return Ok(Token::new(TokenKind::StringLit(content), span, lexeme));
            }
            
            if ch == '\\' {
                lexeme.push(ch);
                self.advance();
                let escaped = self.parse_escape_sequence(start)?;
                content.push(escaped);
                lexeme.push(escaped);  // 简化：实际应记录原始转义序列
                continue;
            }
            
            // 普通字符
            lexeme.push(ch);
            content.push(ch);
            self.advance();
        }
        
        Err(LexError::UnclosedString { start })
    }
    
    /// 解析转义序列，返回对应的字符
    fn parse_escape_sequence(&mut self, pos: SourcePosition) -> Result<char, LexError> {
        match self.current_char() {
            Some('n') => { self.advance(); Ok('\n') }
            Some('r') => { self.advance(); Ok('\r') }
            Some('t') => { self.advance(); Ok('\t') }
            Some('\\') => { self.advance(); Ok('\\') }
            Some('"') => { self.advance(); Ok('"') }
            Some('\'') => { self.advance(); Ok('\'') }
            Some('0') => { self.advance(); Ok('\0') }
            Some('u') => {
                self.advance();
                if self.current_char() != Some('{') {
                    return Err(LexError::InvalidEscape { seq: "\\u".to_string(), pos });
                }
                self.advance(); // 跳过 '{'
                let mut hex = String::new();
                while let Some(ch) = self.current_char() {
                    if ch == '}' {
                        break;
                    }
                    if !ch.is_ascii_hexdigit() {
                        return Err(LexError::InvalidEscape { seq: format!("\\u{{{}}}", hex), pos });
                    }
                    hex.push(ch);
                    self.advance();
                }
                if self.current_char() != Some('}') {
                    return Err(LexError::InvalidEscape { seq: format!("\\u{{{}}}", hex), pos });
                }
                self.advance(); // 跳过 '}'
                
                let codepoint = u32::from_str_radix(&hex, 16)
                    .map_err(|_| LexError::InvalidEscape { seq: format!("\\u{{{}}}", hex), pos })?;
                char::from_u32(codepoint)
                    .ok_or_else(|| LexError::InvalidUnicode { value: codepoint, pos })
            }
            Some(c) => Err(LexError::InvalidEscape { seq: format!("\\{}", c), pos }),
            None => Err(LexError::UnclosedString { start: pos }),
        }
    }
}
```

## 3.11 字符字面量解析

```rust
impl<'a> Lexer<'a> {
    fn lex_char(&mut self, start: SourcePosition) -> Result<Token, LexError> {
        self.advance(); // 跳过开头的 '
        let mut lexeme = String::new();
        lexeme.push('\'');
        
        let ch_val = if self.current_char() == Some('\\') {
            lexeme.push('\\');
            self.advance();
            let escaped = self.parse_escape_sequence(start)?;
            lexeme.push(escaped);
            escaped
        } else {
            let ch = self.current_char().ok_or(LexError::UnclosedChar { start })?;
            lexeme.push(ch);
            self.advance();
            ch
        };
        
        // 期望闭合引号
        if self.current_char() != Some('\'') {
            return Err(LexError::UnclosedChar { start });
        }
        lexeme.push('\'');
        self.advance();
        
        let end = self.current_position();
        let span = SourceSpan::new(start, end);
        Ok(Token::new(TokenKind::CharLit(ch_val), span, lexeme))
    }
}
```

## 3.12 标识符或关键词解析

```rust
impl<'a> Lexer<'a> {
    fn lex_identifier_or_keyword(&mut self, start: SourcePosition) -> Result<Token, LexError> {
        let mut lexeme = String::new();
        
        while let Some(ch) = self.current_char() {
            if is_identifier_char(ch) {
                lexeme.push(ch);
                self.advance();
            } else {
                break;
            }
        }
        
        let end = self.current_position();
        let span = SourceSpan::new(start, end);
        
        // 查询关键词表
        let kind = self.keyword_table.get(&lexeme)
            .unwrap_or_else(|| TokenKind::Ident(lexeme.clone()));
        
        Ok(Token::new(kind, span, lexeme))
    }
}
```

## 3.13 符号解析

```rust
impl<'a> Lexer<'a> {
    fn lex_symbol(&mut self, start: SourcePosition) -> Result<Token, LexError> {
        let ch = self.current_char().unwrap();
        self.advance();
        
        let kind = match ch {
            '(' => TokenKind::LParen,
            ')' => TokenKind::RParen,
            '{' => TokenKind::LBrace,
            '}' => TokenKind::RBrace,
            '[' => TokenKind::LBracket,
            ']' => TokenKind::RBracket,
            ',' => TokenKind::Comma,
            '.' => TokenKind::Dot,
            ';' => TokenKind::Semicolon,
            ':' => {
                if self.current_char() == Some(':') {
                    self.advance();
                    TokenKind::DoubleColon
                } else {
                    TokenKind::Colon
                }
            }
            '=' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::EqEq
                } else if self.current_char() == Some('>') {
                    self.advance();
                    TokenKind::Arrow
                } else {
                    TokenKind::Assign
                }
            }
            '+' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::PlusEq
                } else {
                    TokenKind::Plus
                }
            }
            '-' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::MinusEq
                } else {
                    TokenKind::Minus
                }
            }
            '*' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::StarEq
                } else {
                    TokenKind::Star
                }
            }
            '/' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::SlashEq
                } else {
                    TokenKind::Slash
                }
            }
            '%' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::PercentEq
                } else {
                    TokenKind::Percent
                }
            }
            '<' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::Le
                } else if self.current_char() == Some('<') {
                    self.advance();
                    if self.current_char() == Some('=') {
                        self.advance();
                        TokenKind::ShlEq
                    } else {
                        TokenKind::Shl
                    }
                } else {
                    TokenKind::Lt
                }
            }
            '>' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::Ge
                } else if self.current_char() == Some('>') {
                    self.advance();
                    if self.current_char() == Some('=') {
                        self.advance();
                        TokenKind::ShrEq
                    } else {
                        TokenKind::Shr
                    }
                } else {
                    TokenKind::Gt
                }
            }
            '!' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::NotEq
                } else {
                    TokenKind::Not
                }
            }
            '&' => {
                if self.current_char() == Some('&') {
                    self.advance();
                    TokenKind::And
                } else if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::BitAndEq
                } else {
                    TokenKind::BitAnd
                }
            }
            '|' => {
                if self.current_char() == Some('|') {
                    self.advance();
                    TokenKind::Or
                } else if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::BitOrEq
                } else {
                    TokenKind::BitOr
                }
            }
            '^' => {
                if self.current_char() == Some('=') {
                    self.advance();
                    TokenKind::BitXorEq
                } else {
                    TokenKind::BitXor
                }
            }
            _ => return Err(LexError::UnexpectedChar { ch, pos: start }),
        };
        
        let end = self.current_position();
        let span = SourceSpan::new(start, end);
        let lexeme = ch.to_string();
        
        Ok(Token::new(kind, span, lexeme))
    }
}
```

## 3.14 错误恢复策略

当词法分析器遇到无法识别的字符时，记录错误并跳过该字符，继续分析后续内容，以尽可能在一次编译中报告所有词法错误。

```rust
impl<'a> Lexer<'a> {
    fn recover_from_error(&mut self, err: LexError) -> Option<Token> {
        self.errors.push(err);
        // 跳过当前字符
        self.advance();
        // 尝试继续解析
        None
    }
}
```

## 3.15 性能优化建议

1. **预计算关键词表**：使用 `phf` 或 `lazy_static` 生成静态完美哈希表，加速关键词查找。
2. **避免字符数组拷贝**：尽量使用字符串切片而非复制字符数组，但需注意 UTF-8 边界。
3. **内联热路径**：`advance`、`current_char` 等方法应标记为 `#[inline]`。
4. **使用位标志判断字符类别**：预计算字符类别表（如 256 项数组），快速判断数字、字母、空白。

```rust
#[cfg(test)]

        }
```

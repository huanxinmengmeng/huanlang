# 12. 互操作与跨语言转换框架

## 12.1 互操作框架概述

幻语通过多层互操作架构实现与主流编程语言的深度集成，使开发者能够：

- **直接调用外部库**：通过 FFI 调用 C、Python、Java 等语言的现有库。
- **生成多语言绑定**：自动生成 Kotlin、Swift、Python、Go 等语言的原生绑定。
- **跨语言代码转译**：将幻语代码转换为 C、Rust、Python、Java、Go、JavaScript 等语言的等效代码。
- **库导入与优化**：导入外部库并自动转换为幻语原生库，享受完整类型检查与编译优化。

**互操作架构图**：

## 12.2 第 1 层：FFI 接口层（直接调用外部库）

### 12.2.1 外部块语法

```hl
外部 "语言标识" {
    { 外部函数声明 | 外部变量声明 | 导入语句 }
}
```

**语言标识**：`"C"`、`"Python"`、`"Java"`、`"JavaScript"`、`"Rust"`、`"asm"`。

### 12.2.2 C FFI 调用

C FFI 是幻语与外部库交互的基础，所有兼容 C ABI 的语言均可通过此方式调用。

```hl
外部 "C" {
    -- 函数声明
    函数 malloc(大小 类型 整数) 返回 指针[无符号8]
    函数 free(指针 类型 指针[无符号8])
    函数 printf(格式 类型 字符串, ...) 返回 整数
    
    -- 变量声明（只读）
    定 errno 类型 整数
}

-- 调用
函数 演示C调用()
开始
    令 缓冲区 为 malloc(1024)
    若 缓冲区 不等于 空 则
        printf("分配了 1024 字节\n")
        free(缓冲区)
    结束
结束
```

**编译器处理**：
- 解析外部块，将函数签名记录到符号表。
- 生成 LLVM IR 时，使用正确的 C 调用约定（`ccc`）和名称修饰（无修饰）。
- 链接时自动添加 `-l` 参数（通过包配置的 `[extern]` 节）。

### 12.2.3 Python FFI 调用

Python FFI 通过嵌入 CPython 解释器或生成 Python C API 包装代码实现。

```hl
外部 "Python" 导入 "numpy" 为 np
外部 "Python" 导入 "requests" 为 请求

函数 分析数据()
开始
    令 数组 为 np.数组([1, 2, 3, 4, 5])
    令 均值 为 np.均值(数组)
    显示("平均值: ", 均值)
    
    令 响应 为 请求.获取("https://api.example.com/data")
    显示("状态码: ", 响应.状态码)
结束
```

**实现架构**：

```
┌─────────────────────────────────────────────────────────────────┐
│                    幻语 Python FFI 实现                          │
├─────────────────────────────────────────────────────────────────┤
│  1. 解析 "外部 Python 导入 ..." 语句                             │
│  2. 在运行时初始化 CPython 解释器（若未初始化）                   │
│  3. 对于每个 Python 函数调用：                                   │
│     - 将幻语参数转换为 PyObject*                                 │
│     - 调用 PyObject_CallObject                                   │
│     - 将返回值转换回幻语类型                                     │
│  4. 编译时也可选择将 Python 调用转译为等效幻语代码                │
└─────────────────────────────────────────────────────────────────┘
```

### 12.3.3 绑定生成器实现

```rust
pub struct BindingGenerator {
    target_language: TargetLanguage,
    output_dir: PathBuf,
}

pub enum TargetLanguage {
    Kotlin, Swift, Python, Go, CSharp, Java,
}

impl BindingGenerator {
    pub fn new(target: TargetLanguage, output_dir: PathBuf) -> Self {
        Self { target_language: target, output_dir }
    }
    
    pub fn generate(&self, program: &Program) -> Result<(), BindGenError> {
        // 1. 收集所有带 @导出 标注的项
        let exported = self.collect_exported_items(program);
        
        // 2. 生成 C ABI 包装层（C 头文件和实现）
        self.generate_c_abi_wrapper(&exported)?;
        
        // 3. 生成目标语言绑定代码
        match self.target_language {
            TargetLanguage::Kotlin => self.generate_kotlin_bindings(&exported),
            TargetLanguage::Swift => self.generate_swift_bindings(&exported),
            TargetLanguage::Python => self.generate_python_bindings(&exported),
            _ => unimplemented!(),
        }
    }
    
    fn generate_c_abi_wrapper(&self, items: &[ExportedItem]) -> Result<(), BindGenError> {
        // 生成 C 头文件
        let mut header = String::new();
        header.push_str("#ifndef HUAN_BINDINGS_H\n#define HUAN_BINDINGS_H\n\n");
        header.push_str("#include <stdint.h>\n\n");
        
        for item in items {
            match item {
                ExportedItem::Function(func) => {
                    header.push_str(&format!("{} {}(", 
                        c_type_of(&func.return_type), func.name));
                    for (i, (_, ty)) in func.params.iter().enumerate() {
                        if i > 0 { header.push_str(", "); }
                        header.push_str(&format!("{} arg{}", c_type_of(ty), i));
                    }
                    header.push_str(");\n");
                }
                ExportedItem::Struct(s) => {
                    header.push_str(&format!("typedef struct {{\n"));
                    for (name, ty) in &s.fields {
                        header.push_str(&format!("    {} {};\n", c_type_of(ty), name));
                    }
                    header.push_str(&format!("}} {};\n\n", s.name));
                }
            }
        }
        header.push_str("\n#endif\n");
        
        // 写入文件
        std::fs::write(self.output_dir.join("huan_bindings.h"), header)?;
        
        // 生成 C 实现（调用幻语函数）
        // ...
        Ok(())
    }
}
```

## 12.4 第 3 层：代码转译层（CrossTL 框架）

### 12.4.1 转译器架构

幻语代码转译器采用统一的中间表示（UIR），支持双向转换。

```
源语言 AST → 幻语 UIR → 目标语言 AST → 目标语言源码
```

**支持的语言**：C、C++、Rust、Python、Java、Go、JavaScript、TypeScript、Kotlin、Swift、Dart、Carbon、Zig。

### 12.4.2 转译器接口

```rust
pub trait Transpiler {
```

### 12.4.4 转译器实现核心

```rust
pub struct HuanTranspiler {
    target: TargetLanguage,
}

impl Transpiler for HuanTranspiler {
    fn transpile(&self, program: &Program, target: TargetLanguage) -> Result<String, TranspileError> {
        let mut output = String::new();
        
        for item in program {
            match item {
                Item::Function(func) => {
                    output.push_str(&self.transpile_function(func));
                }
                Item::Struct(s) => {
                    output.push_str(&self.transpile_struct(s));
                }
                // ... 其他项
            }
            output.push('\n');
        }
        
        Ok(output)
    }
}

impl HuanTranspiler {
    fn transpile_function(&self, func: &Function) -> String {
        match self.target {
            TargetLanguage::Rust => {
                let mut s = format!("fn {}({}) -> {} {{\n", 
                    func.name.name,
                    self.format_params_rust(&func.params),
                    self.format_type_rust(&func.return_type));
                for stmt in &func.body {
                    s.push_str(&self.transpile_stmt_rust(stmt));
                }
                s.push_str("}\n");
                s
            }
            TargetLanguage::Python => {
                let mut s = format!("def {}({}) -> {}:\n",
                    func.name.name,
                    self.format_params_python(&func.params),
                    self.format_type_python(&func.return_type));
                for stmt in &func.body {
                    s.push_str(&format!("    {}", self.transpile_stmt_python(stmt)));
                }
                if func.body.is_empty() {
                    s.push_str("    pass\n");
                }
                s
            }
            TargetLanguage::C => {
                let mut s = format!("{} {}({}) {{\n",
                    self.format_type_c(&func.return_type),
                    func.name.name,
                    self.format_params_c(&func.params));
                for stmt in &func.body {
                    s.push_str(&format!("    {}", self.transpile_stmt_c(stmt)));
                }
                s.push_str("}\n");
                s
            }
            // ... 其他语言
            _ => unimplemented!(),
        }
    }
    
    fn transpile_stmt_rust(&self, stmt: &Stmt) -> String {
        match stmt {
            Stmt::Let { name, value, .. } => {
                format!("let {} = {};\n", name.name, self.transpile_expr_rust(value))
            }
            Stmt::If { cond, then_block, else_block, .. } => {
                let mut s = format!("if {} {{\n", self.transpile_expr_rust(cond));
                for st in then_block {
                    s.push_str(&format!("    {}", self.transpile_stmt_rust(st)));
                }
                s.push_str("}");
                if let Some(else_blk) = else_block {
                    s.push_str(" else {\n");
                    for st in else_blk {
                        s.push_str(&format!("    {}", self.transpile_stmt_rust(st)));
                    }
                    s.push_str("}");
                }
                s.push('\n');
                s
            }
            Stmt::Return(Some(expr), _) => {
                format!("return {};\n", self.transpile_expr_rust(expr))
            }
            // ... 其他语句
            _ => "".to_string(),
        }
    }
    
    fn transpile_expr_rust(&self, expr: &Expr) -> String {
        match expr {
            Expr::IntLit(n, _) => n.to_string(),
            Expr::Ident(ident) => ident.name.clone(),
            Expr::BinaryOp { op, left, right, .. } => {
                format!("{} {} {}", 
                    self.transpile_expr_rust(left),
                    self.rust_op(op),
                    self.transpile_expr_rust(right))
            }
            Expr::Call { func, args, .. } => {
                let args_str: Vec<String> = args.iter().map(|a| self.transpile_expr_rust(a)).collect();
                format!("{}({})", self.transpile_expr_rust(func), args_str.join(", "))
            }
            // ... 其他表达式
            _ => "/* TODO */".to_string(),
        }
    }
    
    fn rust_op(&self, op: &BinaryOp) -> &'static str {
        match op {
            BinaryOp::Add => "+",
            BinaryOp::Sub => "-",
            BinaryOp::Mul => "*",
            BinaryOp::Div => "/",
            BinaryOp::Lt => "<",
            BinaryOp::Gt => ">",
            BinaryOp::Eq => "==",
            // ...
            _ => "/* op */",
        }
    }
}
```

### 12.4.5 库导入与转换优化

幻语能够**直接导入其他语言的库**并将其**转译为幻语原生库**。

```bash
huan 导入库 numpy --语言 python --绑定风格 完全转译
```

**工作流程**：
1. **解析**：解析源库的 API（Python 通过 `inspect`，C 通过头文件解析）。
2. **转换**：将源 API 转换为幻语的等价表示。
3. **优化**：应用幻语编译优化。
4. **输出**：生成幻语库（`.hl` 文件）。

**转换优化级别**：

| 级别 | 模式 | 描述 | 性能 |
|------|------|------|------|
| 1 | 动态绑定 | 运行时调用外部库 | 有调用开销 |
| 2 | 静态绑定 | 编译时生成 FFI 包装 | 零开销调用 |
| 3 | 完全转译 | 将库源码转换为幻语并编译 | 完全优化，可内联 |
